# <a name="mathatan-function-javascript"></a>Funzione Math.atan (JavaScript)
Restituisce l'arcotangente di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>Note  
 Obbligatorio `number` argomento è un'espressione numerica per cui è necessaria l'arcotangente.  
  
 Il valore restituito è l'arcotangente di `number` argomento, in radianti.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
