# <a name="infinity-constant-javascript"></a>Costante Infinity (JavaScript)
Un numero è maggiore di numero a virgola mobile più grande. Restituisce un valore iniziale di `Number.POSITIVE_INFINITY`. Infinito negativo (-infinito) è inferiore a virgola mobile più piccolo numero di punti.  
  
## <a name="syntax"></a>Sintassi  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>Note  
 Il `Infinity` costante è un membro del `Global` dell'oggetto e viene resa disponibile quando viene inizializzato il motore di script.
