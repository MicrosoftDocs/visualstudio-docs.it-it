# <a name="mathtanh-function-javascript"></a>Funzione Math.tanh (JavaScript)
Restituisce la tangente iperbolica inversa di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>Note  
 Il valore restituito è lo stesso di `(exp(x) - exp(-x))/(exp(x) + exp(-x))`. Se `x` è `NaN`, il risultato è `NaN`.
