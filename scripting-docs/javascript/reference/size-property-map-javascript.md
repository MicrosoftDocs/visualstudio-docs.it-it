# <a name="size-property-map-javascript"></a>Proprietà size (Map) (JavaScript)
Restituisce il numero di elementi in una mappa.  
  
## <a name="syntax"></a>Sintassi  
  
```JavaScript  
sizeVar = mapObj.size  
```  
  
#### <a name="parameters"></a>Parametri  
 `sizeVar`  
 Obbligatorio. Qualsiasi numero.  
  
 `mapObj`  
 Obbligatorio. Qualsiasi oggetto `Map`.
