# <a name="resolvedoptions-method-intlnumberformat"></a>Metodo resolvedOptions (Intl.NumberFormat)
Restituisce un oggetto che contiene le proprietà e valori di `NumberFormat` oggetto formattatore.  
  
## <a name="syntax"></a>Sintassi  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametri  
 `numberFormatObj`  
 Il `NumberFormat` oggetto per recuperare informazioni dal.  
  
## <a name="remarks"></a>Note  
 Le proprietà dell'oggetto restituito corrispondono alle proprietà calcolate del `NumberFormat` oggetto.
