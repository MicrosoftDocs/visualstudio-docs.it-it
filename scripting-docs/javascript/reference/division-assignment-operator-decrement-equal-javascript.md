# <a name="division-assignment-operator---javascript"></a>Operatore di assegnazione divisione (-) (JavaScript)
Divide il valore di una variabile per il valore di un'espressione e assegna il risultato alla variabile.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
result /= expression   
```  
  
## <a name="parameters"></a>Parametri  
 `result`  
 Qualsiasi variabile di tipo numerico.  
  
 `expression`  
 Qualsiasi espressione numerica.
