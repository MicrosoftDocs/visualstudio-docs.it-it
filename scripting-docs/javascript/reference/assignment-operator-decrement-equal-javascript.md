# <a name="assignment-operator--javascript"></a>Operatore di assegnazione (=) (JavaScript)
Assegna un valore a una variabile.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>Parametri  
 `result`  
 Qualsiasi variabile.  
  
 `expression`  
 Qualsiasi espressione numerica.  
  
## <a name="remarks"></a>Note  
 I = (operatore) si comporta come altri operatori, in modo che le espressioni che contengono un valore. Ciò significa che è possibile concatenare gli operatori di assegnazione nel modo seguente: `j = k = l = 0`. In questo caso `j`, `k`, e `l` uguale a zero.
