# <a name="debugdebuggerenabled-property"></a>Proprietà Debug.debuggerEnabled
Determina se il debug è abilitato per il contesto di script. Debug può essere abilitato o disabilitato se è collegato un debugger.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
