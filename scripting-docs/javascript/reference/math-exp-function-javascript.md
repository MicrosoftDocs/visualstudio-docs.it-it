# <a name="mathexp-function-javascript"></a>Funzione Math.exp (JavaScript)
Restituisce il numero e (base dei logaritmi naturali) elevato a potenza.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>Note  
 Obbligatorio `number` argomento è un'espressione numerica che rappresenta la potenza di *e*.  
  
 Il valore restituito è un numero. La costante *e* è di Eulero, number, approssimativamente al 2,71828 e `number` è l'argomento specificato.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
