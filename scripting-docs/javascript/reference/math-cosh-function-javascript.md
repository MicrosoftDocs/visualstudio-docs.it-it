# <a name="mathcosh-function-javascript"></a>Funzione Math.cosh (JavaScript)
Restituisce il coseno iperbolico di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>Note  
 Il valore restituito è lo stesso di `(exp(x) + exp(-x))/2`. Se `x` è `NaN`, il risultato è `NaN`.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
