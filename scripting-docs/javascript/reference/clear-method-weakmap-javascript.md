# <a name="clear-method-weakmap-javascript"></a>Metodo clear (WeakMap) (JavaScript)
Rimuove tutti gli elementi da un oggetto `WeakMap`.  
  
## <a name="syntax"></a>Sintassi  
  
```JavaScript  
weakmapObj.clear()  
```  
  
#### <a name="parameters"></a>Parametri  
 `weakmapObj`  
 Obbligatorio. Il `WeakMap` oggetto da cancellare.
