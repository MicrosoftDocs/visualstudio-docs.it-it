# <a name="mathsinh-function-javascript"></a>Funzione Math.sinh (JavaScript)
Restituisce il seno iperbolico inverso di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>Note  
 Il valore restituito è lo stesso di `(exp(x) - exp(-x))/2`. Se x è `NaN`, il risultato è `NaN`.
