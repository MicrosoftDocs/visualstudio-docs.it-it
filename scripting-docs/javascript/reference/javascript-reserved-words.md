# <a name="javascript-reserved-words"></a>Parole riservate in JavaScript
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]un numero di parole riservate che non è possibile utilizzare come identificatori. Parole riservate hanno un significato specifico per il [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] lingua, come fanno parte della sintassi del linguaggio. Utilizzando una parola riservata causa un errore di compilazione durante il caricamento dello script.  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]include inoltre un elenco di parole riservate future. Queste parole non sono attualmente parte di [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] language, anche se sono riservati per utilizzi futuri. Per ulteriori informazioni sulle parole chiave riservate future in JavaScript, vedere [parole riservate Future JavaScript](../../javascript/reference/javascript-future-reserved-words.md).  
  
 Quando si scelgono gli identificatori è anche importante evitare le parole che sono già i nomi di funzione intrinseca [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] oggetti o funzioni, ad esempio `String` o `parseInt`.
