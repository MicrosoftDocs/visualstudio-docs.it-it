# <a name="mathatanh-function-javascript"></a>Funzione Math.atanh (JavaScript)
Restituisce la tangente iperbolica inversa di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>Note  
 Se `x` è `NaN`, il risultato è `NaN`.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
