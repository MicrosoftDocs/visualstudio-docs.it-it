# <a name="clear-method-map-javascript"></a>Metodo clear (Map) (JavaScript)
Rimuove tutti gli elementi da una mappa.  
  
## <a name="syntax"></a>Sintassi  
  
```JavaScript  
mapObj.clear()  
```  
  
#### <a name="parameters"></a>Parametri  
 `mapObj`  
 Obbligatorio. Cancellare la mappa.
