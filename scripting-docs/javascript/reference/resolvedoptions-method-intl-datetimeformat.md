# <a name="resolvedoptions-method-intldatetimeformat"></a>Metodo resolvedOptions (Intl.DateTimeFormat)
Restituisce un oggetto in cui sono contenuti le proprietà e i valori dell'oggetto formattatore di data/ora.  
  
## <a name="syntax"></a>Sintassi  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametri  
 `dateTimeFormatObj`  
 Il `DateTimeFormat` oggetto per recuperare informazioni dal.  
  
## <a name="remarks"></a>Note  
 Le proprietà dell'oggetto restituito corrispondono alle proprietà calcolate del `DateTimeFormat` oggetto.
