# <a name="modulus-assignment-operator--javascript"></a>Operatore di assegnazione di modulo (JavaScript)
Divide il valore di una variabile per il valore di un'espressione e assegna il resto della divisione alla variabile.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
result %= expression  
```  
  
## <a name="arguments"></a>Argomenti  
 `result`  
 Qualsiasi variabile.  
  
 `expression`  
 Qualsiasi espressione numerica.  
  
## <a name="remarks"></a>Note  
 L'utilizzo dell'operatore `%=` equivale esattamente a specificare:  
  
```  
result = result % expression  
```
