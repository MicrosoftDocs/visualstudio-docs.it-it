# <a name="size-property-set-javascript"></a>Proprietà size (Set) (JavaScript)
Restituisce il numero di elementi in un set.  
  
## <a name="syntax"></a>Sintassi  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>Parametri  
 `sizeVar`  
 Obbligatorio. Qualsiasi numero.  
  
 `setObj`  
 Obbligatorio. Qualsiasi oggetto `Set`.
