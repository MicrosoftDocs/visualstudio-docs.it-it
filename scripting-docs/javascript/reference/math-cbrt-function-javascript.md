# <a name="mathcbrt-function-javascript"></a>Funzione Math.cbrt (JavaScript)
Restituisce la radice cubica di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>Note  
 Se `number` è `NaN`, il risultato è `NaN`.
