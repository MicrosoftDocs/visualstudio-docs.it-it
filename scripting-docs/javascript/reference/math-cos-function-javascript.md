# <a name="mathcos-function-javascript"></a>Funzione Math.cos (JavaScript)
Restituisce il coseno di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>Note  
 Obbligatorio `radians` argomento è un'espressione numerica che contiene un angolo, espresso in radianti.  
  
 Il valore restituito è il coseno dell'argomento numerico di `radians`.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
