# <a name="tostring-method-weakmap-javascript"></a>Metodo toString (WeakMap) (JavaScript)
Restituisce una rappresentazione di stringa di un `WeakMap` oggetto.  
  
## <a name="syntax"></a>Sintassi  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>Parametri  
 `weakmapObj`  
 Obbligatorio. Oggetto `WeakMap`.
