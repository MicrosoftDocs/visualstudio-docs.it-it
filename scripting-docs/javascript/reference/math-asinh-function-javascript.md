# <a name="mathasinh-function-javascript"></a>Funzione Math.asinh (JavaScript)
Restituisce il seno iperbolico inverso di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.asinh(x)   
```  
  
## <a name="remarks"></a>Note  
 Se `x` è `NaN`, il risultato è `NaN`.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
