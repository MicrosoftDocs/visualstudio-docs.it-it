# <a name="nan-constant-javascript"></a>Costante NaN (JavaScript)
Una costante speciale che specifica un'espressione thatNot non è un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
NaN   
```  
  
## <a name="remarks"></a>Note  
 Il `NaN` costante (non un numero) è un membro del `Global` dell'oggetto e viene resa disponibile quando viene inizializzato il motore di script.
