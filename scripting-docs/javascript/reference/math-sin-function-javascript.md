# <a name="mathsin-function-javascript"></a>Funzione Math.sin (JavaScript)
Restituisce il seno di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>Note  
 Il `radians` argomento è un'espressione numerica che contiene un angolo, espresso in radianti.  
  
 Il valore restituito è il seno dell'argomento numerico.
