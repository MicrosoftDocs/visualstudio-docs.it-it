# <a name="void-operator-javascript"></a>Operatore void (JavaScript)
Impedisce a un'espressione di restituire un valore.  
  
## <a name="syntax"></a>Sintassi  
  
```  
void expression   
```  
  
## <a name="remarks"></a>Note  
 Il `expression` è qualsiasi [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] espressione.  
  
 Il `void` operatore viene valutata l'espressione e restituisce `undefined`. È utile nelle situazioni in cui valutare un'espressione, ma non si desidera i risultati visibili al resto dello script.
