# <a name="mathceil-function-javascript"></a>Funzione Math.ceil (JavaScript)
Restituisce l'intero minimo maggiore o uguale al relativo argomento numerico.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>Note  
 L'argomento `number` obbligatorio è un'espressione numerica.  
  
 Il valore restituito è un valore intero uguale al valore integer più piccolo maggiore o uguale al relativo argomento numerico.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
