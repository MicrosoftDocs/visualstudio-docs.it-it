# <a name="isfinite-function-javascript"></a>Funzione isFinite (JavaScript)
Determina se il numero specificato è finito.  
  
## <a name="syntax"></a>Sintassi  
  
```  
isFinite(number)   
```  
  
## <a name="remarks"></a>Note  
 Obbligatorio `number` argomento è un valore numerico.  
  
 Il `isFinite` risultato della funzione `true` se `number` è qualsiasi valore diverso da `NaN`, infinito negativo o un numero infinito positivo. In caso contrario, restituisce **false**.
