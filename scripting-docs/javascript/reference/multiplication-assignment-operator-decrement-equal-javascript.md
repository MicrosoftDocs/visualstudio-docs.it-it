# <a name="multiplication-assignment-operator--javascript"></a>Operatore di assegnazione di moltiplicazione (*=) (JavaScript)
Moltiplica il valore di una variabile per il valore di un'espressione e assegna il risultato alla variabile.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
result *= expression  
```  
  
## <a name="parameters"></a>Parametri  
 *risultato*  
 Qualsiasi variabile.  
  
 *espressione*  
 Qualsiasi espressione.
