# <a name="mathrandom-function-javascript"></a>Funzione Math.random (JavaScript)
Restituisce un numero pseudocasuale compreso tra 0 e 1.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>Note  
 Il numero pseudocasuale generato è compreso tra 0 (inclusi) su 1 (esclusivo), vale a dire, il numero restituito può essere zero, ma sarà sempre minore di 1. Generatore di numeri casuali viene inizializzato automaticamente quando [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] viene caricata.
