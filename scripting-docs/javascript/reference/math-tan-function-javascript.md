# <a name="mathtan-function-javascript"></a>Funzione Math.tan (JavaScript)
Restituisce la tangente di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Note  
 Obbligatorio `radians` argomento è un'espressione numerica che contiene un angolo, espresso in radianti.  
  
 Il valore restituito è la tangente dell'argomento numerico di `radians`.
