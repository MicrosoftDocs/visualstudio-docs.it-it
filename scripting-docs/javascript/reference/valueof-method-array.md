# <a name="valueof-method-array"></a>Metodo valueOf (Array)
Restituisce il valore primitivo dell'opzione specificata.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
array.valueOf()  
```  
  
#### <a name="parameters"></a>Parametri  
 Questo metodo non ha parametri.  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce l'istanza di matrice.  
  
## <a name="remarks"></a>Note  
 Nell'esempio seguente, l'oggetto di un'istanza di matrice Ã¨ lo stesso come il valore restituito di questo metodo.  
  
```JavaScript  
var arr = [1, 2, 3, 4];  
var s = arr.valueOf();  
  
if (arr === s)  
document.write("same");  
else  
document.write("different");  
  
// Output:  
// same  
  
```
