# <a name="mathtrunc-function-javascript"></a>Funzione Math.trunc (JavaScript)
Restituisce la parte intera di un numero, rimuovendo eventuali cifre frazionarie.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
Math.trunc(number)  
```  
  
## <a name="remarks"></a>Note  
 L'argomento `number` obbligatorio è un'espressione numerica per la quale è necessario l'intero troncato.  
  
 Se `number` è `NaN`, restituisce `NaN`.
