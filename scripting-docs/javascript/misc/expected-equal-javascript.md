# <a name="expected--javascript"></a>Previsto '=' (JavaScript)
Si è tentato di creare una variabile da utilizzare con le istruzioni di compilazione condizionale non è stato inserito un segno di uguale tra la variabile e il valore da assegnare a esso.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere un segno di uguale. Ad esempio:  
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
