# <a name="expected-"></a>Previsto '}'
Non si includere la parentesi graffa destra che contrassegna la fine del corpo della funzione, ciclo, blocco di codice o inizializzatore di oggetto. Un esempio di questo errore potrebbe essere un' **per** ciclo con solo la parentesi graffa sinistra contrassegnando il corpo del ciclo.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere la parentesi graffa destra che contrassegna la fine della funzione, ciclo, blocco o inizializzatore di oggetto.
