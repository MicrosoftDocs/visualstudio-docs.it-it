# <a name="expected-identifier-string-or-number"></a>Previsto identificatore, stringa o numero
Sintassi del valore letterale non corretta utilizzato per dichiarare un oggetto valore letterale. Le proprietà di un valore letterale di oggetto devono essere un identificatore, stringa o un numero. Un valore letterale di oggetto (detto anche un inizializzatore di oggetto"") è costituito da un elenco delimitato da virgole di coppie property: value, racchiuse tra parentesi. Ad esempio:  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Accertarsi di utilizzare la sintassi corretta letterale.
