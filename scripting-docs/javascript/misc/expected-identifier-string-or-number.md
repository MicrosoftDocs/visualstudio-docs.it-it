# <a name="expected-identifier-string-or-number"></a>Previsto identificatore, stringa o numero
Sintassi non corretta dei valori letterali usata per dichiarare un oggetto letterale. Le proprietà di un valore letterale di oggetto devono essere un identificatore, stringa o un numero. Un valore letterale di oggetto (detto anche un inizializzatore di oggetto"") è costituito da un elenco delimitato da virgole di coppie property: value, racchiuse tra parentesi. Ad esempio:  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Assicurarsi di che usare la sintassi dei valori letterali appropriata.
