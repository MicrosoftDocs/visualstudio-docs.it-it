# <a name="number-expected"></a>Previsto numero
Si è provato a richiamare il **Number.prototype.toString** oppure **Number.prototype.valueOf** metodo in un oggetto di un tipo diverso da **numero**. L'oggetto di questo tipo di chiamata deve essere di tipo **numero**.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Richiamare solo le **Number.prototype.toString** oppure **Number.prototype.valueOf** metodi su oggetti di tipo **numero**.
