# <a name="expected-end"></a>Previsto ' \@end '
Si è provato a creare un blocco di codice compilato in modo condizionale, ma non è stata inclusa l'istruzione <strong>@end</strong> alla fine. [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] le istruzioni possono essere compilate in modo condizionale in un blocco di <strong>@end @if/</strong> .  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Aggiungere l'istruzione <strong>@end</strong> corrispondente.
