# <a name="expected-39end39"></a>Previsto &#39; @end&#39;
Ha tentato di creare un blocco di codice compilato in modo condizionale, ma non include il  **@end**  istruzione alla fine. [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]le istruzioni possono essere compilate in modo condizionale racchiusi all'interno di un  **@if/ @end**  blocco.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere la corrispondente  **@end**  istruzione.
