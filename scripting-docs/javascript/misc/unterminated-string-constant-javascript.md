# <a name="unterminated-string-constant-javascript"></a>Costante stringa senza terminazione (JavaScript)
Non è stato terminato la costante di stringa con virgolette di chiusura. Le costanti stringa devono essere racchiusi tra una coppia di virgolette doppie.  
  
> [!NOTE]
> È possibile usare coppie corrispondenti di virgolette singole o doppie. Le virgolette doppie possono essere contenute all'interno di stringhe racchiuse tra virgolette singole e virgolette singole possono essere contenute all'interno di stringhe racchiuse tra virgolette doppie.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Aggiungere virgolette di chiusura alla fine della stringa.
