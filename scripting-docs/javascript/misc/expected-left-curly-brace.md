# <a name="expected-"></a>Previsto '{'
Non sono inclusi la parentesi graffa sinistra che contrassegna l'inizio del corpo della funzione. Il codice che costituisce il corpo di una funzione, anche se si tratta di una singola riga, deve essere contenuto all'interno di parentesi graffe. Si noti che l'utilizzo di parentesi graffe nei corpi delle funzioni è più rigido di cicli.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere la parentesi graffa sinistra che contrassegna l'inizio del corpo della funzione.
