# <a name="expected-3939"></a>Previsto &#39; {&#39;
Non include la parentesi graffa sinistra che contrassegna l'inizio del corpo della funzione. Codice che costituisce il corpo di una funzione, anche se si tratta di una singola riga, deve essere contenuto all'interno delle parentesi graffe. Si noti che l'utilizzo di parentesi graffe in corpi di funzione è più restrittivo rispetto nei cicli.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere la parentesi graffa sinistra che contrassegna l'inizio del corpo della funzione.
