# <a name="expected--javascript"></a>Previsto ')' (JavaScript)
Si è provato a racchiudere un'espressione in un set di parentesi, ma non è stata inclusa la parentesi di chiusura. Alcune espressioni devono essere racchiuse in un set di parentesi di apertura e di chiusura. Si noti l'uso delle parentesi nell'esempio seguente.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Aggiungere le parentesi giuste all'espressione di valutazione.
