# <a name="expected-var"></a>Previsto 'var'
Previsto ' var ' (ERRnoVar 1034))  
  
 Non è stato possibile aggiungere la parola chiave var quando era necessaria.  
  
 Per ulteriori informazioni sulla parola chiave var, vedere [istruzione var](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Statements/var).
