# <a name="expected-identifier-javascript"></a>Previsto identificatore (JavaScript)
È stato usato un elemento diverso da un identificatore in un contesto in cui ne era richiesto uno. Un identificatore può essere:  
  
- una variabile,  
  
- una proprietà,  
  
- una matrice,  
  
- o un nome di funzione.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Modificare l'espressione in modo che un identificatore venga visualizzato a sinistra del segno di uguale.
