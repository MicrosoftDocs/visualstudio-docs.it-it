# <a name="expected-identifier-javascript"></a>Previsto identificatore (JavaScript)
È stato utilizzato un valore diverso da un identificatore in un contesto in cui uno è stato richiesto. Può essere un identificatore:  
  
-   una variabile,  
  
-   una proprietà,  
  
-   una matrice,  
  
-   o un nome di funzione.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare l'espressione in modo che un identificatore viene visualizzato a sinistra del segno di uguale.
