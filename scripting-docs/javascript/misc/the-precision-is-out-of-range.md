# <a name="the-precision-is-out-of-range"></a>La precisione non è compresa nell'intervallo
Si è tentato di passare un argomento non valido per la funzione **Number.prototype.toPrecision**. L'argomento **toPrecision** deve essere compreso tra 1 e 21 (inclusi).  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Verificare che l'argomento `toPrecision` non è troppo grande o troppo piccolo.
