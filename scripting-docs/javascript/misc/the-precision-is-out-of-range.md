# <a name="the-precision-is-out-of-range"></a>La precisione non è compresa nell'intervallo
Si è provato a passare un argomento non valido al **numero di funzione. Prototype. toPrecision**. L'argomento di **toPrecision** deve essere compreso tra 1 e 21 (inclusi).  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che l'argomento `toPrecision` non sia troppo grande o troppo piccolo.
