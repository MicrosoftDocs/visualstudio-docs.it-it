# <a name="scriptepropagate-error-code"></a>Codice di errore SCRIPT_E_PROPAGATE
Un errore di script viene propagato al chiamante, che potrebbe essere in un altro thread. L'host deve passare il codice di errore al chiamante.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
