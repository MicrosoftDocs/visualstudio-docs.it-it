# <a name="script_e_propagate-error-code"></a>Codice di errore SCRIPT_E_PROPAGATE
Un errore di script viene propagato al chiamante, che potrebbe trovarsi in un thread diverso. L'host deve passare il codice di errore al chiamante.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
