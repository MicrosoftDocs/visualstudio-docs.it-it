# <a name="scriptuichandling-enumeration"></a>Enumerazione SCRIPTUICHANDLING
Rappresenta il modo che il controllo dell'interfaccia utente deve essere gestito.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
typedef enum tagSCRIPTUICHANDLING {     SCRIPTUICHANDLING_ALLOW = 0,     SCRIPTUICHANDLING_NOUIERROR = 1,     SCRIPTUICHANDLING_NOUIDEFAULT = 2, } SCRIPTUICHANDLING;   
```  
  
## <a name="enumeration-value"></a>Valore di enumerazione  
  
|||  
|-|-|  
|SCRIPTUICHANDLING_ALLOW|Consenti il controllo da visualizzare.|
