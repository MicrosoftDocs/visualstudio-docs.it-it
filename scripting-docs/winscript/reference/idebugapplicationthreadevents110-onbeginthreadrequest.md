# <a name="idebugapplicationthreadevents110-onbeginthreadrequest"></a>IDebugApplicationThreadEvents110 ::OnBeginThreadRequest
Una chiamata nel thread con thread di PDM ha iniziato il passaggio.  
  
> [!IMPORTANT]
> [Interfaccia IDebugApplicationThreadEvents110](../../winscript/reference/idebugapplicationthreadevents110-interface.md) viene implementata da PDM v11.0 e versioni successive. Rilevata in activdbg100.h.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT OnBeginThreadRequest( void );  
```  
  
#### <a name="parameters"></a>Parametri  
 Questo metodo non ha parametri.
