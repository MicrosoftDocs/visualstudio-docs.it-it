# <a name="idebugapplicationthreadevents110-onbeginthreadrequest"></a>IDebugApplicationThreadEvents110 ::OnBeginThreadRequest
Una chiamata nel thread tramite il thread del PDM cambio è iniziata.  
  
> [!IMPORTANT]
>  [Interfaccia IDebugApplicationThreadEvents110](../../winscript/reference/idebugapplicationthreadevents110-interface.md) è implementata da PDM v 11.0 e versione successiva. Rilevata in activdbg100.h.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT OnBeginThreadRequest( void );  
```  
  
#### <a name="parameters"></a>Parametri  
 Questo metodo non ha parametri.
