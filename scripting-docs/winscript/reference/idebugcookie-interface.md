# <a name="idebugcookie-interface"></a>Interfaccia IDebugCookie
Consente l'impostazione del cookie di debug, da utilizzare con l'interfaccia `IMachineDebugManagerCookie`. Per ulteriori informazioni, vedere [interfaccia IMachineDebugManagerCookie](../../winscript/reference/imachinedebugmanagercookie-interface.md). Questa interfaccia viene implementata da Process Debug Manager (PDM) e utilizzata dai debugger di script.  
  
## <a name="methods"></a>Metodi  
 Oltre ai metodi ereditati da `IUnknown`, l'interfaccia `IDebugCookie` espone i metodi seguenti.  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IDebugCookie::SetDebugCookie](../../winscript/reference/idebugcookie-setdebugcookie.md)|Imposta il cookie dell'applicazione di debug.|
