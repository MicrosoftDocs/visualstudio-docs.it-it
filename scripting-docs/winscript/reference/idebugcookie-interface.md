# <a name="idebugcookie-interface"></a>Interfaccia IDebugCookie
Consente il debug cookie da impostare per l'utilizzo con il `IMachineDebugManagerCookie` interfaccia. Per ulteriori informazioni, vedere [interfaccia IMachineDebugManagerCookie](../../winscript/reference/imachinedebugmanagercookie-interface.md). Questa interfaccia Ã¨ implementata dal processo di Debug Manager (PDM) e utilizzata dal debugger di script.  
  
## <a name="methods"></a>Metodi  
 Oltre ai metodi ereditati da `IUnknown`, `IDebugCookie` interfaccia espone i metodi seguenti.  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IDebugCookie::SetDebugCookie](../../winscript/reference/idebugcookie-setdebugcookie.md)|Imposta il cookie dell'applicazione di debug.|
