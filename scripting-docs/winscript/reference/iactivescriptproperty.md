# <a name="iactivescriptproperty"></a>IActiveScriptProperty
L'interfaccia `IActiveScriptProperty` viene utilizzata per ottenere e impostare le proprietà di configurazione.  
  
## <a name="methods-in-vtable-order"></a>Metodi nell'ordine Vtable  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[GetProperty](../../winscript/reference/iactivescriptproperty-getproperty.md)|Ottiene la proprietà specificata dal parametro.|  
|[SetProperty](../../winscript/reference/iactivescriptproperty-setproperty.md)|Imposta la proprietà specificata dal parametro.|
