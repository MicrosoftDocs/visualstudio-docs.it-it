# <a name="iactivescriptstringcompare-interface"></a>Interfaccia IActiveScriptStringCompare
L'interfaccia `IActiveScriptCompare` viene utilizzata per definire il metodo di confronto tra stringhe per il motore di scripting.  
  
## <a name="methods-in-vtable-order"></a>Metodi nell'ordine Vtable  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IActiveScriptStringCompare::StrComp](../../winscript/reference/iactivescriptstringcompare-strcomp.md)|Definisce il metodo di confronto tra stringhe per il motore di scripting.|
