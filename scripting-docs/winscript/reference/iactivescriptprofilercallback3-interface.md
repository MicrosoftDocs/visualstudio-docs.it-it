# <a name="iactivescriptprofilercallback3-interface"></a>Interfaccia IActiveScriptProfilerCallback3
Fornisce un metodo per notificare al profiler che le esecuzioni profilate si verificano nel contesto di un Web Worker.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo IActiveScriptProfilerCallback3::SetWebWorkerId](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|Notifica al profiler l'ID del ruolo di lavoro da usare per questa sessione di profilatura.|
