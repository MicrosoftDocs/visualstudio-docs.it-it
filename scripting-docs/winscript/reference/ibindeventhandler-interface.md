# <a name="ibindeventhandler-interface"></a>Interfaccia IBindEventHandler
Consente al chiamante di fornire un oggetto che gestisce un gestore eventi specificato.  
  
## <a name="methods"></a>Metodi  
 Oltre ai metodi ereditati da `IUnknown`, il `IBindEventHandler` interfaccia espone i metodi seguenti.  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|Associa un evento a un oggetto.|
