# <a name="ienumremotedebugapplications-interface"></a>Interfaccia IEnumRemoteDebugApplications
Enumera gli oggetti dell'applicazione. Questa interfaccia pu√≤ essere utilizzata per enumerare le applicazioni in esecuzione in un computer per la finestra di dialogo "Collega all'applicazione".  
  
 Oltre ai metodi ereditati da `IUnknown`, il `IEnumRemoteDebugApplications` interfaccia espone i metodi seguenti.  
  
## <a name="methods-in-vtable-order"></a>Metodi nell'ordine Vtable  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IEnumRemoteDebugApplications::Clone](../../winscript/reference/ienumremotedebugapplications-clone.md)|Crea un enumeratore che contiene lo stesso stato dell'enumeratore corrente.|  
|[IEnumRemoteDebugApplications::Next](../../winscript/reference/ienumremotedebugapplications-next.md)|Recupera un determinato numero di segmenti in una sequenza di enumerazione.|  
|[IEnumRemoteDebugApplications::Reset](../../winscript/reference/ienumremotedebugapplications-reset.md)|Reimposta una sequenza di enumerazione all'inizio.|  
|[IEnumRemoteDebugApplications::Skip](../../winscript/reference/ienumremotedebugapplications-skip.md)|Ignora un determinato numero di segmenti in una sequenza di enumerazione.|
