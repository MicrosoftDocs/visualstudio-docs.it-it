# <a name="iactivescripttraceinfo-interface"></a>Interfaccia IActiveScriptTraceInfo
Fornisce metodi per avviare e arrestare la traccia.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo IActiveScriptTraceInfo::StartScriptTracing](../../winscript/reference/iactivescripttraceinfo-startscripttracing-method.md)|Avvia la traccia di script.|  
|[Metodo IActiveScriptTraceInfo::StopScriptTracing](../../winscript/reference/iactivescripttraceinfo-stopscripttracing-method.md)|Arresta la traccia.|
