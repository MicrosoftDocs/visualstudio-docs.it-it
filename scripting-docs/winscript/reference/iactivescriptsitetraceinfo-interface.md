# <a name="iactivescriptsitetraceinfo-interface"></a>Interfaccia IActiveScriptSiteTraceInfo
Fornisce un metodo per inviare le informazioni di traccia script.  
  
## <a name="methods"></a>Metodi  
  
|||  
|-|-|  
|Metodo|Descrizione|  
|[Metodo IActiveScriptSiteTraceInfo::SendScriptTraceInfo](../../winscript/reference/iactivescriptsitetraceinfo-sendscripttraceinfo-method.md)|Invia le informazioni di traccia che include il tipo di evento, il contesto e l'istruzione di script.|
