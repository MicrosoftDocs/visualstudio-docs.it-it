# <a name="iremotedebugapplicationexsetlocale"></a>IRemoteDebugApplicationEx:SetLocale
Imposta la lingua per la localizzazione del debugger.  
  
## <a name="syntax"></a>Sintassi  
  
```  
HRESULT SetLocale(  
   DWORD  dwLangID  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `dwLangID`  
 [in] ID della lingua.
