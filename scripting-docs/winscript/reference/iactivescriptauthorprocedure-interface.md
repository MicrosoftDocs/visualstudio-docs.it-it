# <a name="iactivescriptauthorprocedure-interface"></a>Interfaccia IActiveScriptAuthorProcedure
Oltre ai metodi ereditati da `IUnknown`, il `IScriptAuthorProcedure` interfaccia espone il metodo seguente.  
  
## <a name="methods-in-vtable-order"></a>Metodi nell'ordine Vtable  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analizza una routine di codice, aggiunge il testo della stored procedure codice allo script del motore di creazione e crea un `IScriptEntry` oggetto corrispondente alla routine del codice.|
