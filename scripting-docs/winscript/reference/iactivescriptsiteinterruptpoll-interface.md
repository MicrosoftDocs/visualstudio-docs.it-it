# <a name="iactivescriptsiteinterruptpoll-interface"></a>Interfaccia IActiveScriptSiteInterruptPoll
Il `IActiveScriptSiteInterruptPoll` interfaccia consente a un host specificare che deve terminare uno script.  
  
## <a name="methods"></a>Metodi  
 Oltre ai metodi ereditati da `IUnknown`, il `IActiveScriptSiteInterruptPoll` interfaccia espone il metodo seguente.  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|Consente a un host specificare che deve terminare uno script.|
