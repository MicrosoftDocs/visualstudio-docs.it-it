# <a name="javascript-language-reference"></a>Riferimento del linguaggio JavaScript

JavaScript è un linguaggio di script che può essere incorporato in pagine Web e in altre applicazioni.  
  
L'implementazione Microsoft di JavaScript è conforme alla specifica del linguaggio ECMAScript quinta edizione. Fornisce anche funzionalità aggiuntive non incluse negli standard ECMA.

> [!IMPORTANT]
> Microsoft si è impegnata nell'ambito della community per rendere la [documentazione Web di MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) il riferimento per lo sviluppo di una versione di anteprima, a partire dal reindirizzamento di tutte le 500 pagine del riferimento all'API JavaScript di Microsoft da [docs.Microsoft.com](https://docs.microsoft.com) alle rispettive controparti MDN e al porting del riferimento alle estensioni JS legacy per finalità storiche e di manutenzione.
