# <a name="visual-studio-documentation"></a>Documentazione di Visual Studio

Introduzione Questo repository contiene i file di origine per la documentazione tecnica di Visual Studio. Gli argomenti sono pubblicati in [docs.microsoft.com](https://docs.microsoft.com/visualstudio).

Questo repository è stato spostato il 23 giugno 2017 da https://github.com/Microsoft/vsdocs. Il traffico verso gli URL precedenti viene reindirizzato qui.

La documentazione di Visual Basic e Visual C# si trova nel [repository dotnet docs](https://github.com/dotnet/docs/tree/master/docs) e la documentazione di Visual C++ si trova nel [repository docs di C++](http://github.com/MicrosoftDocs/cpp-docs).

## <a name="contribute-to-the-documentation"></a>Collaborare alla stesura della documentazione

Accogliamo con favore i tuoi contributi per aiutarci a migliorare i documenti di Visual Studio. Tutti gli articoli in questo repository usano Markdown dal sapore GitHub.All the articles in this repository use GitHub-flavored markdown.

All'interno di questo archivio alcune cartelle sono dedicate ad aree funzionali specifiche di Visual Studio, ad esempio **debugger** per gli argomenti relativi al debug, **ide** per gli argomenti relativi all'ambiente IDE (Interactive Development Environment) di Visual Studio e così via. In ogni cartella, la sottocartella **/media** contiene i file delle immagini degli argomenti.

Per altre informazioni su come contribuire, vedere la [guida per i collaboratori](CONTRIBUTING.md).
