# <a name="visual-studio-tools-for-unity"></a>Visual Studio Tools per Unity
![Visual Studio Tools per Unity](../media/hero.png)

## <a name="overview"></a>Panoramica
Visual Studio Tools per Unity è un'estensione gratuita di Visual Studio che consente di trasformare Visual Studio in uno strumento potente per lo sviluppo di giochi e applicazioni multipiattaforma con Unity.

L'editor di Unity è la scelta ideale per comporre scenari di gioco, ma non è possibile usarlo per scriverci codice. Con Visual Studio Tools per Unity è possibile usare le funzionalità di produttività e di modifica e debug del codice di Visual Studio già note per creare script di editor e di gioco per i progetti Unity con C# ed eseguirne il debug con le potenti funzionalità di debug di Visual Studio.

Ma Visual Studio Tools per Unity non è solo questo. È infatti strettamente integrato con l'editor di Unity, di conseguenza non è necessario passare dall'uno all'altro per eseguire semplici attività. Include inoltre funzionalità per ottimizzare la produttività specifiche di Unity e permette di accedere rapidamente alla documentazione di Unity.

### <a name="compatible-with-visual-studio-community-on-windows-and-macos-and-bundled-with-unity"></a>Compatibile con Visual Studio community in Windows e macOS e in bundle con Unity
Visual Studio e Visual Studio per Mac community sono disponibili gratuitamente ed è incluso nell'installazione di Unity. Per ulteriori informazioni sull'installazione e la configurazione, vedere la [documentazione](getting-started-with-visual-studio-tools-for-unity.md) introduttiva Visual Studio Tools per Unity.

## <a name="intellisense-for-unity-messages"></a>IntelliSense per messaggi di Unity
IntelliSense semplifica e velocizza l'[implementazione di messaggi API di Unity](using-visual-studio-tools-for-unity.md#intellisense-for-unity-api-messages) come `OnCollisionEnter`, inclusi i relativi parametri.

![Finestra di dialogo IntelliSense che mostra OnCollisionEnter](../media/vs/intellisense-example.png)

## <a name="superior-debugging"></a>Debug di livello superiore
Visual Studio Tools per Unity supporta tutte le solide funzionalità di [debug](using-visual-studio-tools-for-unity.md#unity-debugging) disponibili in Visual Studio:

* Punti di interruzione, anche condizionali.
* Valutazione di espressioni complesse nella finestra Espressioni di controllo.
* Controllo e modifica del valore di variabili e argomenti.
* Drill down in strutture di dati e oggetti complessi.

![Arrestato in corrispondenza di un punto di interruzione che controlla le variabili](../media/vs/debugging-inspecting.png)

## <a name="integrated-suggestions-for-best-practices-and-performance-insights"></a>Suggerimenti integrati per le procedure consigliate e le informazioni dettagliate sulle prestazioni
Scrivi codice migliore che acquisisce le procedure consigliate con la comprensione approfondita dei progetti Unity di Visual Studio.

![Confronto tra stringhe di refactoring di Visual Studio con CompareTag](../media/vs/unity-diagnostics.png)

## <a name="codelens-support-for-unity-scripts-and-messages"></a>Supporto di CodeLens per script e messaggi di Unity
Gli script Unity e le funzioni di messaggio sono decorati con suggerimenti per semplificare il riconoscimento di ciò che viene fornito da Unity e del codice.

 ![Nuovo script che Mostra gli hint di CodeLens per lo script Unity e il messaggio Unity](../media/vs/codelens-support.png)

> [!NOTE]
> Il supporto CodeLens è disponibile in Visual Studio 2019.

## <a name="optimized-view-of-all-your-scripts-to-match-unity"></a>Visualizzazione ottimizzata di tutti gli script per la corrispondenza di Unity
Esplora progetti Unity (UPE) rappresenta un modo alternativo per visualizzare i file di progetto nel Esplora soluzioni standard. L'UPE filtra i file visualizzati e li presenta in una gerarchia che corrisponde a Unity ( **visualizzazione > Esplora progetti Unity** in Visual Studio 2019).

![Esplora progetti Unity](../media/vs/unity-project-explorer.png)

> [!NOTE]
> Esplora progetti Unity è disponibile in Visual Studio 2019. In Visual Studio per Mac, per impostazione predefinita il comportamento del riquadro della soluzione è simile per i progetti Unity. non sono necessarie altre visualizzazioni.
