# <a name="commandname-element"></a>CommandName (elemento)
L'elemento `CommandName` specifica il testo visualizzato nella categoria della tastiera nella finestra di dialogo **Opzioni** e nell'elenco **Comandi** della finestra di dialogo **Personalizza.**

## <a name="syntax"></a>Sintassi

```
<CommandName>MyCommand</CommandName>
```

## <a name="attributes-and-elements"></a>Attributi ed elementi
 Nelle sezioni seguenti vengono descritti gli attributi, gli elementi figlio e gli elementi padre.

### <a name="attributes"></a>Attributi
 No.

### <a name="child-elements"></a>Elementi figlio
 No.

### <a name="parent-elements"></a>Elementi padre

|Elemento|Descrizione|
|-------------|-----------------|
|[Elemento Strings](../extensibility/strings-element.md)|Raggruppa elementi di `ButtonText` testo, ad esempio e `CommandName`.|
