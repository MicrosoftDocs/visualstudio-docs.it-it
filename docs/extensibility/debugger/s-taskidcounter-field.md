# <a name="s_taskidcounter-field"></a>Campo s_taskIdCounter
Identificatore disponibile successivo per un <xref:System.Threading.Tasks.Task> oggetto .

 **Spazio dei nomi:** <xref:System.Threading.Tasks?displayProperty=fullName>

 **Assembly:** mscorlib (in mscorlib.dll)

 Poiché non è possibile accedere a questo membro interno dal .NET Framework, in Common Intermediate Language (CIL) viene fornita la sintassi seguente.

## <a name="syntax"></a>Sintassi

```csharp
.field static assembly int32 s_taskIdCounter
```
