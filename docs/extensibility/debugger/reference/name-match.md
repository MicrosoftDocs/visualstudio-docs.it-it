# <a name="namematch"></a>NAME_MATCH
Seleziona l'opzione maiuscole per i nomi corrispondenti.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```
