# <a name="pending_bp_state_flags"></a>PENDING_BP_STATE_FLAGS
Specifica i flag di stato del punto di interruzione in sospeso.

## <a name="syntax"></a>Sintassi

```cpp
enum enum_PENDING_BP_STATE_FLAGS { 
   PBPSF_NONE        = 0x0000,
   PBPSF_VIRTUALIZED = 0x0001
};
typedef DWORD PENDING_BP_STATE_FLAGS;
```

```csharp
public enum enum_PENDING_BP_STATE_FLAGS { 
   PBPSF_NONE        = 0x0000,
   PBPSF_VIRTUALIZED = 0x0001
};
```

## <a name="fields"></a>Campi
 `PBPSF_NONE`Segnaposto.

 `PBPSF_VIRTUALIZED`Specifica un punto di interruzione virtualizzato in sospeso, che deve essere associato ogni volta che viene caricato nuovo codice.
