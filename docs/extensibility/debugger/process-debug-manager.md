# <a name="process-debug-manager"></a>Gestione debug processi
Gestione debug processi (PDM) è un componente di Visual Studio che gestisce programmi e processi, rendendoli disponibili per il gestore di debug della sessione e i motori di debug.

 PDM gestisce tutti i processi di cui è possibile eseguire il debug. Per eseguire il debug, è necessario registrare un programma con PDM. Questa registrazione viene eseguita al momento dell'avvio del programma, tramite una porta o un motore di debug.
