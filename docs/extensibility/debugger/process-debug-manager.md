# <a name="process-debug-manager"></a>Gestione debug processo
Process Debug Manager (PDM) è un componente di Visual Studio che gestisce programmi e processi, rendendoli disponibili per la gestione del debug della sessione e per i motori di debug.

 Il PDM gestisce tutti i processi di cui è possibile eseguire il debug. Per eseguire il debug, è necessario registrare un programma con il PDM. Questa registrazione viene eseguita al momento dell'avvio del programma, tramite una porta o un motore di debug.
