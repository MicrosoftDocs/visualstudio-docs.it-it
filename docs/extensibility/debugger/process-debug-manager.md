# <a name="process-debug-manager"></a>Gestione debug processo
Il gestore di debug dei processi (PDM) Ã¨ un componente di Visual Studio che gestisce programmi e processi, rendendoli disponibili al gestore di debug della sessione e ai motori di debug.

 Il PDM gestisce tutti i processi che possono essere sottoposti a debug. Per eseguire il debug, un programma deve essere registrato con il PDM. Questa registrazione viene eseguita al momento dell'avvio del programma, da una porta o da un motore di debug.
