# <a name="objects"></a>Oggetti
Le informazioni di riferimento seguenti vengono illustrati gli oggetti utilizzati per creare un'istanza dell'editor di testo principale.

## <a name="in-this-section"></a>Contenuto della sezione
- [Oggetto VSCodeWindow](../extensibility/vscodewindow-object.md) descrive l'oggetto finestra di codice, una finestra del documento specializzata che può includere uno o più visualizzazioni di testo.

- [Oggetto VSCodeWindowManager](../extensibility/vscodewindowmanager-object.md) descrive l'oggetto manager finestra di codice che viene implementato dal servizio di linguaggio ed è responsabile della gestione delle aree di controllo.

- [Oggetto VSTextBuffer](../extensibility/vstextbuffer-object.md) descrive l'oggetto del buffer di testo, che rappresenta un flusso di testo Unicode, i dati per un file.

- [Oggetto VSTextView](../extensibility/vstextview-object.md) descrive la visualizzazione di testo, una finestra che consente agli utenti di visualizzare e modificare testo Unicode del buffer di testo.
