# <a name="net-compiler-platform-package-version-reference"></a>Informazioni di riferimento sulla versione del pacchetto della piattaforma del compilatore .NET.NET compiler platform package version reference

Nella tabella seguente vengono illustrate le versioni del [pacchetto della piattaforma del compilatore .NET (Roslyn)](https://www.nuget.org/packages/Microsoft.Net.Compilers/) supportate per le diverse versioni di Visual Studio.The following table shows which .NET compiler platform (Roslyn) package versions are supported for different versions of Visual Studio.

Ad esempio, per garantire che l'analizzatore personalizzato funzioni in tutte le versioni di Visual Studio 2017, deve essere destinato alla versione 2.0 di Microsoft.Net.Compilers.As an example, to ensure that your custom analyzer works on all versions of Visual Studio 2017, it should target version 2.0 of Microsoft.Net.Compilers.

| Versione del pacchetto Roslyn | Versione minima di Visual Studio supportata |
| - | - |
| 3.x | Visual Studio 2019 |
| 2.10.0 | Visual Studio 2017 versione 15.9 |
| 2.9.0 | Visual Studio 2017 versione 15.8 |
| 2.8.2 | Visual Studio 2017 versione 15.7 |
| 2.7.0 | Visual Studio 2017 versione 15.6 |
| 2.6.1 | Visual Studio 2017 versione 15.5 |
| 2.4.0 | Visual Studio 2017 versione 15.4 |
| 2.3.2 | Visual Studio 2017 versione 15.3 |
| 2.2.0 | Visual Studio 2017 versione 15.2 |
| 2.1.0 | Visual Studio 2017 versione 15.1 |
| 2.0.0 | Visual Studio 2017 RTM |
| 1.3.2 | Aggiornamento di Visual Studio 2015 3 |
| 1.2.2 | Aggiornamento di Visual Studio 2015 2 |
| 1.1.1 | Aggiornamento di Visual Studio 2015 1 |
| 1.0.1 | Visual Studio 2015 RTM |

> [!TIP]
> Per i pacchetti Roslyn in cui la versione minima supportata di Visual Studio è una versione di Visual Studio 2017, tutte le versioni di Visual Studio 2019 sono supportate anche perché sono state successive.
