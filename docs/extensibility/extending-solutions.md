# <a name="extend-solutions"></a>Estendi soluzioni
Una soluzione è un raggruppamento di uno o più progetti che interagiscono tra loro per creare un'applicazione.

## <a name="in-this-section"></a>Contenuto della sezione
 Le procedure dettagliate seguenti illustrano alcuni modi in cui è possibile estendere le soluzioni.

1. [Estendere il filtro Esplora soluzioni](../extensibility/extending-the-solution-explorer-filter.md) spiega come filtrare le informazioni visualizzate nel Esplora soluzioni.

2. [Gestire il caricamento di progetti in una soluzione](../extensibility/managing-project-loading-in-a-solution.md) illustra come gestire il caricamento di progetti per migliorare le prestazioni quando un utente carica una soluzione.
