# <a name="list-of-available-services"></a>Elenco di servizi disponibili

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]e Visual Studio SDK supporta i servizi seguenti. Alcuni pacchetti offrono i propri servizi che non sono elencati qui, ad esempio i servizi di linguaggio non dispongono di un singolo GUID del servizio. È necessario utilizzare il nome della lingua per trovare il GUID del servizio di linguaggio nel Registro di sistema.

Usare i GUID del servizio elencati di seguito o ottenuti da un'altra origine (ad esempio, i servizi di linguaggio) per ottenere l'interfaccia primaria o le interfacce visualizzate con ogni servizio.
