# <a name="list-of-available-services"></a>Elenco di servizi disponibili

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] e Visual Studio SDK supportano i servizi seguenti. Alcuni pacchetti offrono i propri servizi che non sono elencati qui. ad esempio, i servizi di linguaggio non hanno un solo GUID del servizio. È necessario utilizzare il nome della lingua per trovare il GUID del servizio di linguaggio nel registro di sistema.

Usare i GUID del servizio elencati di seguito o ottenuti da un'altra origine (ad esempio, servizi di linguaggio) per ottenere l'interfaccia o le interfacce primarie visualizzate con ogni servizio.
