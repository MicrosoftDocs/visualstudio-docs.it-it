# <a name="source-control-vspackage-features"></a>Funzionalità dei pacchetti VSPackage di controllo del codice sorgente
In questa sezione vengono descritte le varie funzionalità di un controllo del codice sorgente VSPackage.This section describes the various features of a source control VSPackage. Vengono descritti i dettagli di registrazione e selezione per tale VSPackage e vengono illustrate tre delle principali funzionalità correlate al controllo del codice sorgente: gestione degli eventi QEQS (Query-Edit Query-Save), sostituzione dei glifi e dell'interfaccia utente personalizzata per le funzioni del controllo del codice sorgente.

## <a name="in-this-section"></a>Contenuto della sezione
- [Registrazione e selezione](../../extensibility/internals/registration-and-selection-source-control-vspackage.md)

 Descrive i meccanismi di registrazione e selezione dei pacchetti.

- [Query Edit Query Save](../../extensibility/internals/query-edit-query-save-source-control-vspackage.md)

 Viene illustrato il ruolo di Query-Edit Query-Save eventi e come vengono gestiti dal controllo del codice sorgente VSPackage.

- [Controllo dei glifi](../../extensibility/internals/glyph-control-source-control-vspackage.md)

 Descrive i livelli del controllo glifo e come implementarli.

- [Interfaccia utente personalizzata](../../extensibility/internals/custom-user-interface-source-control-vspackage.md)

 Delinea gli elementi dell'interfaccia utente che un controllo del codice sorgente VSPackage può specificare.
