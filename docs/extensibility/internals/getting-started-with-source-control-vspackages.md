# <a name="get-started-with-source-control-vspackages"></a>Introduzione al controllo del codice sorgente VSPackageGet started with source control VSPackages

Un controllo del codice sorgente VSPackage è un'alternativa più avanzata al plug-in del controllo del codice sorgente. Per ulteriori informazioni sui plug-in del controllo del codice sorgente, vedere [Creare un plug-in](../../extensibility/internals/creating-a-source-control-plug-in.md)del controllo del codice sorgente. Un controllo del codice sorgente VSPackage consente il controllo completo sul modello di controllo [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] del codice sorgente, funzionalità e interfaccia utente (UI) e si integra nell'ambiente come un VSPackage.
