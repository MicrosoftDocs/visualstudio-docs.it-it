# <a name="web-projects"></a>Progetti Web
Un sistema di progetto Web è un sistema di progetto che crea progetti Web. I progetti Web a loro volta creano applicazioni Web. È possibile programmare una pagina Web [!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)] utilizzando [!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)]i linguaggi di programmazione tradizionali, ad esempio o , per raccogliere ed elaborare informazioni da un utente, archiviarle in un database e così via. Le pagine Web con codice associato sono denominate pagine Web intelligenti.
