# <a name="create-uml-modeling-projects-and-diagrams"></a>Creare diagrammi e progetti di modellazione UML

A partire da Visual Studio 2017, le finestre di progettazione UML sono state rimosse da Visual Studio. Per altre informazioni, vedere questo [post di blog](https://blogs.msdn.microsoft.com/visualstudioalm/2016/10/14/uml-designers-have-been-removed-layer-designer-now-supports-live-architectural-analysis/).

Se è necessario comunque utilizzare UML, è possibile usare Visual Studio 2015 Enterprise edition, che è compatibile side-by-side con versioni successive di Visual Studio.

Per informazioni sull'uso di UML in Visual Studio 2015, vedere:

* [Analizzare e modellare l'architettura](https://msdn.microsoft.com/library/57b85fsc%28v=vs.140%29.aspx)
* [Creare modelli per l'app](create-models-for-your-app.md)
* [Usare modelli nel processo di sviluppo](use-models-in-your-development-process.md)
* [Convalidare il sistema durante lo sviluppo](validate-your-system-during-development.md)
* [Estendere modelli e diagrammi UML](extend-uml-models-and-diagrams.md)
