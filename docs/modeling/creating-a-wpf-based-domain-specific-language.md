Informazioni ed esempi relativi a questa funzionalità sono disponibili nel sito Web degli strumenti di visualizzazione e modellazione di Visual Studio all'indirizzo [http://go.microsoft.com/fwlink/?LinkId=186128](https://code.msdn.microsoft.com/Visualization-and-Modeling-313535db)
