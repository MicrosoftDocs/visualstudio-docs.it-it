# <a name="control-flow-activity-designers"></a>ActivityDesigner Flusso di controllo
[!INCLUDE[wfd1](../includes/wfd1-md.md)] include una serie di attività fornite dal sistema che è possibile usare per la costruzione dei flussi di lavoro. Contenuto della sezione sono presentate le attività fornite dal sistema usate per controllare il flusso all'interno di un flusso di lavoro. Negli argomenti riportati di seguito vengono illustrate queste attività e vengono fornite indicazioni sulle relative modalità di utilizzo.

## <a name="in-this-section"></a>Contenuto della sezione
 [DoWhile](../workflow-designer/dowhile-activity-designer.md) Esegue almeno una volta l'attività contenuta nel corpo, fino a quando una condizione specificata non restituisce **true**.

 [Foreach \<T> ](foreach-t-activity-designer.md) Esegue l'attività contenuta nel corpo per ogni elemento in una raccolta specificata.

 [Se](../workflow-designer/if-activity-designer.md) Valuta una condizione ed esegue un'attività in base ai risultati della valutazione.

 In [parallelo](../workflow-designer/parallel-activity-designer.md) Esegue contemporaneamente una raccolta di attività figlio.

 [ActivityDesigner ParallelForEach \<T> ](../workflow-designer/parallelforeach-t-activity-designer.md) Enumera gli elementi di una raccolta ed esegue un'istruzione incorporata per ogni elemento della raccolta in parallelo.

 [Seleziona](../workflow-designer/pick-activity-designer.md) Esegue uno dei diversi rami in risposta a un evento che fornisce il controllo di flusso basato su eventi.

 [PickBranch](../workflow-designer/pickbranch-activity-designer.md) Fornisce un percorso potenziale di esecuzione all'interno di un' <xref:System.Activities.Statements.Pick> attività.

 [Sequenza](../workflow-designer/sequence-activity-designer.md) Contiene una raccolta ordinata di attività figlio eseguite nell'ordine.

 [Passa \<T> ](switch-t-activity-designer.md) a Valuta un'espressione specificata ed esegue l'attività da una raccolta di attività la cui chiave associata corrisponde al valore ottenuto dalla valutazione.

 [Mentre](../workflow-designer/while-activity-designer.md) Esegue l'attività contenuta nel corpo mentre una condizione specificata restituisce **true**.
