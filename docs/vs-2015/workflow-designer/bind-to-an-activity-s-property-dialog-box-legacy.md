# <a name="bind-to-an-activity39s-property-dialog-box-legacy"></a>Eseguire l'associazione a un'attività&#39;finestra di dialogo Proprietà (Legacy)
Questo argomento viene descritto come usare il **associare alla proprietà dell'attività** nella finestra di dialogo legacy [!INCLUDE[wfd1](../includes/wfd1-md.md)]. Usare la [!INCLUDE[wfd2](../includes/wfd2-md.md)] legacy quando è necessario fare riferimento a [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] o [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)].  
  
 Un tipo di istanza di proprietà di dipendenza può essere associato alla proprietà pubblica di un'altra attività o evento. Per altre informazioni sull'associazione di attività, vedere [usando le proprietà di dipendenza](http://go.microsoft.com/fwlink?LinkID=65007).  
  
 Si seleziona una proprietà da associare usando la **associare a una proprietà di attività** nella finestra di dialogo. Si apre questa finestra di dialogo facendo clic sui puntini di sospensione **[...]**  alla fine della casella di testo della proprietà selezionata nel **proprietà** finestra, oppure facendo clic sull'icona di punto esclamativo blu visualizzata accanto al nome di proprietà nel Visualizzatore proprietà.  
  
 La tabella seguente descrive gli elementi dell'interfaccia utente di **associare a una proprietà di attività** nella finestra di dialogo.  
  
|Elemento dell'interfaccia utente|Descrizione|  
|----------------|-----------------|  
|**Eseguire l'associazione a un membro esistente**|Selezionare un membro che si desidera associare nel riquadro di visualizzazione albero. Il riquadro al di sotto della visualizzazione albero visualizza un messaggio che indica se il membro è un tipo valido per l'associazione o no. Fare clic su **OK** da associare al membro valido selezionato.|  
|**Eseguire l'associazione a un nuovo membro**|Creare un campo o una proprietà per il membro nuovo da associare. Immettere un **nuovo nome del membro**. Scegliere se si desidera creare una proprietà di dipendenza o un campo pubblico selezionando **campo creare** oppure **crea proprietà**. Fare clic su **OK** per creare il nuovo membro.|
