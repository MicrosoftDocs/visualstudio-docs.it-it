# <a name="flowchart-activity-designers"></a>ActivityDesigner Diagramma di flusso
[!INCLUDE[wfd1](../includes/wfd1-md.md)] include alcune attività fornite dal sistema che è possibile usare per la costruzione di flussi di lavoro. Contenuto della sezione sono presentate le attività fornite dal sistema usate per gestire diagrammi di flusso all'interno di un flusso di lavoro. Negli argomenti riportati di seguito vengono illustrate queste attività e vengono fornite indicazioni sulle relative modalità di utilizzo.

## <a name="in-this-section"></a>Contenuto della sezione
 [Diagramma](../workflow-designer/flowchart-activity-designer.md) **di flusso utilizzato per creare flussi di lavoro che definiscono e gestiscono controlli di flusso complessi.**  [FlowDecision](../workflow-designer/flowdecision-activity-designer.md) Fornisce un ramo per il flusso di controllo in una delle due alternative, a seconda che venga soddisfatta una condizione specificata.

 [FlowSwitch \<T >](../workflow-designer/flowswitch-t-activity-designer.md) Fornisce la diramazione per il flusso di controllo in base al criterio di corrispondenza quando sono necessari più di due rami alternativi.
