# <a name="wpf-data-binding-using-linq-to-xml-example"></a>Esempio di data binding WPF tramite LINQ to XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [WPF Data Binding Using LINQ to XML esempio](https://docs.microsoft.com/visualstudio/designers/wpf-data-binding-using-linq-to-xml-example).
