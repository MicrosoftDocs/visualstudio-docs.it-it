# <a name="visual-language-dictionary-for-visual-studio"></a>Dizionario del linguaggio visivo per Visual Studio
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Visual Studio usa un linguaggio visuale comune per illustrare i concetti che costituiscono i blocchi predefiniti dell'iconografia e di altre immagini. Quando si progettano nuove icone, utilizzare questo dizionario per identificare i concetti che si desidera trasferire nella propria immagine.

 **Linee guida generali**

- Questi concetti visivi devono essere usati in modo coerente con la definizione del dizionario. In caso contrario, si rischia di confondere gli utenti, che prevedono elementi comuni con significati coerenti.

- Scegliere solo i concetti più rilevanti o importanti da usare nelle icone, in modo che le immagini siano nitide e facili da distinguere.

- Usare solo uno o due concetti in ogni immagine.

- Per altre strategie di progettazione delle icone, vedere [icona progettazione](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign).
