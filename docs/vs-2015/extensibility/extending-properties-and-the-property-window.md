# <a name="extending-properties-and-the-property-window"></a>Estensione delle proprietà e della finestra Proprietà
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La finestra **Proprietà** viene utilizzata per visualizzare e modificare le proprietà e gli eventi della fase di progettazione degli oggetti selezionati.  
  
 Quando si seleziona un oggetto in [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] , le proprietà pubbliche di tale oggetto vengono visualizzate nella finestra **Proprietà** . Per ulteriori informazioni, vedere [Cenni preliminari sulla finestra Proprietà](../extensibility/internals/properties-window-overview.md).
