# <a name="getting-started-with-source-control-vspackages"></a>Introduzione ai pacchetti VSPackage di controllo del codice sorgente
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Un pacchetto VSPackage di controllo del codice sorgente è un'alternativa più avanzata al plug-in del controllo del codice sorgente. (Per altre informazioni sui plug-in del controllo del codice sorgente, vedere [creazione di un plug-in del controllo del codice](../../extensibility/internals/creating-a-source-control-plug-in.md) sorgente). Un pacchetto VSPackage di controllo del codice sorgente consente il controllo completo sul modello di controllo del codice sorgente, sulle funzionalità e sull'interfaccia utente e si integra nell' [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] ambiente come un pacchetto VSPackage.
