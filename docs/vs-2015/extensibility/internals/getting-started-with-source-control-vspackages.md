# <a name="getting-started-with-source-control-vspackages"></a>Introduzione ai pacchetti VSPackage di controllo del codice sorgente
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Un VSPackage di controllo del codice sorgente è un'alternativa più avanzata per il plug-in del controllo del codice sorgente. (Vedere [creazione di un plug-in controllo sorgente](../../extensibility/internals/creating-a-source-control-plug-in.md) controllare plug-in per altre informazioni sull'origine.) Controllo completo sul modello di controllo di origine, funzionalità e interfaccia utente (UI) consente a un VSPackage di controllo del codice sorgente e integra il [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] ambiente come un pacchetto VSPackage.
