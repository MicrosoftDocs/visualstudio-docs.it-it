# <a name="color-theming-tools"></a>Strumenti dei temi di colore
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [strumenti dei temi di colore](https://docs.microsoft.com/visualstudio/extensibility/internals/color-theming-tools).  
  
il SDK di Visual Studio offre diversi strumenti che consentono di creare e gestire i temi di colore.  
  
-   Il [dell'Editor dei colori VSIX](../../extensibility/internals/vsix-color-editor.md) strumento è possibile creare e modificare i colori personalizzati per Visual Studio.  
  
-   Il [compilatore dei colori VSIX](../../extensibility/internals/vsix-color-compiler.md) lo strumento converte file con estensione XML tema di Visual Studio esistenti in pkgdef per l'estensione.
