# <a name="extending-user-settings-and-options"></a>Estensione di opzioni e impostazioni utente
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

In Visual Studio sono disponibili due modalità principali per gestire le preferenze dell'utente. Le pagine **Strumenti/Opzioni** consentono agli utenti di impostare valori propri per molti tipi diversi di impostazioni, sia generali che specifiche del linguaggio. Le impostazioni utente consentono agli utenti di specificare le proprietà di configurazione generali di Visual Studio. Per ulteriori informazioni sulle pagine opzioni, vedere [Opzioni e pagine](../extensibility/internals/options-and-options-pages.md)opzioni. Per ulteriori informazioni sulle impostazioni utente, vedere [supporto per le impostazioni utente](../extensibility/internals/support-for-user-settings.md).
