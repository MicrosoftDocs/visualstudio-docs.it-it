# <a name="mparent-field"></a>Campo m_parent
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [campo m_parent](https://docs.microsoft.com/visualstudio/extensibility/debugger/m-parent-field).  
  
Il campo sottostante per il <xref:System.Threading.Tasks.Task?displayProperty=fullName> proprietà padre.  
  
 **Namespace:** <xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **Assembly:** mscorlib (in mscorlib. dll)  
  
 Poiché è possibile accedere a questo membro interno da .NET Framework, la sintassi seguente viene fornita in comune Intermediate Language (CIL).  
  
## <a name="syntax"></a>Sintassi  
  
```  
.field private initonly class System.Threading.Tasks.Task m_parent  
```
