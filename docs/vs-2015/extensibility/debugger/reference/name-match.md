# <a name="namematch"></a>NAME_MATCH
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [NAME_MATCH](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/name-match).  
  
Seleziona l'opzione maiuscole per i nomi corrispondenti.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp#  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>Membri  
 nmNone  
 Non è stata specificata alcuna opzione.  
  
 nmCaseSensitive  
 Indica che i nomi per cui trovare una corrispondenza tra maiuscole e minuscole.  
  
 nmCaseInsensitive  
 Indica che i nomi per cui trovare una corrispondenza non sono tra maiuscole e minuscole.
