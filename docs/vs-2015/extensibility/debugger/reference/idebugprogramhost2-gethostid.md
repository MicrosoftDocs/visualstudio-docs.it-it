# <a name="idebugprogramhost2gethostid"></a>IDebugProgramHost2::GetHostId
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [IDebugProgramHost2::GetHostId](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugprogramhost2-gethostid).  
  
Ottiene l'identificatore di processo del processo che ospita questo programma.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp#  
HRESULT GetHostId(   
   AD_PROCESS_ID* pdwId  
);  
```  
  
```csharp  
int GetHostId(   
   AD_PROCESS_ID[] pdwId  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `pdwId`  
 [in, out] Un' [AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md) struttura compilata con le informazioni sull'identificatore di processo.  
  
## <a name="return-value"></a>Valore restituito  
 Se ha esito positivo, restituisce `S_OK`; in caso contrario, restituisce un codice di errore.
