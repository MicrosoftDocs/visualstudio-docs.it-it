# <a name="visual-studio-debugging-samples"></a>Esempi di debug di Visual Studio
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Due esempi illustrano come estendere il debugger di Visual Studio usando [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)] .  
  
- L'esempio di analizzatore di espressioni del debugger, per i linguaggi gestiti, è incluso in [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)] . Nella [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)] finestra di fare clic sulla scheda **esempi** , fare clic su **tutto** o su **C#** nell'elenco di lingue, quindi fare clic su **debugger**.  
  
- L'esempio del motore di debug di Visual Studio, per le lingue native, è disponibile in MSDN Code Gallery. Per altre informazioni, vedere l'esempio in GitHub: [esempio di motore di debug di Visual Studio](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/master/Visual%20Studio%20Product%20Team/Visual%20Studio%20Debug%20Engine%20Sample).
