# <a name="enabling-the-azure-sdk"></a>Abilitazione di Azure SDK
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Azure SDK per .NET semplifica la creazione, la configurazione, la compilazione, il debug, l'esecuzione e la creazione di pacchetti di servizi e applicazioni Web scalabili in Azure.  
  
 È possibile scaricare Azure SDK dalla finestra di dialogo **nuovo progetto** .  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Per abilitare Azure SDK  
  
1. Scegliere Crea nuovo progetto dal menu **file** . Fare clic su **nuovo** e quindi su **progetto**.  
  
2. Espandere il nodo per **Visual Basic** o **C#** e quindi fare clic su **cloud**.  
  
3. Nel riquadro centrale selezionare **ottenere Azure SDK per .NET**, quindi fare clic su **OK**.  
  
     Seguire le istruzioni del programma di installazione.  
  
    > [!NOTE]
    > Per installare gli strumenti di Azure, è necessario aver installato Internet Information Services 7,0 e i componenti di sviluppo dell'applicazione ASP.NET. Se questi componenti non sono installati, il programma di installazione degli strumenti di Azure richiede di installarli.
