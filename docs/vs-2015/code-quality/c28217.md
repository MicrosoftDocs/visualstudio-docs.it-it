# <a name="c28217"></a>C28217
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28217](https://docs.microsoft.com/visualstudio/code-quality/c28217).  
  
avviso C28217: per funzione, il numero di parametri per l'annotazione non corrisponde a quello trovato nel file  
  
 Le annotazioni nella riga corrente e nella riga all'interno del messaggio non sono coerenti. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto per essere visualizzato nell'utilizzo normale.
