# <a name="c6997"></a>C6997
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C6997](https://docs.microsoft.com/visualstudio/code-quality/c6997).  
  
Avviso 6997: le annotazioni in questo percorso non hanno significato e verranno ignorate.  
  
 Le annotazioni non possono essere applicate a `extern "C" {...}`. Applicare le annotazioni a un oggetto specifico.
