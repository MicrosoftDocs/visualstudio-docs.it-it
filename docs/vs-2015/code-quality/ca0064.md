# <a name="ca0064"></a>CA0064

Non è stata eseguita alcuna analisi perché il set di regole specificato non contiene regole FxCop.

Questo avviso può verificarsi in una delle situazioni seguenti:

- Se si verifica questo avviso insieme a avvisi di CA0063, si è verificato un problema durante il caricamento del file di RuleSet. Per ulteriori informazioni, vedere la documentazione di [CA0063](ca0063.md) .

- Se si verifica questo errore insieme agli avvisi di CA0062, è molto probabile che l'analisi del codice non sia in grado di trovare l'assembly o gli assembly che contengono le regole specificate dal set di regole. Per ulteriori informazioni, vedere la documentazione di [CA0062](ca0062.md) .

- In caso contrario, questo avviso si verifica in genere quando il set di regole è vuoto o tutte le regole abilitate in un set di regole figlio sono state disabilitate. Usare l' [Editor set di regole](../code-quality/working-in-the-code-analysis-rule-set-editor.md) di Visual Studio per abilitare alcune regole nel set di regole.
