# <a name="c6287"></a>C6287
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C6287: codice ridondante: le sottoespressioni di sinistra e destra sono identiche  
  
 Questo avviso indica che è stato rilevato un elemento ridondante in un'espressione.  
  
 È difficile giudicare la gravità del problema senza esaminare il codice. Un test duplicato è innocuo, ma le conseguenze dell'eliminazione del secondo test possono essere gravi. Il codice deve essere controllato per assicurarsi che un test non sia stato omesso.  
  
## <a name="example"></a>Esempio  
 Il codice seguente genera questo avviso:  
  
```  
void f(int x)  
{  
  if ((x == 1) && (x == 1))   
  {  
    //logic   
  }  
  if ((x != 1) || (x != 1))  
  {  
    //logic  
  }  
}  
```  
  
 Nel codice seguente vengono illustrati diversi metodi per correggere questo avviso:  
  
```  
void f(int x, int y)  
{  
  /* Remove the redundant sub-expression: */  
  if (x == 1)   
  {  
     // logic   
  }  
  if (x != 1)   
  {  
    // logic  
  }  
  /* or test the missing variable: */  
  if ((x == 1) && (y == 1))  
  {  
     // logic  
  }  
  if ((x != 1) || (y != 1))  
  {  
     // logic  
  }  
}  
```
