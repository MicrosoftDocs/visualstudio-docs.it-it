# <a name="c6504"></a>C6504
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C6504: annotazione non valida: la proprietà può essere utilizzata solo con valori di puntatore, puntatore a membro o tipo di matrice  
  
 Questo avviso indica l'utilizzo di una proprietà in un tipo di dati non compatibile. Per ulteriori informazioni sui tipi di dati supportati dalle proprietà, vedere [Proprietà annotazione](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Esempio  
 Nel codice riportato di seguito l'avviso viene generato perché la proprietà `_Null_` non può essere utilizzata per il tipo di dati riferimento.  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_ Point& pt)  
{  
    // code ...  
}  
  
```  
  
 Per correggere il problema, usare il codice seguente:  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_  Point* pt)  
{  
    // code ...  
}  
```  
  
 Il codice difettoso illustrato in precedenza genera anche l'avviso [C6516](../code-quality/c6516.md) perché i conflitti di proprietà hanno generato un'annotazione non valida.
