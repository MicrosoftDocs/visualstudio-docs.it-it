# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Avviso C28223: Token o identificatore previsto mancante per l'annotazione, parametro  
  
 Questo avviso indica che un parametro in un'annotazione che dovrà essere un identificatore, e si è verificato un altro simbolo. Ciò indica in genere una macro di codice errato dell'annotazione. L'utilizzo delle parole chiave C o C++ in questa posizione causerà questo errore.
