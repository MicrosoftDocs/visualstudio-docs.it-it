# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28211: Previsto nome contesto statico per SAL_context  
  
 Questo avviso indica che l'operando di `_Static_context_` annotazione deve essere il nome di un contesto aggiuntivo definito dello strumento. Questo avviso non dovrebbe mai verificarsi nell'utilizzo normale.
