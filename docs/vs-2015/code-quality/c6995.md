# <a name="c6995"></a>C6995
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C6995: non è stato possibile salvare il file di Log XML  
  
 Questo avviso indica che lo strumento di analisi del codice non è possibile creare il log di difetti, che è l'output di analisi del codice.  
  
 Questo errore potrebbe indicare un errore del disco o indicare che non hai le autorizzazioni per creare un file nella directory specificata.
