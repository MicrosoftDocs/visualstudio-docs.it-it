# <a name="c1254"></a>C1254
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C1254](https://docs.microsoft.com/visualstudio/code-quality/c1254).  
  
avviso C1254: mancata corrispondenza delle versioni di plug-in: versione non corrisponde alla versione del driver PREfast  
  
 Lo strumento di analisi del codice genera questo avviso quando si verifica un errore interno con la versione del plug-in, non nel codice analizzato.
