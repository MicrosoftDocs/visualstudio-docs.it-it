# <a name="ca0057"></a>CA0057

Le violazioni non è possibile eseguire il mapping al set corrente di destinazioni e le regole.

Questo errore si verifica se FxCopCmd.exe viene richiamato manualmente con un file di progetto o un report XML contenente le violazioni che non è possibile eseguire il mapping al set corrente di destinazioni e le regole importato.

Si verifica solo se l'analisi sono fatta utilizzando la versione dello strumento da riga di comando.
