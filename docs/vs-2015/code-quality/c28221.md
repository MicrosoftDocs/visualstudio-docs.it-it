# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28221: espressione String prevista per il parametro nell'annotazione  
  
 Questo avviso indica che è previsto un parametro in un'annotazione deve essere una stringa, e si è verificato un altro tipo. Questo in genere indica una macro di annotazione codice errato e non deve essere visualizzato nell'utilizzo normale.
