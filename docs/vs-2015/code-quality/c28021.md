# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28021: il parametro \<param > annotato con \<anno > deve essere un puntatore  
  
 Questo avviso viene segnalato quando l'oggetto annotato non è un tipo di puntatore. Questa annotazione non può essere utilizzata con `void` o tipi integrali.
