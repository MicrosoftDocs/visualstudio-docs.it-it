# <a name="c28287"></a>C28287
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28287](https://docs.microsoft.com/visualstudio/code-quality/c28287).  
  
avviso C28287: per la funzione, errore di sintassi in at\_annotazione () (nome parametro non riconosciuto)  
  
 Lo strumento di analisi del codice restituisce questo avviso quando la `SAL_at` (`__drv_at`) viene utilizzata l'annotazione e l'espressione del parametro non può essere interpretato nel contesto corrente. Potrebbe trattarsi di tramite un parametro con errori di ortografia o nome del membro o un errore di ortografia di "return" o "this" parole chiave.
