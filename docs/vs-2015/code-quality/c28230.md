# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28230](https://docs.microsoft.com/visualstudio/code-quality/c28230).  
  
avviso C28230: il tipo del parametro non ha membro.  
  
 Questo avviso indica che un argomento di un'annotazione tenta di accedere a un `struct`, `class`, o `union` e il membro denominato non esiste.
