# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28230: il tipo di parametro non ha membro.  
  
 Questo avviso indica che un argomento di un'annotazione tenta di accedere a `struct` , `class` o `union` e il membro denominato non esiste.
