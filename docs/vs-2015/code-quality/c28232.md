# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28232](https://docs.microsoft.com/visualstudio/code-quality/c28232).  
  
avviso C28232: _Pre\_, post\_, o Deref\_ non applicato ad alcuna annotazione  
  
 Questo avviso indica che un `_Pre_`, `_Post_`, o `_Deref_` operatore appare in un'espressione di annotazione senza un'annotazione funzionale successive; è stato ignorato il modificatore, ma è presente un'annotazione di codice errato.
