# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28020](https://docs.microsoft.com/visualstudio/code-quality/c28020).  
  
avviso C28020: l'espressione \<expr > non è true per questa chiamata  
  
 Questo avviso viene segnalato quando il _Satisfies\_ espressione elencata non è true. Spesso è presente un parametro errato.  
  
 Se questo avviene in una dichiarazione di funzione, le annotazioni indicano una condizione di comunicazione Impossibile.
