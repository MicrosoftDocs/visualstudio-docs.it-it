# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28020: l'espressione \<expr > non è true per questa chiamata  
  
 Questo avviso viene segnalato quando le \_Satisfies\_ espressione elencata non è true. Spesso è presente un parametro errato.  
  
 Se questo avviene in una dichiarazione di funzione, le annotazioni indicano una condizione di comunicazione Impossibile.
