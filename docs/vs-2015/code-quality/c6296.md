# <a name="c6296"></a>C6296
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C6296: Ciclo for: Il corpo del ciclo eseguito solo una volta  
  
 Questo avviso indica che un ciclo for non funzionino come previsto. Quando l'indice Ã¨ senza segno e un ciclo di conteggi da zero, il cui corpo viene eseguito una sola volta.  
  
## <a name="example"></a>Esempio  
 Il codice seguente genera questo avviso:  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 Per risolvere questo problema, usare il codice seguente:  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
