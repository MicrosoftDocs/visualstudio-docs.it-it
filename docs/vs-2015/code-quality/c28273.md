# <a name="c28273"></a>C28273
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28273](https://docs.microsoft.com/visualstudio/code-quality/c28273).  
  
avviso C28273: per la funzione, le informazioni non sono coerenti con la dichiarazione di funzione  
  
 Questo avviso indica un errore nelle annotazioni, non nel codice analizzato. Le annotazioni visualizzate in una definizione di funzione sono coerenti con quelli visualizzati in una dichiarazione. Le due annotazioni devono essere risolti in modo che corrispondano.
