# <a name="c28196"></a>C28196
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28196](https://docs.microsoft.com/visualstudio/code-quality/c28196).  
  
avviso C28196: il requisito non soddisfatto. (L'espressione non restituisce true.)  
  
 L'avviso indica che il prototipo di funzione per la funzione analizzata è `__notnull`, `__null` o `__drv_valueIs` su un parametro di `_Out_` o il valore restituito, ma il valore restituito è coerente con tale annotazione.
