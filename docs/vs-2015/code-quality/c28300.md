La versione più recente di questo argomento è reperibile in [C28300](https://docs.microsoft.com/visualstudio/code-quality/c28300).  
  
avviso C28300: < nome_parametro >: operandi dell'espressione di tipi incompatibili per l'operatore < nome_operatore >  
  
 Questo avviso viene segnalato quando i tipi degli operandi in un parametro non sono compatibili con l'operatore.
