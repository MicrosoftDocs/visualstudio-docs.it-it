# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28214: i nomi di parametro di attributo devono essere p1... p9  
  
 Questo avviso indica che quando si crea una dichiarazione di annotazione, i nomi dei parametri sono limitati a p1... p9. Questo avviso non dovrebbe mai verificarsi nell'utilizzo normale.
