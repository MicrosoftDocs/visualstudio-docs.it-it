# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28227](https://docs.microsoft.com/visualstudio/code-quality/c28227).  
  
Avviso C28227: L'annotazione non può essere anche PrimOp nella dichiarazione precedente  
  
 Questo avviso indica che l'annotazione denominata è dichiarato come un'annotazione normale e anche in precedenza è stato dichiarato come un PrimOp. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto affinché sia disponibile in uso tipico.
