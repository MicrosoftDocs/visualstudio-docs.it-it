# <a name="c28290"></a>C28290
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28290: l'annotazione per la funzione contiene un numero di riferimenti esterni maggiore del numero di parametri effettivi  
  
 Lo strumento di analisi codice genera questo avviso quando l'annotazione per la funzione contiene un numero di riferimenti esterni maggiore del numero di parametri effettivi.
