# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28226](https://docs.microsoft.com/visualstudio/code-quality/c28226).  
  
Avviso C28226: Annotazione non può essere anche un PrimOp (nella dichiarazione corrente)  
  
 Questo avviso indica che l'annotazione denominato viene dichiarato come un PrimOp e anche in precedenza è stato dichiarato come un'annotazione di tipo normale. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto per essere visualizzato nell'utilizzo normale.
