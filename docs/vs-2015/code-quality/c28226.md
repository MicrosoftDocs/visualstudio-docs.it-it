# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Avviso C28226: l'annotazione non può essere anche un PrimOp (nella dichiarazione corrente)  
  
 Questo avviso indica che l'annotazione denominata viene dichiarata come PrimOp e che in precedenza è stata dichiarata come annotazione normale. Questa operazione non dovrebbe essere possibile se le macro standard vengono utilizzate per le annotazioni. Questo avviso non dovrebbe essere visualizzato nel normale utilizzo.
