# <a name="c28275"></a>C28275
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28275](https://docs.microsoft.com/visualstudio/code-quality/c28275).  
  
avviso C28275: il parametro per macro_value\_ è null  
  
 Questo avviso indica che si verifica un errore interno nel file del modello, non nel codice analizzato. Il *macroValue* funzione è stata chiamata senza parametri.
