# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28160: annotazione errore  
  
 Questo avviso viene segnalato quando un `__drv_error` annotazione è stata rilevata. Questa annotazione è utilizzata per contrassegnare le procedure di codifica che devono essere corretto e possono essere usate con un `__drv_when` annotazione per indicare specifiche combinazioni di parametri.
