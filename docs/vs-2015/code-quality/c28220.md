# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28220](https://docs.microsoft.com/visualstudio/code-quality/c28220).  
  
avviso C28220: prevista espressione Integer per l'annotazione il parametro nell'annotazione  
  
 Questo avviso indica che un parametro in un'annotazione deve essere un'espressione integer, e si è verificato un altro tipo. Ciò indica in genere una macro di codice errato dell'annotazione.
