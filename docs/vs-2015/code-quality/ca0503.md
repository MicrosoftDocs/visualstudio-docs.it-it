# <a name="ca0503"></a>CA0503

La proprietà è deprecata. Usare la proprietà di sostituzione.

CA0503 avviso viene generato quando una proprietà di analisi di codice deprecato contiene un valore nel file di configurazione compilazione o il progetto. Per risolvere questo problema, usare la proprietà specificata che lo sostituisce.
