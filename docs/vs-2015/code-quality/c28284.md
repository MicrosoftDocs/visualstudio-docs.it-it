# <a name="c28284"></a>C28284
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28284](https://docs.microsoft.com/visualstudio/code-quality/c28284).  
  
avviso C28284: per il simbolo, i predicati non sono attualmente supportati per i simboli non di funzione  
  
 Questo avviso indica che un'annotazione condizionale (predicato, `__drv_when`) è stato trovato in un valore diverso da una funzione.
