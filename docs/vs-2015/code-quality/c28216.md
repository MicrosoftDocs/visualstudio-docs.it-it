# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28216](https://docs.microsoft.com/visualstudio/code-quality/c28216).  
  
avviso C28216: il _Check_return\_ annotazione si applica soltanto a postcondizioni per il parametro di funzione specifica.  
  
 Il `_Check_return_` annotazione viene applicata in un contesto diverso da `__post`; potrebbe essere necessario un `__post` (o `__drv_out`) modificatore oppure può essere inserito in modo non corretto.
