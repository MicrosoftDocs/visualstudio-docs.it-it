# <a name="c6258"></a>C6258
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C6258: l'uso di TerminateThread non consente la pulizia corretta del thread.  
  
 Questo avviso indica che è stata rilevata una chiamata a TerminateThread.  
  
 TerminateThread è una funzione pericolosa da usare solo nei casi più estremi. Per ulteriori informazioni sui problemi associati alla chiamata TerminateThread, vedere questo argomento in MSDN Library: [TerminateThread](https://go.microsoft.com/fwlink/?LinkId=150233).  
  
### <a name="to-properly-terminate-threads"></a>Per terminare correttamente i thread  
  
1. Creare un oggetto evento usando la funzione `CreateEvent`.  
  
2. Creare i thread.  
  
3. Ogni thread monitora lo stato dell'evento chiamando la funzione `WaitForSingleObject`.  
  
4. Ogni thread termina la propria esecuzione quando l'evento è impostato sullo stato segnalato (`WaitForSingleObject` restituisce `WAIT_OBJECT_0`).  
  
   Per ulteriori informazioni, vedere uno degli argomenti seguenti:  
  
   [Terminazione di un thread](https://go.microsoft.com/fwlink/?LinkId=150234)  
  
   [WaitForSingleObject](https://go.microsoft.com/fwlink/?LinkId=150235)  
  
   [SetEvent](https://go.microsoft.com/fwlink/?LinkId=150232)
