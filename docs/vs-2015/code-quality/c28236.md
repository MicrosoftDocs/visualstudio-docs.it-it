# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28236: l'annotazione non può essere utilizzata in un'espressione  
  
 Questo avviso indica che è stato effettuato un tentativo di utilizzare una funzione dichiarata come un'annotazione in un contesto dell'espressione. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto affinché sia disponibile in uso tipico.
