# <a name="c28272"></a>C28272
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28272](https://docs.microsoft.com/visualstudio/code-quality/c28272).  
  
avviso C28272: l'annotazione per la funzione di parametro durante l'analisi non è coerente con la dichiarazione di funzione  
  
 Questo avviso indica un errore nelle annotazioni, non nel codice analizzato. Le annotazioni visualizzate in una definizione di funzione sono coerenti con quelli visualizzati in una dichiarazione. Le due annotazioni devono essere risolti in modo che corrispondano.
