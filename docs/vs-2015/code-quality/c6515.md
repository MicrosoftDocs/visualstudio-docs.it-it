# <a name="c6515"></a>C6515
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C6515: annotazione non valida: \<nome > la proprietà può essere utilizzata solo con valori di tipo puntatore  
  
 Questo avviso indica che una proprietà per l'uso sui puntatori sia stata applicata a un tipo non puntatore. Per un elenco delle proprietà di annotazione, vedere [delle proprietà di annotazione](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Esempio  
 Il codice seguente genera questo avviso:  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 Per risolvere questo problema, usare il codice seguente:  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
