# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28213: è necessario utilizzare l'annotazione `_Use_decl_annotations_` per fare riferimento, senza alcuna modifica, a una dichiarazione precedente. \<perché >  
  
 Questo avviso viene segnalato quando viene fatto riferimento a una dichiarazione precedente senza l'annotazione `_Use_decl_annotations_` richiesta.
