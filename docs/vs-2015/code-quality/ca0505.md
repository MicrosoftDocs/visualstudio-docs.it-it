# <a name="ca0505"></a>CA0505

La proprietà deprecata verrà ignorata perché la proprietà superceding è definita.

CA505 avviso viene generato quando deprecato specificato sia superceding proprietà di analisi del codice contengono valori in un progetto o file di configurazione di compilazione. Per risolvere questo problema, rimuovere la proprietà deprecata.
