# <a name="ca0505"></a>CA0505

La proprietà deprecata verrà ignorata perché la proprietà supercedente è definita.

L'avviso CA505 viene generato quando le proprietà di analisi del codice deprecate e con la supercessione specificate contengono valori in un progetto o in un file di configurazione della compilazione. Per risolvere questo problema, rimuovere la proprietà deprecata.
