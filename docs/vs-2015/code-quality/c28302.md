# <a name="c28302"></a>C28302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28302: per il parametro di riferimento di C++ <parameter_name> un `_Deref_` operatore aggiuntivo è stato trovato in \<annotation> .  
  
 Questo avviso viene segnalato quando si utilizza un livello aggiuntivo di `_Deref_` in un parametro.  
  
 SAL2 non richiede l'uso di un livello aggiuntivo di `_Deref_` quando si gestiscono parametri di riferimento. Questa particolare annotazione non è ambigua e viene interpretata correttamente, ma deve essere corretta.  
  
 Spesso è possibile correggere questo problema rimuovendo semplicemente l' `__deref` annotazione precedente e usando la sintassi SAL2. Talvolta può essere necessario usare `_At_` per fare riferimento all'oggetto specifico da annotare.
