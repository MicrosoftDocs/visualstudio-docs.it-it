# <a name="c6292"></a>C6292
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C6292: ciclo for definito in modo non corretto: conteggi fino al massimo  
  
 Questo avviso indica che un ciclo for potrebbe non funzionare come previsto.  
  
 Si verifica quando un ciclo viene conteggiato da un valore massimo, ma presenta una condizione di terminazione inferiore. Questo ciclo verr√† terminato solo dopo l'overflow di Integer.  
  
## <a name="example"></a>Esempio  
 Il codice seguente genera questo avviso:  
  
```  
void f( )  
{  
   int i;  
  
   for (i = 100; i >= 0; i++)  
   {  
      // code ...  
   }  
}  
```  
  
 Per correggere il problema, usare il codice seguente:  
  
```  
void f( )  
{  
   int i;  
  
  for (i = 100; i >= 0; i--)  
   {  
      // code ...  
   }  
}  
```
