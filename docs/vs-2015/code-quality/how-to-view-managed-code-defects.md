# <a name="how-to-view-managed-code-defects"></a>Procedura: Visualizzare gli errori del codice gestito
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [procedura: visualizzazione difetti di codice gestito](https://docs.microsoft.com/visualstudio/code-quality/how-to-view-managed-code-defects).  
  
Nell'analisi del codice, errori, avvisi e i messaggi vengono segnalati nella finestra Analisi codice. Finestra Analisi codice viene visualizzata automaticamente quando si esegue l'analisi del codice in un progetto o soluzione.  
  
 Nella finestra Analisi codice, è possibile:  
  
-   **Scegliere il titolo di avviso.**  
  
     Informazioni aggiuntive sull'avviso viene visualizzato nella finestra Analisi codice. La riga nel file di origine che ha attivato l'avviso è visualizzata ed evidenziata nell'editor del codice sorgente.  
  
-   **Scegliere l'id dell'avviso.** ID avviso sono preceduti dal prefisso **CA** e includere un numero a quattro cifre. Ad esempio, CA1234.  
  
     L'argomento della Guida di MSDN Library per l'avviso viene visualizzato. L'argomento della Guida vengono fornite informazioni dettagliate sulle cause dell'avviso e include esempi che illustrano come il problema di avviso viene generato e risolto.
