# <a name="how-to-view-managed-code-defects"></a>Procedura: Visualizzare gli errori del codice gestito
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nell'analisi del codice, gli errori, gli avvisi e i messaggi vengono segnalati nella finestra analisi codice. La finestra analisi codice viene visualizzata automaticamente quando si esegue l'analisi del codice su un progetto o una soluzione.

 Nella finestra analisi codice Ã¨ possibile:

- **Scegliere il titolo dell'avviso.**

     Nella finestra analisi codice vengono visualizzate ulteriori informazioni sull'avviso. La riga nel file di origine che ha generato l'avviso viene visualizzata ed evidenziata nell'editor del codice sorgente.

- **Scegliere l'ID avviso.** Gli ID avviso sono preceduti da una **CA** e includono un numero a quattro cifre. Ad esempio, CA1234.

     Viene visualizzato l'argomento della Guida di MSDN Library relativo all'avviso. L'argomento della guida contiene informazioni dettagliate sulla causa dell'avviso e include esempi che illustrano il modo in cui si verifica il problema di avviso e viene risolto.
