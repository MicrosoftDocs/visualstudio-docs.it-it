# <a name="c28231"></a>C28231
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28231](https://docs.microsoft.com/visualstudio/code-quality/c28231).  
  
avviso C28231: annotazione è valida solo per matrice  
  
 Questo avviso indica che un argomento di un'annotazione deve essere una matrice, ed è stato rilevato un altro tipo.
