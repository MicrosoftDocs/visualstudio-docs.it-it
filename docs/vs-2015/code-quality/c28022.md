# <a name="c28022"></a>C28022
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28022: le classi di funzioni \<classlist1 > in questa funzione non corrispondono alle classi di funzioni \<classlist2 > sul typedef utilizzato per definirle.  
  
 Questo avviso viene segnalato quando si verifica un errore nelle annotazioni. Sia il typedef e della funzione stessa hanno `_Function_class_` annotazioni, ma non corrispondono. Se vengono utilizzati entrambi devono corrispondere.
