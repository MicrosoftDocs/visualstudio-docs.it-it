La versione più recente di questo argomento è reperibile in [C1253](https://docs.microsoft.com/visualstudio/code-quality/c1253).  
  
avviso C1253: Impossibile caricare il file di modello.  
  
 Lo strumento di analisi del codice genera questo avviso quando si verifica un errore interno nel file del modello, non nel codice analizzato.
