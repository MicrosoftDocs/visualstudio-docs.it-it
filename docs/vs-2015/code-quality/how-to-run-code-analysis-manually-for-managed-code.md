# <a name="how-to-run-code-analysis-manually-for-managed-code"></a>Procedura: Eseguire manualmente l'analisi codice per il codice gestito
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Lo strumento di analisi codice fornisce informazioni all'utente sui possibili errori nel codice sorgente. È possibile eseguire l'analisi del codice automaticamente con ogni compilazione di un progetto di codice ed è anche possibile eseguire manualmente l'analisi del codice. Le regole che vengono controllate durante l'esecuzione di analisi del codice sono specificati nella pagina analisi del codice delle pagine delle proprietà di progetto. Per altre informazioni, vedere [Procedura: Configura analisi codice per un progetto di codice gestito](../code-quality/how-to-configure-code-analysis-for-a-managed-code-project.md)  
  
### <a name="to-run-code-analysis-manually"></a>Per eseguire manualmente l'analisi del codice  
  
1. Nelle **Esplora soluzioni**, fare clic con il progetto.  
  
2. Nel **Analyze** menu, fare clic su **Esegui analisi del codice** *nome progetto*.
