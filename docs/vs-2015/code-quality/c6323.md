# <a name="c6323"></a>C6323
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C6323: utilizzo di un operatore aritmetico su tipi Boolean  
  
 Questo avviso si verifica se gli operatori aritmetici utilizzati sui tipi di dati Boolean. Utilizzo dell'operatore non corretto potr√† restituire risultati non corretti. Indica inoltre che lo scopo del programmatore non viene riflessa nel codice.  
  
## <a name="example"></a>Esempio  
 Il codice seguente genera questo avviso:  
  
```  
int test(bool a, bool b)  
{  
    int c = a + b;     //C6323  
    return c;  
}  
```  
  
 Per risolvere questo problema, usare l'operatore e il tipo di dati corretto.  
  
```  
int test(int a, int b)  
{  
    int c = a + b;       
    return c;  
}  
```
