La versione più recente di questo argomento è reperibile in [C28241](https://docs.microsoft.com/visualstudio/code-quality/c28241).  
  
avviso C28241: l'annotazione per la funzione su parametro non è riconosciuto  
  
 È stato usato un nome di annotazione non riconosciuta. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto affinché sia disponibile in uso tipico.
