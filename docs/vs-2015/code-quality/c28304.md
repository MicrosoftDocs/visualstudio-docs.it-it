# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28304: < nome_parametro > parametro di riferimento per C++, un non correttamente posizionato `_Notref_` Impossibile trovare un operatore applicato a \<token >.  
  
 Il `_Notref_` operatore deve essere usato solo in circostanze speciali che includono i parametri di riferimento di C++ e solo nelle macro fornite dal sistema. Deve essere seguito immediatamente da un `_Deref_` operatore o un'annotazione funzionale.
