# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28283](https://docs.microsoft.com/visualstudio/code-quality/c28283).  
  
avviso C28283: per il simbolo, le specifiche di dimensione non sono ancora supportata  
  
 L'avviso indica che l'annotazione sentinel"dimensione specifica" ha ricevuto un valore diverso da zero. In pratica, il chiamante ha tentato di dire che la stringa termina con un carattere diverso da zero binario.
