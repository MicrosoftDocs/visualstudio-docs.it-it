# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [C28279](https://docs.microsoft.com/visualstudio/code-quality/c28279).  
  
avviso C28279: per il simbolo, un 'begin' è stato trovato senza il corrispondente 'end'  
  
 Il linguaggio di annotazione supporta un begin ed end (`{` e `}` in C) costrutto e l'associazione è diventato un utilizzo non equilibrato. Questa situazione può essere evitata se vengono usate le macro.
