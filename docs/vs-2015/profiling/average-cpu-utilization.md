# <a name="average-cpu-utilization"></a>Utilizzo CPU medio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Mostra l'utilizzo medio dei core logici del sistema del processo profilato per la durata del processo. Nel grafico non viene indicato quale core specifico è attivo in un determinato momento. Ad esempio, se due core sono ognuno in esecuzione al 50% della capacità per un periodo di tempo specificato, in questa visualizzazione verrà indicato l'utilizzo di un solo core logico.
