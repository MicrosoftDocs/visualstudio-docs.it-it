# <a name="spanspan-destructor"></a>Distruttore span::~span
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [span:: ~ span distruttore](https://docs.microsoft.com/visualstudio/profiling/span-tilde-span-destructor).  
  
Elimina l'oggetto `span` e ne rilascia le risorse.  
  
## <a name="syntax"></a>Sintassi  
  
```  
~span();  
```  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** cvmarkersobj.h  
  
 **Spazio dei nomi:** Concurrency::diagnostic
