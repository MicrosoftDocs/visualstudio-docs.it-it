# <a name="markerseriesmarkerseries-destructor"></a>Distruttore marker_series::~marker_series
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [marker_series:: ~ marker_series distruttore](https://docs.microsoft.com/visualstudio/profiling/marker-series-tilde-marker-series-destructor).  
  
Elimina l'oggetto marker_series e rilascia tutte le risorse allocate.  
  
## <a name="syntax"></a>Sintassi  
  
```  
~marker_series();  
```  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** cvmarkersobj.h  
  
 **Spazio dei nomi:** Concurrency::diagnostic
