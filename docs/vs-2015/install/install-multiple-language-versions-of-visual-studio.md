# <a name="install-multiple-language-versions-of-visual-studio"></a>Installare più versioni localizzate di Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Per la documentazione più recente di Visual Studio, vedere [Installare i Language Pack](/visualstudio/install/install-visual-studio?view=vs-2019#step-6---install-language-packs-optional).

Installando uno o più Language Pack, è possibile eseguire Visual Studio 2015 in lingue diverse nello stesso computer. È necessario ottenere un Language Pack separato per ogni lingua da installare.

## <a name="important-considerations"></a>Considerazioni importanti

- Quando si installa un Language Pack, per ogni lingua vengono installati automaticamente file di risorse per le funzionalità installate.

- I pacchetti servizio di Visual Studio si applicano a tutte le versioni di Visual Studio in lingue diverse installate. Non è più necessario installare un pacchetto servizio per ogni versione localizzata di Visual Studio installata.

- È possibile disinstallare i Language Pack usando la modalità di manutenzione del programma di installazione di Visual Studio. Nel **Pannello di controllo**aprire **Programmi e funzionalità**, scegliere l'edizione di Visual Studio, quindi scegliere **Disinstalla/Cambia**.
