# <a name="wpf-targets-files"></a>File WPF con estensione targets
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

  
[!INCLUDE[TLA#tla_winclient](../includes/tlasharptla-winclient-md.md)] estende [!INCLUDE[TLA#tla_msbuild](../includes/tlasharptla-msbuild-md.md)] con l'aggiunta di un insieme di attività specifiche di [!INCLUDE[TLA2#tla_wpf](../includes/tla2sharptla-wpf-md.md)] combinate in un file speciale con estensione targets, **Microsoft.WinFX.targets**. Questo file combina l'insieme delle attività di [!INCLUDE[TLA2#tla_msbuild](../includes/tla2sharptla-msbuild-md.md)] necessarie per compilare un progetto [!INCLUDE[TLA2#tla_msbuild](../includes/tla2sharptla-msbuild-md.md)] in [!INCLUDE[TLA#tla_winclient](../includes/tlasharptla-winclient-md.md)].
