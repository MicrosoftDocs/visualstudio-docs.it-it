# <a name="lib-task"></a>Attività LIB
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Esegue il wrapping dello strumento di gestione librerie Microsoft a 32 bit, lib.exe. Gestione librerie crea e gestisce una libreria di file oggetto COFF (Common Object File Format). Gestione librerie può inoltre creare file di esportazione e librerie di importazione per fare riferimento a definizioni esportate. Per altre informazioni, vedere [Riferimenti a LIB](https://msdn.microsoft.com/library/ecc7f643-bbd4-47a3-8dc6-b360f880db91) ed [Esecuzione di LIB](https://msdn.microsoft.com/library/d54f5c81-7147-4b2c-a8db-68ce6eb1eabd).
