# <a name="fetching-data-into-your-application"></a>Recupero di dati nell'applicazione
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Il caricamento dei dati nell'applicazione viene descritto il processo di recupero dei dati da un'origine dati, ad esempio un database o un servizio e popolare gli oggetti nel modello di dati di un'applicazione. Gli argomenti seguenti descrivono come recuperare i dati e caricare i dati in modelli di dati disponibili.
