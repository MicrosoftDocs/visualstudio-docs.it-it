# <a name="data-user-interface-elements"></a>Elementi dell'interfaccia utente dati
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Questa sezione contiene le informazioni su tutte le finestre di dialogo e le procedure guidate usate durante la progettazione dell'accesso ai dati nelle applicazioni [!INCLUDE[vbprvb](../includes/vbprvb-md.md)] o Visual C#.  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
## <a name="in-this-section"></a>In questa sezione  
 [Dati Smart tag](http://msdn.microsoft.com/en-us/1e0a848f-c57b-47ab-b884-eaaa40726f43)  
 Fornisce informazioni su alcuni comandi smart tag disponibili per i dati.  
  
 [Aggiungi finestra di dialogo set di dati](http://msdn.microsoft.com/en-us/0e03c0ff-212b-4bfa-ac51-3c2adb71ead0)  
 Inserisce un oggetto esistente tipizzata set di dati o un nuovo set di dati non tipizzato (un'istanza del **DataSet** classe) in un form o un componente.  
  
 [Advanced SQL nella finestra di dialogo Opzioni di generazione](http://msdn.microsoft.com/en-us/41420450-1ff4-4a1a-b85b-6f6901538fef)  
 Consente di controllare come vengono create le stored procedure o le istruzioni SQL per un adattatore dati specificando le opzioni per gli aggiornamenti, la concorrenza ottimistica e l'aggiornamento del set di dati.  
  
 [Scegliere una finestra di dialogo di istanza di origine di dati](http://msdn.microsoft.com/en-us/51c47f06-fdc5-453e-9178-0a5a2c5c9f34)  
 Consente di selezionare il set di dati desiderato da un elenco di set di dati del progetto.  
  
 [Scegliere un'origine dati di tipo merge con la finestra di dialogo](http://msdn.microsoft.com/en-us/accafff7-f6bd-481c-a121-fe8a76cd681d)  
 Consente di selezionare l'origine dati con cui effettuare l'operazione di merge quando sono disponibili più origini dati.  
  
 [Scegliere la finestra di dialogo chiave](http://msdn.microsoft.com/en-us/4ddbfbb7-a80a-412a-b80d-291d86376ca3)  
 Consente di selezionare una chiave nel caso in cui una colonna sia parte di un vincolo a più chiavi.  
  
 [Editor della raccolta](http://msdn.microsoft.com/library/030095bd-fb9a-4b21-b628-fc1cc5985bb7)  
 Fornisce i collegamenti ad argomenti su diversi Editor di raccolte che consentono di creare e modificare i singoli membri di una raccolta.  
  
 [Connessioni mancanti](http://msdn.microsoft.com/en-us/bb9b2e12-7f76-4ee5-acbb-5d20116ee044)  
 Segnala all'utente quando un set di dati contiene un riferimento a una stringa di connessione non inclusa nelle impostazioni dell'applicazione.  
  
 [Finestra di dialogo Errore di configurazione adattatore dati](http://msdn.microsoft.com/en-us/9ce65cd2-0c7d-4f51-8685-d68be5f3009b)  
 Visualizza gli errori che si sono verificati mentre Visual Studio stava provando a creare un'istanza di un adattatore dati e a impostarne le proprietà.  
  
 [Configurazione guidata adattatore dati](http://msdn.microsoft.com/en-us/efff90cb-0e4c-4eb3-87dc-65dd9d418809)  
 Configura le stored procedure o i comandi SQL usati da un adattatore dati per leggere i dati di un set di dati da un database e per riscriverli. Questo argomento illustra come eseguire la procedura guidata e cosa fare una volta completata.  
  
 [Finestra di dialogo Anteprima adattatore dati](http://msdn.microsoft.com/en-us/1f614cd3-4530-457e-84af-00ccbaea08cc)  
 Consente di vedere come verranno inseriti i dati in un set di dati da un adattatore dati. Questa funzionalità è utile per verificare se l'adattatore restituisce i dati previsti, se il mapping delle tabelle funziona correttamente e quale effetto producono i valori impostati per i vari parametri.  
  
 [Finestra di dialogo account di accesso di origine dati](http://msdn.microsoft.com/en-us/6f2d9a57-53c3-4841-bd37-a3643eb68d2e)  
 Consente di richiedere l'accesso a un'origine dati, in genere un database, per cui non si è ancora stati autenticati.  
  
 [Scheda set di dati, della casella degli strumenti](http://msdn.microsoft.com/en-us/fa5f2d6f-924d-4262-ba1b-e9e7f90e7764)  
 Visualizza gli oggetti che è possibile aggiungere a un set di dati tipizzato.  
  
 [Si desidera includere la password nella stringa di connessione nella finestra di dialogo](http://msdn.microsoft.com/en-us/193696a7-5213-4396-8328-05ac2df6ee94)  
 Consente di controllare se una password è incorporata nella stringa di connessione.  
  
 [Modificare la finestra di dialogo chiave](http://msdn.microsoft.com/en-us/f5c80e39-3a42-4284-b222-6ca009fd9675)  
 Consente di definire e modificare le chiavi.  
  
 [Finestra di dialogo vincolo di chiave esterna](http://msdn.microsoft.com/en-us/45d15629-1f4d-40a7-8708-c9ddfebedc1e)  
 Consente di applicare un vincolo di chiave esterna a una o più colonne di una tabella del set di dati correlata a un'altra tabella.  
  
 [Dialogo Genera Dataset](http://msdn.microsoft.com/en-us/c0efdbaf-13b1-4ee8-ade6-f8a784126cdc)  
 Consente di generare un nuovo set di dati tipizzato dalle informazioni fornite da uno o più adattatori dati e di aggiungere le tabelle in un set di dati esistente.  
  
 [Più istanze BindingSource finestra di dialogo](http://msdn.microsoft.com/en-us/db76f70c-4fb5-479d-9b64-a67158d48f97)  
 Consente di selezionare l'oggetto <xref:System.Windows.Forms.BindingSource> da usare nel caso in cui siano disponibili più oggetti <xref:System.Windows.Forms.BindingSource>.  
  
 [Finestra di dialogo Anteprima dati](http://msdn.microsoft.com/en-us/aa4f0d04-2695-4bb8-946d-54a97ae7287f)  
 Consente di esaminare i dati restituiti dalle query nel progetto.  
  
 [Finestra di dialogo Anteprima Script SQL](http://msdn.microsoft.com/en-us/e9571e8b-821c-492d-9bc8-b44eba898bdd)  
 Viene visualizzato come parte del [configurazione guidata adattatore dati](http://msdn.microsoft.com/en-us/efff90cb-0e4c-4eb3-87dc-65dd9d418809) per poter visualizzare lo script SQL userà la procedura guidata per creare stored procedure per leggere e scrivere dati.  
  
 [Finestra di dialogo relazione](http://msdn.microsoft.com/en-us/ab8f4b0e-af4c-4725-a550-e2b2ebe43a02)  
 Consente di creare una relazione (un **DataRelation** oggetto) che mantiene le informazioni sui record padre-figlio nelle due tabelle dati di un set di dati.  
  
 [Finestra di dialogo Generatore criteri di ricerca](http://msdn.microsoft.com/library/0b306b92-f35e-45ef-a4be-3f653cd00c3d)  
 Consente di creare una query con parametri in un Windows Form associato a dati e di aggiungere automaticamente i controlli necessari per eseguire la query.  
  
 [Finestra di dialogo Mapping tabella](http://msdn.microsoft.com/en-us/fb4cec1e-f3c8-4773-b409-c2de15293fea)  
 Consente di specificare quali colonne di una tabella di database o di un'altra origine dati corrispondono alle colonne di una tabella del set di dati.  
  
 [Finestra di dialogo vincolo UNIQUE](http://msdn.microsoft.com/en-us/e71a60d7-fae2-4bd0-a1e8-43aae351707d)  
 Consente di applicare un vincolo UNIQUE a una o più colonne di una tabella di un set di dati non tipizzato.
