# <a name="connecting-to-data-in-visual-studio"></a>Connessione ai dati in Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] offre strumenti per connettere l'applicazione a dati da molte origini diverse, ad esempio database, servizi e oggetti. Le connessioni ai dati vengono create a livello di codice, in seguito al trascinamento di oggetti dati nella finestra di progettazione di un form o di un componente, oppure mediante l'esecuzione di una delle procedure guidate relative ai dati. Gli argomenti riportati di seguito descrivono come creare connessioni a database, servizi e oggetti dall'applicazione.
