# <a name="could-not-retrieve-schema-information-for-database-object-ltobject-namegt"></a>Impossibile recuperare le informazioni sullo schema per il &lt; nome dell'oggetto di database&gt;
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Questo messaggio viene in genere visualizzato quando un oggetto in **Esplora server/Esplora database** viene copiato negli Appunti, eliminato dal database e quindi incollato nella finestra di progettazione. La visualizzazione del messaggio è pertanto dovuta al fatto che l'oggetto di database non esiste più.

### <a name="to-correct-this-error"></a>Per correggere l'errore

- Fare clic su **OK** per chiudere la finestra di dialogo.
