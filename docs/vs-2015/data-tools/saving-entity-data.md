# <a name="saving-entity-data"></a>Salvataggio di dati di entità
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Quando si creano, aggiornano o eliminano entità mediante le quali è possibile connettersi a un database tramite Data Services WCF, è necessario salvare di nuovo in modo esplicito le modifiche al database. Negli argomenti riportati di seguito viene illustrato come salvare le modifiche ai dati di entità nelle applicazioni Data Services WCF.  
  
|Argomento|Descrizione|  
|-----------|-----------------|  
|[Procedura: aggiungere, modificare ed eliminare entità](http://msdn.microsoft.com/library/a00f8933-b232-4445-95ba-adc634f055d8)|Viene illustrato come salvare le modifiche nei dati di entità tramite un servizio dati.|  
|[Procedura: eseguire query asincrone sul servizio dati](http://msdn.microsoft.com/library/902a2dc1-d0e9-4b00-90a8-becc4cb1f6a7)|Viene illustrato come salvare in modo asincrono le modifiche tramite un servizio dati.|
