# <a name="how-to-open-the-data-sources-window"></a>Procedura: Aprire la finestra Origini dati
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Il **Zdroje dat** vengono visualizzate le origini dati nel progetto. Si utilizza il **Zdroje dat** finestra per creare un'interfaccia utente dei controlli con associazione a dati trascinando elementi dalla finestra nei form del progetto. Visualizzare [controlli di associare Windows Form ai dati in Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 È possibile popolare la **Zdroje dat** finestra eseguendo il **configurazione dell'origine dati** procedura guidata. Per altre informazioni, vedere [configurazione guidata origine dati](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f).  
  
### <a name="to-open-the-data-sources-window"></a>Per aprire la finestra Origini dati  
  
-   Nella barra dei menu, scegliere **View**, **Other Windows**, **Zdroje dat** (o premere i tasti Maiusc + Alt + D).  
  
    > [!IMPORTANT]
    >  È possibile aggiungere un'origine dati a un progetto solo se il progetto supporta la creazione e l'utilizzo delle origini dati. Ad esempio, non è possibile aprire il **Zdroje dat** finestra in un progetto per un'app Windows Store.
