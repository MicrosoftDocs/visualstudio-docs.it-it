# <a name="idiaenumdebugstreamsreset"></a>IDiaEnumDebugStreams::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [Idiaenumdebugstreams](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumdebugstreams-reset).  
  
Reimposta una sequenza di enumerazione all'inizio.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp#  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce `S_OK`.
