# <a name="idiasectioncontribgetaddressoffset"></a>IDiaSectionContrib::get_addressOffset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [Get_addressoffset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasectioncontrib-get-addressoffset).  
  
Recupera la parte offset di indirizzo del contributo.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp#  
HRESULT get_addressOffset (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `pRetVal`  
 [out] Restituisce la parte offset di indirizzo del contributo.  
  
## <a name="return-value"></a>Valore restituito  
 Se ha esito positivo, restituisce `S_OK`. Restituisce `S_FALSE` se questa proprietà non è supportata. In caso contrario, verrà restituito un codice di errore.
