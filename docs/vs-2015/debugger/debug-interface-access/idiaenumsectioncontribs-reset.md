# <a name="idiaenumsectioncontribsreset"></a>IDiaEnumSectionContribs::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [Idiaenumsectioncontribs](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumsectioncontribs-reset).  
  
Reimposta una sequenza di enumerazione all'inizio.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp#  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce `S_OK`.
