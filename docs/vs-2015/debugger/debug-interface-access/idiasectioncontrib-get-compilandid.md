# <a name="idiasectioncontribgetcompilandid"></a>IDiaSectionContrib::get_compilandId
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [Get_compilandid](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasectioncontrib-get-compilandid).  
  
Recupera l'identificatore del modulo nella sezione.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp#  
HRESULT get_compilandId (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `pRetVal`  
 [out] Restituisce l'identificatore del modulo per la sezione.  
  
## <a name="return-value"></a>Valore restituito  
 Se ha esito positivo, restituisce `S_OK`. Restituisce `S_FALSE` se questa proprietà non è supportata. In caso contrario, verrà restituito un codice di errore.
