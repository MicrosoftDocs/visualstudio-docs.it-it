# <a name="idiaenumstackframesreset"></a>IDiaEnumStackFrames::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [Idiaenumstackframes](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumstackframes-reset).  
  
Riporta all'inizio la sequenza di enumerazione.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp#  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce `S_OK`.
