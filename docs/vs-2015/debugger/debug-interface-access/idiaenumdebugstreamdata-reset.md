# <a name="idiaenumdebugstreamdatareset"></a>IDiaEnumDebugStreamData::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [Idiaenumdebugstreamdata](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumdebugstreamdata-reset).  
  
Reimposta il valore di inizio di una sequenza enumerata.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp#  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce `S_OK`.
