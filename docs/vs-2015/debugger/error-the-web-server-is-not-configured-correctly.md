# <a name="error-the-web-server-is-not-configured-correctly"></a>Errore: il server Web non è configurato in modo corretto
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [errore: il server web non è configurato correttamente](https://docs.microsoft.com/visualstudio/debugger/error-the-web-server-is-not-configured-correctly).  
  
Alcune cause possibili di questo errore sono:  
  
-   Tentativo di eseguire il debug di un'applicazione Web .NET copiata in un computer diverso, rinominato manualmente o spostato.  
  
-   Indisponibilità di un numero sufficiente di connessioni IIS. Per altre informazioni su come distribuire un sito Web in IIS, vedere [Creare un sito Web](http://www.iis.net/learn/get-started/getting-started-with-iis/create-a-web-site).  
  
-   Se si sta provando a eseguire il debug di un'applicazione ASP.NET, vedere [pubblicazione in IIS](https://docs.asp.net/en/latest/publishing/iis.html) per istruzioni sulla distribuzione in un computer remoto che esegue IIS 8 o versione successiva, o [Remote Debugging ASP.NET on a Remote IIS 7.5 Computer](../debugger/remote-debugging-aspnet-on-a-remote-iis-7-5-computer.md) per istruzioni sulla distribuzione in un computer remoto che esegue IIS 7.5.
