# <a name="tree-menu-commands"></a>comandi del menu Struttura ad albero
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Il menu **albero** fornisce comandi per espandere o comprimere le visualizzazioni.  
  
 **Espandi un livello**  
 Espande l'elemento attualmente selezionato in una vista di sistema visualizzandone un livello al di sotto di esso.  
  
 **Espandi ramo**  
 Espande tutti i livelli sotto l'elemento attualmente selezionato.  
  
 **Espandi tutto**  
 Espande e Visualizza tutti i livelli in una vista di sistema.  
  
 **Collapse**  
 Comprime tutti i livelli sotto l'elemento attualmente selezionato.
