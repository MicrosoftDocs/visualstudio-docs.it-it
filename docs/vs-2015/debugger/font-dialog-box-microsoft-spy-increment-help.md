# <a name="font-dialog-box-microsoft-spy-help"></a>Finestra di dialogo Tipo di carattere (Guida di Microsoft Spy++)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Utilizzare questa finestra di dialogo per configurare il carattere carattere utilizzato in una finestra delle [visualizzazioni di Spy + +](../debugger/spy-increment-views.md) . Per visualizzare questa finestra di dialogo, espandere il menu **Visualizza** e scegliere **carattere**.  
  
 Sono disponibili le impostazioni seguenti:  
  
 **Carattere**  
 Carattere carattere per la finestra visualizzazioni di Spy + + con lo stato attivo.  
  
 **Stile**  
 Stile del carattere per la finestra delle visualizzazioni di Spy + + con lo stato attivo.  
  
 **Dimensione**  
 Dimensioni del carattere per la finestra delle visualizzazioni di Spy + + con lo stato attivo.  
  
 **Imposta come predefinito**  
 Salva le impostazioni dei tipi di carattere correnti quando si aprono le visualizzazioni di Spy + +.
