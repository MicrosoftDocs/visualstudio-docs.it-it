# <a name="how-to-debug-inline-assembly-code"></a>Procedura: eseguire il debug di codice assembly inline
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

NOTA]
>  Le finestre di dialogo e i comandi di menu visualizzati potrebbero non corrispondere a quelli descritti nella Guida in quanto dipendono dall'edizione o dalle impostazioni in uso. Per modificare le impostazioni, scegliere Importa/esporta impostazioni dal menu Strumenti. Per altre informazioni, vedere [Personalizzazione delle impostazioni di sviluppo in Visual Studio](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
 Il debugger fornisce due finestre di debug codice assembly inline, il **Disassembly** finestra e il **registra** finestra.  
  
## <a name="procedure"></a>Routine  
  
#### <a name="to-debug-inline-assembly-code"></a>Per eseguire il debug di codice assembly inline  
  
1.  Usare la **Disassembly** finestra per visualizzare le istruzioni dell'assembly.  
  
2.  Usare la **registra** finestra per visualizzare il contenuto del registro.
