# <a name="error-mixed-mode-debugging-for-ia64-processes-is-unsupported"></a>Errore: il debug in modalità mista per i processi IA64 non è supportato
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versione più recente di questo argomento è reperibile in [errore: debug in modalità mista per i processi IA64 non è supportato](https://docs.microsoft.com/visualstudio/debugger/error-mixed-mode-debugging-for-ia64-processes-is-unsupported).  
  
Il debugger di [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] non supporta il debug di codice nativo e gestito misto in un processo basato su Itanium.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Compilare una versione a 32 bit dell'applicazione per il debug.
