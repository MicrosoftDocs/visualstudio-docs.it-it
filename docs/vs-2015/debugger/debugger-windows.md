# <a name="debugger-windows"></a>Finestre del debugger
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

È possibile aprire la maggior parte delle finestre del debugger durante il debug del programma. Per visualizzare un elenco delle finestre del debugger, impostare un punto di interruzione e avviare il debug. Quando si raggiunge il punto di interruzione e l'esecuzione viene arrestata, fare clic su **debug/Windows**.
