# <a name="error-mixed-mode-debugging-is-supported-only-when-using-microsoft-net-framework-20-or-greater"></a>Errore: Il debug in modalità mista è supportato solo quando si usa Microsoft .NET Framework 2.0 o versione successiva
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Per eseguire il debug di codice nativo e gestito misto, è necessario disporre di [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] versione 2.0, 3.0, 3.5 o 4. Il debug in modalità mista con versioni di [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] precedenti non è supportato.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiornare [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] alla versione 2.0, 3.0, 3.5 o 4.
