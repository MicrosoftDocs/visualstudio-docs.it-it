# <a name="globalizing-applications"></a>Globalizzazione di applicazioni
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Globalizzazione è il processo di progettazione e sviluppo di un prodotto software per il funzionamento in più impostazioni cultura. Questa sezione si applica alle pagine di Windows Form e Web Form.

## <a name="in-this-section"></a>Contenuto della sezione
 [Classi specifiche delle impostazioni cultura per Windows Forms e Web Form globali](../ide/culture-specific-classes-for-global-windows-forms-and-web-forms.md) Vengono illustrate le classi che formattano date, ore, numeri, valuta e altre informazioni in base all'impostazione delle impostazioni cultura.

 [Procedure consigliate per lo sviluppo di applicazioni internazionali](https://msdn.microsoft.com/library/f08169c7-aad8-4ec3-9a21-9ebd3b89986c) Fornisce informazioni sulla programmazione per un pubblico internazionale, ad esempio problemi di progettazione e terminologia.
