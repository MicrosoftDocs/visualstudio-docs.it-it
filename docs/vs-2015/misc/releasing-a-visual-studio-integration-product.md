# <a name="releasing-a-visual-studio-integration-product"></a>Rilascio del prodotto di integrazione di Visual Studio
Gli argomenti di questa sezione illustrano come rilasciare il prodotto di integrazione di Visual Studio compilando e installando VSPackage e usando il test di integrazione con Visual Studio (VSIT).

## <a name="in-this-section"></a>Contenuto della sezione
 [Installazione di VSPackage](../misc/installing-vspackages.md) Viene illustrato come compilare e installare VSPackage e come supportare gli utenti che eseguono più versioni di Visual Studio nello stesso momento.

 [Utilità regpkg](../extensibility/internals/regpkg-utility.md) Descrive l'utilità RegPkg, usata per registrare un VSPackage e prepararlo per la distribuzione.

 [Guida dell'utente e dell'installazione di Premier Partner Edition](https://msdn.microsoft.com/8ee4dad7-95d3-4f2d-a8d4-3ba9a80ecae2) Fornisce ai partner di VSIP una versione ridistribuibile di Visual Studio Integrated Development Environment (IDE) in modo da poter aggiungere i propri strumenti di sviluppo e linguaggi di computer.
