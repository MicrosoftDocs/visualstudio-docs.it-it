# <a name="flowchart-activity-designers"></a>ActivityDesigner Diagramma di flusso

Progettazione flussi di lavoro include diverse attività fornite dal sistema che è possibile usare per la costruzione dei flussi di lavoro. Contenuto della sezione sono presentate le attività fornite dal sistema usate per gestire diagrammi di flusso all'interno di un flusso di lavoro. Negli argomenti riportati di seguito vengono illustrate queste attività e vengono fornite indicazioni sulle relative modalità di utilizzo.
