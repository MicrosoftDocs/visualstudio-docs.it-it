# <a name="workflow-designer-ui-help"></a>Guida dell'interfaccia utente della finestra di progettazione dei flussi di lavoro

In questa sezione contiene gli argomenti della Guida sensibile al contesto per la progettazione del flusso di lavoro di Windows. Quando si lavora in Visual Studio 2010, è possibile accedere a questi argomenti dall'interfaccia utente premendo **F1** quando è aperta una finestra di dialogo di progettazione flussi di lavoro.

Nella sezione sono inoltre contenuti diversi argomenti relativi alla navigazione e ai messaggi di errore.
