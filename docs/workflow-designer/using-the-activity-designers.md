# <a name="using-the-activity-designers"></a>Utilizzo degli ActivityDesigner

La finestra di progettazione del flusso di lavoro di Windows include un numero di attività fornite dal sistema che è possibile utilizzare quando si crea l'attività e flussi di lavoro. Contenuto della sezione vengono descritte le categorie di attività fornite dal sistema e le relative finestre di progettazione disponibili. Negli argomenti relativi alle singole attività e alle rispettive finestre di progettazione all'interno di ciascuna categoria vengono illustrate le operazioni eseguite da tali attività e viene fornito materiale sussidiario sul relativo uso.
