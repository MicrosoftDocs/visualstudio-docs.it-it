# <a name="messaging-activity-designers"></a>Activity Designer Messaging

Gli ActivityDesigner Messaging vengono usati per creare e configurare le attività di messaggistica di inviare e ricevano messaggi di Windows Communication Foundation (WCF) da all'interno di un'applicazione Windows Workflow Foundation (WF). Cinque attività di messaggistica sono state introdotte in .NET Framework 4. La finestra di progettazione del flusso di lavoro sono disponibili due finestre progettazione modello che consentono di gestire la messaggistica all'interno di un flusso di lavoro.

Gli argomenti contenuti in questa sezione ed elencati nella tabella seguente forniscono indicazioni su come usare l'attività di progettazione del flusso di lavoro e i progettisti di modelli.

- <xref:System.Activities.Activity>

- <xref:System.ServiceModel.Activities.CorrelationScope>

- <xref:System.ServiceModel.Activities.Receive>

- <xref:System.ServiceModel.Activities.Send>

- <xref:System.ServiceModel.Activities.ReceiveReply>

- <xref:System.ServiceModel.Activities.SendReply>

- <xref:System.ServiceModel.Activities.TransactedReceiveScope>
