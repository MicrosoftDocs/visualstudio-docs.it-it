# <a name="integration-of-xml-literals-with-xml-schema-explorer"></a>Integrazione di valori letterali XML con XML Schema Explorer

Visual Basic supporta valori letterali XML, il che significa che è possibile incorporare frammenti XML direttamente nel codice Visual Basic. Per altre informazioni, vedere [Cenni preliminari sui valori letterali XML](http://go.microsoft.com/fwlink/?LinkId=140325).

## <a name="how-to"></a>Procedura

Se un file XSD nel progetto di Visual Basic include un valore letterale XML, è possibile visualizzare il XML schema impostato **XML Schema Explorer**. Per visualizzare il set di schemi associato con un valore letterale XML, fare clic su un nodo XML in un valore letterale XML o un'importazione di spazi dei nomi XML e selezionare **Mostra in Schema Explorer**.

![Valori letterali XML di Visual Basic; XML Schema Explorer](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer1.gif)

Verrà visualizzata la **XML Schema Explorer** side-by-side con i file di Visual Basic.

![Valori letterali XML di Visual Basic; XML Schema Explorer](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer2.gif)
