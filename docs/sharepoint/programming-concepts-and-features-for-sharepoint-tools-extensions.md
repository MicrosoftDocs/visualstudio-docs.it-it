# <a name="programming-concepts-and-features-for-sharepoint-tools-extensions"></a>Concetti di programmazione e funzionalità per le estensioni degli strumenti di SharePoint
  Esistono diverse importanti concetti di programmazione e funzionalità correlate agli strumenti di SharePoint in Visual Studio. Questi concetti e caratteristiche svolgono un ruolo importante nella maggior parte delle estensioni degli strumenti di SharePoint.  
  
## <a name="in-this-section"></a>Contenuto della sezione
 [Chiamare i modelli a oggetti SharePoint](../sharepoint/calling-into-the-sharepoint-object-models.md)  
 Confronta con il client di SharePoint e i modelli a oggetti server SharePoint dalle estensioni degli strumenti di SharePoint e fornisce informazioni su come creare *comandi SharePoint* per effettuare chiamate nel modello a oggetti server SharePoint.  
  
 [Usare il servizio di progetto SharePoint](../sharepoint/using-the-sharepoint-project-service.md)  
 Descrive le funzionalità che il servizio di progetto SharePoint consente alle estensioni di strumenti di SharePoint e vengono fornite istruzioni per il recupero del servizio in diversi tipi di estensioni.  
  
 [Associare dati personalizzati con le estensioni degli strumenti di SharePoint](../sharepoint/associating-custom-data-with-sharepoint-tools-extensions.md)  
 Viene descritto come associare dati personalizzati a determinati tipi di oggetti nelle estensioni di strumenti di SharePoint.
