# <a name="sharepoint-development-samples"></a>Esempi di sviluppo di SharePoint
  È possibile utilizzare Visual Studio per scaricare e installare gli esempi di applicazioni di SharePoint dal [MSDN Code Gallery](http://go.microsoft.com/fwlink/?LinkId=254185).  
  
 È possibile scaricare ogni esempio singolarmente oppure scaricare un pacchetto di esempi contenente gli esempi correlati che condividono una tecnologia o un argomento. Si riceverà una notifica al momento della pubblicazione delle modifiche apportate al codice sorgente per ogni esempio scaricato.
