# <a name="customize-the-editor"></a>Personalizzare l'editor

È possibile personalizzare la formattazione, le tabulazioni, i tipi di carattere e i colori, IntelliSense e molte altre funzionalità dell'editor di testo per un linguaggio di programmazione specifico o per tutti. Gli argomenti di questa sezione illustrano come eseguire alcune di queste personalizzazioni.

## <a name="options-dialog-box"></a>Opzioni (finestra di dialogo)

Per altre informazioni sulle opzioni dell'editor di testo, vedere [Finestra di dialogo Opzioni dell'editor di testo](../ide/reference/text-editor-options-dialog-box.md).
