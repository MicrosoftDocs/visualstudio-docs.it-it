# <a name="f1-help"></a>Guida F1

Non è stato possibile trovare gli argomenti cercati. È possibile [inviare consigli per un miglioramento](#help-us-improve-f1).

## <a name="cursor-position"></a>Posizione del cursore

Se si stanno cercando informazioni della Guida F1 per una parola chiave o un nome di API, verificare che il cursore sia posizionato all'interno o subito a destra della parola prima di premere **F1**. Se non funziona ancora, selezionare l'elenco [Collegamenti utili](#useful-links) o eseguire una ricerca in https://docs.microsoft.com.

## <a name="help-us-improve-f1"></a>Consigli per il miglioramento della Guida F1

Contribuisci a migliorare l'esperienza della Guida F1. Registra un problema della Guida del prodotto facendo clic sul pulsante **Invia commenti sul prodotto** riportato di seguito. Includi gli elementi per i quali necessitavi di assistenza quando hai premuto **F1**, ad esempio il simbolo o la finestra su cui si trovava il cursore. Microsoft ringrazia per il contributo.
