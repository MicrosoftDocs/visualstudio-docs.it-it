# <a name="useenv-devenvexe"></a>/UseEnv (devenv.exe)
Avvia [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] e carica le variabili di ambiente nella finestra di dialogo **Directory di VC++**.  
  
## <a name="syntax"></a>Sintassi  
  
```  
Devenv /useenv  
```  
  
## <a name="example"></a>Esempio  
 L'esempio seguente avvia [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] e carica le variabili di ambiente nella finestra di dialogo **Directory di VC++**.  
  
```  
Devenv.exe /useenv  
```
