# <a name="-devenvexe"></a>/? (devenv.exe)

Visualizza una finestra di messaggio contenente tutte le opzioni `devenv` e una breve descrizione di ognuna.

## <a name="syntax"></a>Sintassi

```shell
devenv /?
```
