# <a name="options-text-editor-html-web-forms-miscellaneous"></a>Opzioni, Editor di testo, HTML (Web Form), Varie

Usare la pagina di opzioni **Varie** per impostare le preferenze per la modalità di gestione di elementi incollati nella pagina dell'editor HTML. Per accedere alla pagina, nella barra dei menu scegliere **Strumenti** > **Opzioni** e quindi espandere **Editor di testo** > **HTML (Web Form)** > **Varie**.

## <a name="miscellaneous-html-options"></a>Altre opzioni HTML

- **Formatta HTML quando si incolla**

   Quando questa casella di controllo è selezionata, l'editor riformatta gli elementi incollati nella pagina, usando le regole di formattazione definite. Se la casella di controllo non è selezionata, gli elementi vengono incollati con la stessa formattazione dell'elemento originale.

- **Richiedi '<' per attivare finestra completamento tag**

   Quando questa casella di controllo è selezionata, viene visualizzata la finestra completamento tag quando si digita il carattere maggiore di ("<").
