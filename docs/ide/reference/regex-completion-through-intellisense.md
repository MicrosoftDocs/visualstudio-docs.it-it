# <a name="regex-completion-through-intellisense-menu"></a>Completamento di un'espressione regolare (regex) tramite IntelliSense

Questo refactoring si applica a:

- C#

- Visual Basic

**Cosa:** Completamento dell'espressione regolare (Regex) tramite il menu IntelliSense.

**Quando:** Si desidera scrivere un'espressione regolare con supporto da IntelliSense. IntelliSense offre una funzionalità di completamento di base e una spiegazione del significato dei singoli caratteri dell'espressione regolare. 

**Motivo:** La scrittura di espressioni regolari è difficile e IntelliSense può aiutarti a scriverlo.

## <a name="how-to"></a>Procedure

1. Posizionare il cursore nella stringa dell'espressione regolare.
2. Premere **CTRL +** + **barra spaziatrice** per attivare il menu **IntelliSense** .
3. Selezionare il carattere da aggiungere alla stringa dell'espressione regolare.
