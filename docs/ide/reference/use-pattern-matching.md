# <a name="use-pattern-matching"></a>Usa i criteri di ricerca

Questo refactoring si applica a:

- C#

**Cosa:** Usare i criteri di ricerca.

**Quando:** Si vogliono usare le nuove funzionalità di criteri di ricerca in [C#](https://docs.microsoft.com/dotnet/csharp/whats-new/csharp-9#pattern-matching-enhancements) .

**Motivo:** Questo refactoring renderà il codice più chiaro e conciso.

## <a name="how-to"></a>Procedure

1. Posizionare il punto di inserimento all'interno dell'istruzione.

2. Premere **CTRL** + **.** per attivare il menu **Azioni rapide e refactoring**.

3. Selezionare **Usa i criteri di ricerca**.
