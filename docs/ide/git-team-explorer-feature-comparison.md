# <a name="side-by-side-comparison-of-git-and-team-explorer"></a>Confronto affiancato tra git e Team Explorer

È stata avviata la prima versione di una nuova esperienza git in Visual Studio 2019 [versione 16,8](/visualstudio/releases/2019/release-notes/). Questa nuova esperienza consente di ridurre il cambio del contesto con una semplice finestra di **modifiche git** che include attività git comuni. Include anche una finestra di **repository git** a livello di schermata per operazioni git più avanzate, ad esempio la gestione di rami e l'esplorazione del repository.

Se si usa Team Explorer, ecco una guida dettagliata che illustra come è possibile usare la nuova esperienza git.

> [!NOTE]
> Nella sezione seguente sono incluse le schermate dimensionate per adattarsi alle colonne di una tabella. Fare clic su ogni schermata per visualizzarne una versione più ampia. Se si usa un dispositivo touchscreen, toccare ogni schermata per visualizzarne una versione più grande.
