# <a name="how-to-debug-inline-assembly-code"></a>Procedura: eseguire il debug di codice assembly inline
> [!NOTE]
>  Le finestre di dialogo e i comandi di menu visualizzati potrebbero non corrispondere a quelli descritti nella Guida in quanto dipendono dall'edizione o dalle impostazioni in uso. Per modificare le impostazioni, scegliere Importa/esporta impostazioni dal menu Strumenti. Per altre informazioni, vedere [Personalizzare l'IDE di Visual Studio](../ide/personalizing-the-visual-studio-ide.md).  
  
 Il debugger fornisce due finestre di debug codice assembly inline, il **Disassembly** finestra e **registra** finestra.  
  
## <a name="procedure"></a>Routine  
  
#### <a name="to-debug-inline-assembly-code"></a>Per eseguire il debug di codice assembly inline  
  
1.  Utilizzare il **Disassembly** consente di visualizzare le istruzioni dell'assembly.  
  
2.  Utilizzare il **registra** consente di visualizzare il contenuto del registro.
