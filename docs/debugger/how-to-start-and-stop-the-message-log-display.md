# <a name="how-to-start-and-stop-the-message-log-display"></a>Procedura: avviare e arrestare la visualizzazione del log dei messaggi
Quando un [visualizzazione messaggi](../debugger/messages-view.md) finestra sia attiva, una **Avvia registrazione** o **Arresta registrazione** scelto verr√† visualizzato in Spy + + **messaggi** menu e corrispondente **Avvia registrazione** o **Arresta registrazione** pulsante della barra degli strumenti diventa attivo.  
  
### <a name="to-start-the-message-log-display"></a>Per avviare la visualizzazione del log dei messaggi
