# <a name="search-tools-for-spy-views"></a>Strumenti di ricerca per le visualizzazioni di Spy++
Utilizzare queste finestre di dialogo per specificare quali dati vengono visualizzati nella finestra visualizzazioni di Spy + +:

## <a name="in-this-section"></a>Contenuto della sezione
 [Finestra di dialogo Trova finestra](../debugger/find-window-dialog-box.md) Utilizzato per visualizzare le proprietà o i messaggi da una finestra specifica.

 [Finestra di dialogo ricerca finestre](../debugger/window-search-dialog-box.md) Utilizzato per trovare il nodo per una finestra specifica nella visualizzazione di Windows.

 Finestra di [dialogo Ricerca messaggi](../debugger/message-search-dialog-box.md) Utilizzato per trovare il nodo di un messaggio specifico in visualizzazione messaggio.

 Finestra di [dialogo ricerca processi](../debugger/process-search-dialog-box.md) Utilizzato per trovare il nodo per un processo specifico nella visualizzazione del processo.

 Finestra di [dialogo Ricerca thread](../debugger/thread-search-dialog-box.md) Utilizzato per trovare il nodo per un thread specifico nella visualizzazione thread.
