# <a name="output-tab-message-options-dialog-box"></a>Scheda Output, finestra di dialogo Opzioni messaggio
Utilizzare il **Output** tab per specificare quali dati di ogni messaggio all'elenco [visualizzazione messaggi](../debugger/messages-view.md). Per visualizzare il [finestra di dialogo Opzioni messaggio](../debugger/message-options-dialog-box.md), scegliere **messaggi di Log** dal **Spy** menu.  
  
 Le impostazioni seguenti sono disponibili sul **Output** scheda:  
  
 **Numeri di riga**  
 Visualizzare i numeri di riga.  
  
 **Livello di nidificazione del messaggio**  
 Prefisso messaggi annidati con un periodo per ogni livello.  
  
 **Parametri dei messaggi non elaborati**  
 Visualizzazione esadecimale **wParam** e **lParam** valori.  
  
 **Parametri di messaggio decodificato**  
 Visualizzare i risultati di decodifica del messaggio-specifica il **wParam** e **lParam** valori.  
  
 **Valori restituiti non elaborati**  
 Visualizzazione esadecimale **lResult** valore restituito.  
  
 **Valori restituiti decodificati**  
 Visualizzare i risultati di decodifica del messaggio-specifica il **lResult** valore restituito.  
  
 **Ora di origine del messaggio**  
 Il tempo trascorso dall'avvio del sistema Windows (solo per messaggi inseriti).  
  
 **Posizione del Mouse messaggio**  
 Coordinate dello schermo del puntatore del mouse quando il messaggio Ã¨ stato registrato (solo per messaggi inseriti).  
  
 **Numero massimo di righe**  
 Limitare il numero di righe che vengono mantenuti nella visualizzazione di messaggi attualmente selezionata.  
  
 **Anche i messaggi di Log in File**  
 Specificare un file di output per il log dei messaggi. Questo file di output viene scritto contemporaneamente con la finestra di messaggio di log.  
  
 **Salva le impostazioni come impostazioni predefinite**  
 Salvare le impostazioni precedenti per nuove finestre di flusso di messaggio. Queste impostazioni vengono salvate quando si esce da Spy + +.
