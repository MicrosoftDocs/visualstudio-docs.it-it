# <a name="message-search-dialog-box"></a>Finestra di dialogo Ricerca messaggi
Utilizzare questa finestra di dialogo per individuare e selezionare il nodo di un messaggio specifico in [visualizzazione messaggi](../debugger/messages-view.md). Per visualizzare questa finestra di dialogo, spostare lo stato attivo per una finestra di visualizzazione dei messaggi. Quindi scegliere **Trova messaggio** dalle **ricerca** menu.

 Sono disponibili le seguenti impostazioni:

 **Strumento di ricerca** lo strumento di trascinamento della selezione utilizzato per selezionare una finestra.

 **Nascondi Spy + +** rimuove la finestra principale di Spy + + e lascia il **ricerca messaggi** nella finestra di dialogo visibile nella parte superiore alle altre applicazioni.

 **Gestire** l'ID univoco del messaggio per la ricerca.

 **Tipo** il tipo di messaggio per la ricerca.

 **Messaggio** l'ID del messaggio per la ricerca.

 **Eseguire la ricerca direzione verso l'alto o verso il basso** la direzione iniziale della ricerca.
