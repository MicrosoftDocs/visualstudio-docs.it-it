# <a name="spy-menu-commands"></a>Spy (comandi di menu)
Il **Spy** menu sono disponibili comandi per lavorare con Spy + + Windows e le viste.

 **Windows** apre una nuova visualizzazione di Windows.

 **Processi** apre una nuova visualizzazione processi.

 **Thread** apre una nuova visualizzazione thread.

 **Registrare i messaggi** apre una nuova visualizzazione di messaggi e consente di aprire la [finestra di dialogo Opzioni messaggio](../debugger/message-options-dialog-box.md) per la visualizzazione.

 **Trovare la finestra** consente di visualizzare il **FindWindow** finestra di dialogo per selezionare una finestra con il mouse.
