# <a name="tree-menu-commands"></a>comandi del menu Struttura ad albero
Il **albero** menu sono disponibili comandi per espandere o comprimere le visualizzazioni.

 **Espandi un livello** espande l'elemento attualmente selezionato in una vista di sistema tramite la visualizzazione di un livello sotto tale.

 **Il ramo** espande tutti i livelli sotto l'elemento attualmente selezionato.

 **Espandi tutto** viene espansa e consente di visualizzare tutti i livelli in una vista di sistema.

 **Comprimi** comprime tutti i livelli sotto l'elemento attualmente selezionato.
