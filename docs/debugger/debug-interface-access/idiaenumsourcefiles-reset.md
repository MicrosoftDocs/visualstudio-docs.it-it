# <a name="idiaenumsourcefilesreset"></a>IDiaEnumSourceFiles::Reset

## <a name="syntax"></a>Sintassi

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Valore restituito
 Restituisce `S_OK`.
