# <a name="idiaenumdebugstreamdatagetcount"></a>IDiaEnumDebugStreamData::get_Count
Recupera il numero di record nel flusso di dati di debug.  
  
## <a name="syntax"></a>Sintassi  
  

## <a name="return-value"></a>Valore restituito
