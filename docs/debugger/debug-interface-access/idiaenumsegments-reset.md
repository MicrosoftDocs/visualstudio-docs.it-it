# <a name="idiaenumsegmentsreset"></a>IDiaEnumSegments::Reset
Riporta all'inizio la sequenza di enumerazione.

## <a name="syntax"></a>Sintassi

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Valore restituito
 Restituisce `S_OK`.
