# <a name="idiaenumsegmentsreset"></a>IDiaEnumSegments::Reset
Reimposta l'inizio di una sequenza di enumerazione.

## <a name="syntax"></a>Sintassi

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Valore restituito
 Restituisce `S_OK`.
