Recupera la versione <xref:System.Runtime.InteropServices.ComTypes.IEnumVARIANT> dell'enumeratore.

## <a name="syntax"></a>Sintassi


#### <a name="parameters"></a>Parametri

## <a name="return-value"></a>Valore restituito
