# <a name="idiaenumdebugstreamdatareset"></a>IDiaEnumDebugStreamData::Reset
Reimposta il valore di inizio di una sequenza enumerata.  
  
## <a name="syntax"></a>Sintassi  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce `S_OK`.
