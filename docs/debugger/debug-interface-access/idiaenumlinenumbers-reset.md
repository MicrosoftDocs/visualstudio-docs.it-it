# <a name="idiaenumlinenumbersreset"></a>IDiaEnumLineNumbers::Reset
Reimposta una sequenza di enumerazione all'inizio.

## <a name="syntax"></a>Sintassi

```C++
HRESULT Reset();
```

## <a name="return-value"></a>Valore restituito
 Restituisce `S_OK`.
