## <a name="prerequisites"></a>Prerequisiti

- Visual Studio 2017 con carico di lavoro di Python installato. Per istruzioni, vedere [Uso di Python in Visual Studio - Passaggio 0](../tutorial-working-with-python-in-visual-studio-step-00-installation.md).

È anche possibile usare una versione precedente di Visual Studio con Python Tools for Visual Studio installato. Vedere [Installazione del supporto di Python in Visual Studio](../installing-python-support-in-visual-studio.md). Alcune funzionalità potrebbero non essere disponibili nelle versioni precedenti, come indicato nella [matrice di funzionalità](../overview-of-python-tools-for-visual-studio.md#features-matrix)
