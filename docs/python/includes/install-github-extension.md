Per lavorare con i repository di GitHub all'interno di Visual Studio, installare l'estensione GitHub per Visual Studio. A tale scopo, eseguire il programma di installazione di Visual Studio, selezionare **modifica**e selezionare la scheda **singoli componenti** . Scorrere verso il basso fino alla sezione **strumenti del codice** , selezionare l' **estensione GitHub per Visual Studio**e selezionare **modifica**.
