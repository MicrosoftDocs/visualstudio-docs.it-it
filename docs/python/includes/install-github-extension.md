Per lavorare con i repository di GitHub all'interno di Visual Studio, installare l'estensione GitHub per Visual Studio. A tale scopo, eseguire il programma di installazione di Visual Studio, **Code tools** selezionare **Modifica** **GitHub extension for Visual Studio**e selezionare la scheda **Modify** **Singoli componenti** .
