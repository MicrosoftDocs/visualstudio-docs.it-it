Per lavorare con i repository di GitHub all'interno di Visual Studio, installare l'estensione GitHub per Visual Studio. A tal fine, eseguire il programma di installazione di Visual Studio, selezionare **Modifica** e quindi la scheda **Singoli componenti**. Scorrere verso il basso fino alla sezione **Strumenti per il codice**, selezionare **Estensione GitHub per Visual Studio** e quindi **Modifica**.
