1. Per connettersi ai repository di GitHub, eseguire il programma di installazione di Visual Studio, selezionare **Cambia** e quindi la scheda **Singoli componenti**. Scorrere verso il basso fino alla sezione **Strumenti per il codice**, selezionare **Estensione GitHub per Visual Studio** e quindi **Cambia**.
    
    ![Selezione dell'estensione GitHub nel programma di installazione di Visual Studio](../media/installation-github-extension.png)
