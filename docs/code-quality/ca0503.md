# <a name="ca0503"></a>CA0503
La proprietà è deprecata. Utilizzare la proprietà sostitutiva

 CA0503 avviso viene generato quando una proprietà di analisi codice obsoleto contiene un valore nel file di configurazione di compilazione o il progetto. Per risolvere il problema, utilizzare la proprietà specificata che lo sostituisce.
