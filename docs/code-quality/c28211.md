# <a name="c28211"></a>C28211
avviso C28211: previsto nome contesto statico per SAL_context

 Questo avviso indica che l'operando per il `_Static_context_` annotazione deve essere il nome di un contesto definito dello strumento. Questo avviso non dovrebbe mai verificarsi nell'utilizzo normale.
