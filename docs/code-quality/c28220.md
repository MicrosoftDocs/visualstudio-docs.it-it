# <a name="c28220"></a>C28220
avviso C28220: Espressione integer prevista per l'annotazione del parametro nell'annotazione

 Questo avviso indica che un parametro in un'annotazione deve essere un'espressione integer, e si è verificato un altro tipo. Ciò indica in genere una macro di codice errato dell'annotazione.
