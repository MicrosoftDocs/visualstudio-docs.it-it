# <a name="how-to-run-code-analysis-manually-for-managed-code"></a>Procedura: Eseguire manualmente l'analisi del codice per il codice gestito
Lo strumento di analisi del codice fornisce informazioni sul possibile difetto nel codice sorgente. È possibile eseguire l'analisi del codice automaticamente con ogni compilazione di un progetto di codice ed è anche possibile eseguire manualmente l'analisi del codice. Le regole controllate quando viene eseguita l'analisi del codice vengono specificate nella pagina analisi codice delle pagine delle proprietà del progetto. Per altre informazioni, vedere [Procedura: Configurare l'analisi del codice per un progetto](../code-quality/how-to-configure-code-analysis-for-a-managed-code-project.md)di codice gestito.

### <a name="to-run-code-analysis-manually"></a>Per eseguire manualmente l'analisi del codice

1. In **Esplora soluzioni**fare clic sul progetto.

2. Nel menu **analizza** fare clic su **Esegui analisi del codice sul** *nome del progetto*.
