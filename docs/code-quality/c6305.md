# <a name="c6305"></a>C6305
avviso C6305: mancata corrispondenza potenziale tra le quantità sizeof e countof

 Questo avviso indica che è in corso l'aggiunta o la sottrazione di una variabile che contiene un risultato `sizeof` da un puntatore o da un'espressione `countof`. Questo provocherà una scalabilità imprevista nell'aritmetica del puntatore.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 Per correggere il problema, usare il codice seguente:

```
void f(int *p)
{
  // code...
  p += 1;
}
```
