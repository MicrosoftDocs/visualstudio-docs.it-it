# <a name="c28223"></a>C28223
Avviso C28223: Non ha trovato Token o identificatore previsto per l'annotazione, parametro

 Questo avviso indica che un parametro in un'annotazione deve essere un identificatore, e alcuni altri simboli è stato rilevato. Ciò indica in genere una macro codificato in modo non corretto dell'annotazione. L'utilizzo delle parole chiave C o C++ in questa posizione causerà questo errore.
