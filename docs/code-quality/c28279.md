# <a name="c28279"></a>C28279
avviso C28279: per il simbolo, un 'begin' è stato trovato senza un corrispondente 'end'

 Il linguaggio di annotazione supporta un begin ed end (`{` e `}` in C) costrutto e l'associazione è diventato sbilanciata. Questa situazione può essere evitata se vengono utilizzate le macro.
