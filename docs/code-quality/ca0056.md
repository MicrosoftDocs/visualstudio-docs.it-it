# <a name="ca0056"></a>CA0056
Un file di progetto ha una versione non corretta dello strumento di analisi.

Questo avviso viene visualizzato se FxCopCmd.exe viene richiamato manualmente con un file di progetto con una versione dello strumento di analisi del codice non corretta.
