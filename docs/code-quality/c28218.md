# <a name="c28218"></a>C28218
avviso C28218: Per il parametro di funzione, il parametro dell'annotazione non corrisponde a quello trovato nel file

 Le annotazioni nella riga corrente e nella riga all'interno del messaggio non sono coerenti. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto per essere visualizzato nell'utilizzo normale.
