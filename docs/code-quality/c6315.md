# <a name="c6315"></a>C6315
avviso C6315: ordine errato delle operazioni: l'operatore and bit per bit ha maggiore precedenza rispetto a OR bit per bit. Aggiungere le parentesi per chiarire lo scopo

 Questo avviso indica che un'espressione in un contesto di test contiene le operazioni and bit per bit (`&`) e bit per bit (`|`), ma genera una costante perché l'operazione OR bit per bit si verifica per ultima. È necessario aggiungere le parentesi per chiarire lo scopo.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
void f( int i )
{
  if ( i & 2 | 4 ) // warning
  {
    // code
  }
}
```

 Per correggere il problema, aggiungere le parentesi, come illustrato nel codice seguente:

```
void f( int i )
{
  if ( i & ( 2 | 4 ) )
  {
    // code
  }
}
```
