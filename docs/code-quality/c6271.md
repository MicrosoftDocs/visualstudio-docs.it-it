# <a name="c6271"></a>C6271
avviso C6271: argomento supplementare passato a \<funzione >: parametro \<numero > non viene utilizzato dalla stringa di formato

 Questo avviso indica che vengono forniti argomenti aggiuntivi oltre a quelle specificate dalla stringa di formato. Per se stesso, questo difetto non avr√† alcun effetto visibile anche se indica che lo scopo del programmatore non viene riflessa nel codice.

## <a name="example"></a>Esempio
 Esempio di codice seguente genera questo avviso:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d",1,2);
}
```

 Per risolvere questo problema, usare il codice di esempio seguente:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d, %d",1,2);
}
```

 Esempio di codice seguente chiama la funzione di modifica stringa sicura `sprintf_s`per risolvere questo problema:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf_s( buff, 5,"%s %d", 1,2 ); //safe version
}
```
