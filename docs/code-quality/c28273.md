# <a name="c28273"></a>C28273
avviso C28273: per la funzione, le informazioni non sono coerenti con la dichiarazione di funzione  
  
 Questo avviso indica un errore nelle annotazioni, non nel codice analizzato. Le annotazioni che compare a una definizione di funzione non sono coerenti con quelli visualizzati in una dichiarazione. Le due annotazioni devono essere risolti in modo che corrispondano.
