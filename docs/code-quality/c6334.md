# <a name="c6334"></a>C6334
avviso C6334: l'operatore sizeof applicato a un'espressione con un operatore può produrre risultati imprevisti

 Questo avviso indica un utilizzo improprio dell'operatore `sizeof`. L'operatore `sizeof`, quando applicato a un'espressione, restituisce la dimensione del tipo dell'espressione risultante.

 Ad esempio, nel codice seguente:

```
char     a[10];
size_t   x;

x = sizeof (a - 1);
```

 a `x` verrà assegnato il valore 4, non 9, perché l'espressione risultante non è più un puntatore alla matrice `a`, ma semplicemente un puntatore.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
void f( )
{
  size_t x;
  char a[10];

  x= sizeof (a - 4);
  // code...
}
```

 Per correggere il problema, usare il codice seguente:

```
void f( )
{
  size_t x;
  char a[10];

  x= sizeof (a) - 4;
  // code...
}
```
