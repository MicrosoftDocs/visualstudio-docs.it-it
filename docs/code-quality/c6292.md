# <a name="c6292"></a>C6292
avviso C6292: ciclo for definito in modo corretto: conta dal massimo

 Questo avviso indica che un ciclo for potrebbe non funzionare come previsto.

 Si verifica quando un ciclo di conteggi da un massimo, ma dispone di una condizione di interruzione inferiore. Il ciclo verr√† terminato solo dopo che si verifica un overflow di integer.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
void f( )
{
   int i;

   for (i = 100; i >= 0; i++)
   {
      // code ...
   }
}
```

 Per risolvere il problema, utilizzare il codice seguente:

```
void f( )
{
   int i;

  for (i = 100; i >= 0; i--)
   {
      // code ...
   }
}
```
