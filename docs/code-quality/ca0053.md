# <a name="ca0053"></a>CA0053
Non è stato possibile caricare l'assembly di regole.

 Ciò indica che non è stato possibile caricare un assembly di regole mediante reflection. Questa situazione può verificarsi se l'utente dispone di regole personalizzate che non sono state compilate correttamente, o lo strumento di analisi non è possibile caricare l'assembly per altri motivi, ad esempio è stato negato l'accesso al file system.
