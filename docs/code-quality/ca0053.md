# <a name="ca0053"></a>CA0053
Non è stato possibile caricare l'assembly delle regole.

Ciò indica che non è stato possibile caricare un assembly delle regole tramite reflection. Questo problema può verificarsi se l'utente dispone di regole personalizzate non compilate correttamente o se lo strumento di analisi non è in grado di caricare l'assembly per altri motivi, ad esempio se l'accesso file system è stato negato.
