# <a name="c28214"></a>C28214
avviso C28214: i nomi di parametro di attributo devono essere p1... p9

 Questo avviso indica che, quando si costruisce una dichiarazione di annotazione, i nomi di parametro sono limitati a p1... p9. Questo avviso non dovrebbe mai verificarsi nell'utilizzo normale.
