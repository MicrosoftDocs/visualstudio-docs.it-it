# <a name="c28214"></a>C28214
avviso C28214: i nomi dei parametri degli attributi devono essere P1... P9

 Questo avviso indica che quando si costruisce una dichiarazione di annotazione, i nomi dei parametri sono limitati a P1... P9. Questo avviso non dovrebbe mai verificarsi nel normale utilizzo.
