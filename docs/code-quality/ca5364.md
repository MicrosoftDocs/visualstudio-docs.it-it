# <a name="ca5364-do-not-use-deprecated-security-protocols"></a>CA5364: Non usare i protocolli di sicurezza deprecate

|||
|-|-|
|TypeName|DoNotUseDeprecatedSecurityProtocols|
|CheckId|CA5364|
|Category|Microsoft.Security|
|Modifica importante|Non importante|

## <a name="cause"></a>Causa

Questa regola viene attivata quando viene soddisfatta una delle condizioni seguenti:
- Un deprecate <xref:System.Net.SecurityProtocolType?displayProperty=nameWithType> valore è stato fatto riferimento.
- Valore intero che rappresenta un valore obsoleto è stato assegnato a un <xref:System.Net.SecurityProtocolType> variabile.

I valori deprecati sono:
- Ssl3
- TLS
- Tls10
- Tls11

## <a name="rule-description"></a>Descrizione della regola

Transport Layer Security (TLS) protegge la comunicazione tra computer, più comunemente con HTTPS Hypertext Transfer Protocol Secure (). Precedenti versioni di TLS del protocollo sono meno sicure rispetto a TLS 1.2 e 1.3 TLS e sono più soggetto a nuove vulnerabilità. Evitare di precedenti versioni del protocollo per ridurre al minimo i rischi. Per indicazioni sull'identificazione e la rimozione di versioni del protocollo deprecate, vedere [risolvere il problema di TLS 1.0, 2nd Edition](/security/solving-tls1-problem).

## <a name="how-to-fix-violations"></a>Come correggere le violazioni

Non usare le versioni del protocollo TLS deprecate.

## <a name="when-to-suppress-warnings"></a>Soppressione degli avvisi

È possibile eliminare questo avviso se:
- Il riferimento alla versione protocollo deprecato non viene utilizzato per attivare una versione deprecata.
- È necessario connettersi a un servizio legacy che non può essere aggiornato per l'utilizzo di configurazioni sicure TLS.

## <a name="pseudo-code-examples"></a>Esempi di pseudocodice

### <a name="violation"></a>Violazione

```csharp
using System;
using System.Net;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5364 violation
        ServicePointManager.SecurityProtocol = SecurityProtocolType.Tls11 | SecurityProtocolType.Tls12;
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' CA5364 violation
        ServicePointManager.SecurityProtocol = SecurityProtocolType.Tls11 Or SecurityProtocolType.Tls12
    End Sub
End Class
```

### <a name="violation"></a>Violazione

```csharp
using System;
using System.Net;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5364 violation
        ServicePointManager.SecurityProtocol = (SecurityProtocolType) 768;    // TLS 1.1
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' CA5364 violation
        ServicePointManager.SecurityProtocol = CType(768, SecurityProtocolType)   ' TLS 1.1
    End Sub
End Class
```

### <a name="solution"></a>Soluzione

```csharp
using System;
using System.Net;

public class TestClass
{
    public void TestMethod()
    {
        // Let the operating system decide what TLS protocol version to use.
        // See https://docs.microsoft.com/dotnet/framework/network-programming/tls
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' Let the operating system decide what TLS protocol version to use.
        ' See https://docs.microsoft.com/dotnet/framework/network-programming/tls
    End Sub
End Class
```
