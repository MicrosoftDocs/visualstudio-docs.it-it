# <a name="ca0059"></a>CA0059
Errore di opzione della riga di comando.

 Questo errore si verifica se una riga di comando necessaria passare risulta mancante, ad esempio,

 /out o switch /console non è specificato. Questo errore si verifica anche se è presente un'opzione non è valida, ad esempio un valore di timeout non valido.
