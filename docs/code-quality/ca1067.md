# <a name="ca1067-override-equals-when-implementing-iequatable"></a>CA1067: Esegue l'override di Equals quando si implementa IEquatable

|Elemento|valore|
|-|-|
|ID regola|CA1067|
|Category|Microsoft. Design|
|Modifica|Senza interruzioni|

## <a name="cause"></a>Causa

Un tipo implementa <xref:System.IEquatable%601> , ma non esegue l'override del <xref:System.Object.Equals%2A> metodo.

## <a name="rule-description"></a>Descrizione della regola

Un tipo <xref:System.IEquatable%601> che implementa l'interfaccia indica che supporta il confronto di due istanze del tipo per verificarne l'uguaglianza. Ãˆ inoltre consigliabile eseguire l'override delle implementazioni della classe di base dei <xref:System.Object.Equals%2A> <xref:System.Object.GetHashCode> metodi e in modo che il loro comportamento sia coerente con quello dell' <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> implementazione. Per altri dettagli, vedere [qui](/dotnet/api/system.iequatable-1#notes-to-implementers).

L' <xref:System.Object.Equals%2A> implementazione deve restituire risultati coerenti con l' <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> implementazione.

## <a name="how-to-fix-violations"></a>Come correggere le violazioni

Per correggere una violazione, eseguire l'override <xref:System.Object.Equals%2A> e implementarla richiamando l' <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> implementazione. Ad esempio, i due frammenti di codice seguenti mostrano una violazione della regola e come risolverli:

```csharp
using System;

public struct S : IEquatable<S>
{
    private readonly int _value;
    public S(int f)
    {
        _value = f;
    }

    public bool Equals(S other)
        => _value == other._value;
}
```

```csharp
using System;

public struct S : IEquatable<S>
{
    private readonly int _value;
    public S(int f)
    {
        _value = f;
    }

    public bool Equals(S other)
        => _value == other._value;

    public override bool Equals(object obj)
        => obj is S objS && Equals(objS);

    public override int GetHashCode()
        => _value.GetHashCode();
}
```

## <a name="when-to-suppress-warnings"></a>Quando escludere gli avvisi

Non eliminare le violazioni di questa regola.
