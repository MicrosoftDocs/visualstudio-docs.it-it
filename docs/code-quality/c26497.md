# <a name="c26497-use_constexpr_for_function"></a>C26497 USE_CONSTEXPR_FOR_FUNCTION

Questa funzione% Function% potrebbe essere contrassegnata `constexpr` se si desidera la valutazione in fase di compilazione. Vedere [ C++ linee guida di base F. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rf-constexpr).
