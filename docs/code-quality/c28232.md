# <a name="c28232"></a>C28232
avviso C28232: \_Pre @ no__t-1, \_Post @ no__t-3 o \_Deref @ no__t-5 non applicato ad alcuna annotazione

 Questo avviso indica che un operatore `_Pre_`, `_Post_` o `_Deref_` viene visualizzato in un'espressione di annotazione senza una successiva annotazione funzionale. il modificatore è stato ignorato, ma ciò indica un'annotazione codificata in modo errato.
