# <a name="c6336"></a>C6336
avviso C6336: l'operatore aritmetico ha precedenza sull'operatore punto interrogativo, utilizzare parentesi per chiarire lo scopo

 Questo avviso indica un problema di precedenza operatori possibili. Il '+','-',' *' e '/' operatori hanno la precedenza sui '?' operatore. Se la precedenza nell'espressione non Ã¨ corretta, utilizzare parentesi per modificare la precedenza degli operatori.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
int Count();

void f(int flag)
{
  int result;
  result = Count() + flag ? 1 : 2;
  // code...
}
```

 Per risolvere questo problema, aggiungere le parentesi come illustrato nel codice seguente:

```
int Count();

void f(int flag)
{
  int result;
  result = Count() + (flag ? 1 : 2);
  // code...
}
```
