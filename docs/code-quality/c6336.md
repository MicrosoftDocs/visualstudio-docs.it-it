# <a name="c6336"></a>C6336
avviso C6336: operatore aritmetico ha precedenza sull'operatore punto interrogativo, utilizzare le parentesi per chiarire lo scopo  
  
 Questo avviso indica un problema di precedenza possibile operatore. Il '+','-',' *' e '/' operatori hanno la precedenza sul '?' operatore. Se la precedenza nell'espressione non Ã¨ corretta, utilizzare le parentesi per modificare la precedenza dell'operatore.  
  
## <a name="example"></a>Esempio  
 Il codice seguente genera questo avviso:  
  
```  
int Count();  
  
void f(int flag)  
{  
  int result;  
  result = Count() + flag ? 1 : 2;  
  // code...  
}  
```  
  
 Per risolvere il problema, aggiungere parentesi come illustrato nel codice seguente:  
  
```  
int Count();  
  
void f(int flag)  
{  
  int result;  
  result = Count() + (flag ? 1 : 2);  
  // code...  
}  
```
