# <a name="c28219"></a>C28219
avviso C28219: membro di enumerazione previsto per l'annotazione del parametro nell'annotazione

 Un parametro in un'annotazione deve essere un membro di denominata `enum` di un tipo e alcuni altri simboli è stata rilevata, utilizzare un membro di tale `enum` tipo. Ciò indica in genere una macro codificato in modo non corretto dell'annotazione.
