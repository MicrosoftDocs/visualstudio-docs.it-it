# <a name="c28219"></a>C28219
avviso C28219: Membro di enumerazione previsto per l'annotazione del parametro nell'annotazione

 Un parametro in un'annotazione che dovrà essere un membro del nodo indicato `enum` tipo e alcuni altri simboli è stato rilevato; usare un membro di tale `enum` tipo. Ciò indica in genere una macro di codice errato dell'annotazione.
