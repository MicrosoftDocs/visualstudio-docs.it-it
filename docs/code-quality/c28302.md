# <a name="c28302"></a>C28302
avviso C28302: per C++ il parametro di riferimento < parameter_name >, è stato trovato un operatore `_Deref_` aggiuntivo nel > \<annotation.

 Questo avviso viene segnalato quando un livello aggiuntivo di `_Deref_` viene usato in un parametro.

 SAL2 non richiede l'uso di un livello aggiuntivo di `_Deref_` quando si gestiscono parametri di riferimento. Questa particolare annotazione non è ambigua e viene interpretata correttamente, ma deve essere corretta.

 Spesso è possibile correggere questo problema rimuovendo semplicemente l'annotazione `__deref` precedente e usando la sintassi SAL2. Talvolta può essere necessario utilizzare `_At_` per fare riferimento all'oggetto specifico da annotare.
