# <a name="c28137"></a>C28137
avviso C28137: L'argomento della variabile deve invece essere una costante (valore letterale)

 Questo avviso viene segnalato quando in una chiamata di funzione manca la costante obbligatoria (valore letterale). Consultare la documentazione relativa alla funzione.

## <a name="example"></a>Esempio
 Ad esempio, la routine [ExAcquireResourceExclusiveLite](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-exacquireresourceexclusivelite) richiede il valore true o false per il parametro `Wait`. Nell'esempio riportato di seguito il codice genera questo avviso:

```
ExAcquireResourceExclusiveLite(Resource, Wait);
```

 Nell'esempio riportato di seguito il codice evita questo avviso:

```
ExAcquireResourceExclusiveLite(Resource, TRUE);
```
