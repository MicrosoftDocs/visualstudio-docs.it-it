# <a name="c28309"></a>C28309
avviso 28309: < parameter_name >: gli operandi dell'annotazione devono essere di tipo integer/enum/Pointer. Gli operandi void e C++ gli operatori di overload non sono supportati. I valori float sono approssimati come numeri interi. Tipi: \<typelist >.

 Si è provato a usare un void o una funzione in un'espressione di annotazione e l'analisi del codice non è in grado di gestirla.  Questo errore si verifica in genere quando viene usato un `operator==` implementato come funzione, ma possono verificarsi anche altri casi. Esaminare i tipi in \<typelist > per individuare gli indizi relativi al problema.
