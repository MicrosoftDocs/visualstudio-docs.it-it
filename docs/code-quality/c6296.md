# <a name="c6296"></a>C6296
avviso C6296: ciclo for: ciclo eseguito solo una volta corpo

 Questo avviso indica che un ciclo for potrebbe non funzionare come previsto. Quando l'indice Ã¨ senza segno e un ciclo di conteggi da zero, il relativo corpo viene eseguito una sola volta.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i--)
   {
      // code ...
   }
}
```

 Per risolvere il problema, utilizzare il codice seguente:

```
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i++)
   {
      // code ...
   }
}
```
