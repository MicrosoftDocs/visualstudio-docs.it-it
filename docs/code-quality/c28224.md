# <a name="c28224"></a>C28224
avviso C28224: L'annotazione richiede parametri

 Questo avviso indica che l'annotazione denominata viene utilizzata senza parametri e almeno un parametro è obbligatorio. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto per essere visualizzato nell'utilizzo normale.
