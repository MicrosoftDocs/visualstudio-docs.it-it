# <a name="c6509"></a>C6509
avviso C6509: annotazione non valida: Impossibile fare riferimento a' Return ' da una precondizione

 Questo avviso indica che non Ã¨ possibile usare la parola chiave `return` in una precondizione. La parola chiave `return` viene utilizzata per terminare l'esecuzione di una funzione e restituire il controllo alla funzione chiamante.

## <a name="example"></a>Esempio
 Nel codice seguente l'avviso viene generato in quanto `return` viene utilizzato in una precondizione:

```cpp
#include <sal.h>

int f (_In_reads_(return) char *pc)
{
    // code ...
    return 1;
}
```

 Per correggere il problema, usare il codice seguente:

```cpp
#include <sal.h>

int f (_In_reads_(i) char *pc, int i)
{
    // code ...
    return 1;
}
```
