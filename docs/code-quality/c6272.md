# <a name="c6272"></a>C6272
avviso C6272: non float passato come argomento \<number > quando è necessario float nella chiamata a \<function >

 Questo avviso indica che la stringa di formato specifica che è richiesto un valore float, ad esempio una specifica `%f` o `%g` per `printf,`, ma che viene passato un valore non float, ad esempio un Integer o una stringa. È probabile che questo difetto provochi un output errato; in alcuni casi, tuttavia, potrebbe verificarsi un arresto anomalo.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 Per correggere il problema, utilizzare `%i` invece della specifica `%f`, come illustrato nel codice seguente:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 Il codice seguente usa la funzione Safe String Manipulation, `sprintf_s`, per correggere il problema:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
