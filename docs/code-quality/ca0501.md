# <a name="ca0501"></a>CA0501

Impossibile leggere il report di output

Questo errore si verifica quando non è possibile scrivere il report di output a causa di uno dei motivi seguenti:

- Disco pieno

- Nessuna autorizzazione per scrivere nella directory

- La directory non esiste
