# <a name="ca0501"></a>CA0501
Impossibile leggere il report di output

 Ciò si verifica quando il report di output non può essere scritti a causa di uno dei motivi seguenti:

- Disco pieno

- Nessuna autorizzazione di scrittura nella directory

- Directory non esiste
