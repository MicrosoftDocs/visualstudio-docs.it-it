# <a name="c28275"></a>C28275
avviso C28275: il parametro _Macro_value\_ è null  
  
 Questo avviso indica che si verifica un errore interno nel file di modello, non nel codice analizzato. Il *macroValue* funzione è stata chiamata senza parametri.
