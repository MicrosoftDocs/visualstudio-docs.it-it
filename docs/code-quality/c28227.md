# <a name="c28227"></a>C28227
Avviso C28227: L'annotazione non può essere anche PrimOp nella dichiarazione precedente

 Questo avviso indica che l'annotazione denominata viene dichiarata come annotazione normale e anche in precedenza è stato dichiarato come PrimOp. Non deve essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto per la visualizzazione utilizzo tipico.
