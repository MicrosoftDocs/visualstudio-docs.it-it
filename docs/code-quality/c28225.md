# <a name="c28225"></a>C28225
avviso C28225: Numero non corretto di parametri necessari nell'annotazione

 Questo avviso indica che l'annotazione denominata viene usato con un numero errato di parametri. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto affinché sia disponibile in uso tipico.
