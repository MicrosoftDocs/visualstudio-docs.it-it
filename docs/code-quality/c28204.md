avviso C28204: \<funzione >: solo uno degli overload di questo e quello presente \<filename > (\<riga >) vengono annotati per \<paramname >: deve essere annotato entrambi o nessuno.  
  
 Questo avviso viene segnalato quando si verifica un errore nelle annotazioni.
