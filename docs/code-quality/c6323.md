# <a name="c6323"></a>C6323
avviso C6323: utilizzo di un operatore aritmetico su tipi Boolean  
  
 Questo avviso viene visualizzato se vengono utilizzati gli operatori aritmetici in tipi di dati Boolean. Utilizzo dell'operatore non corretto pu√≤ produrre risultati non corretti. Indica inoltre che lo scopo del programmatore vengano riflessi nel codice.  
  
## <a name="example"></a>Esempio  
 Il codice seguente genera questo avviso:  
  
```  
int test(bool a, bool b)  
{  
    int c = a + b;     //C6323  
    return c;  
}  
```  
  
 Per risolvere il problema, utilizzare l'operatore e il tipo di dati corretto.  
  
```  
int test(int a, int b)  
{  
    int c = a + b;       
    return c;  
}  
```
