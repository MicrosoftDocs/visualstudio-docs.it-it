# <a name="c1252"></a>C1252
avviso C1252: Dipendenza circolare o mancante tra plug-in: richiede GUID

 Lo strumento di analisi del codice genera questo avviso quando si verifica un errore interno con dipendenze di plug-in, non nel codice analizzato.
