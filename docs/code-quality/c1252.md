# <a name="c1252"></a>C1252
avviso C1252: dipendenza circolare o mancante tra plug-in: richiede il GUID  
  
 Lo strumento di analisi codice genera questo avviso quando si verifica un errore interno con dipendenze di plug-in, non nel codice analizzato.
