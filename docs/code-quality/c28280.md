# <a name="c28280"></a>C28280
avviso C28280: Per il simbolo, è stato trovato un 'end' senza un 'begin' corrispondente

 Il linguaggio di annotazione supporta un inizio e fine (`{` e `}` in C) costrutto e l'associazione è diventato un utilizzo non equilibrato. Questa situazione può essere evitata se vengono usate le macro.
