# <a name="c28222"></a>C28222
avviso 28222: \_sì\_, non \_alcun\_o \_probabilmente\_ prevista per l'annotazione

 Questo avviso indica che un parametro di un'annotazione deve essere uno dei simboli `_Yes_`, `_No_`o `_Maybe_`ed è stato rilevato un altro simbolo. Questo indica in genere una macro di annotazione codificata in modo errato.
