# <a name="c28222"></a>C28222
avviso 28222: \_Sì\_, \_No\_, o \_forse\_ previsto per l'annotazione

 Questo avviso indica che un parametro in un'annotazione deve essere uno dei simboli `_Yes_`, `_No_`, o `_Maybe_`, ed è stato rilevato un altro simbolo. Ciò indica in genere una macro di codice errato dell'annotazione.
