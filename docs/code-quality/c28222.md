# <a name="c28222"></a>C28222
avviso 28222: Sì\_, No\_, o _Maybe\_ previsto per l'annotazione

 Questo avviso indica che è previsto un parametro in un'annotazione può essere uno dei simboli `_Yes_`, `_No_`, o `_Maybe_`, ed è stato rilevato un altro simbolo. Ciò indica in genere una macro codificato in modo non corretto dell'annotazione.
