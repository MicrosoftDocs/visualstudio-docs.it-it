# <a name="c6274"></a>C6274
avviso C6274: non carattere passato come parametro \<number > quando il carattere è obbligatorio nella chiamata a \<function >

 Questo avviso indica che la stringa di formato specifica che è necessario un carattere (ad esempio, una specifica `%c` o `%C`) ma che viene passato un valore non Integer, ad esempio float, String o struct. È probabile che questo errore provochi un output non corretto.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 Per correggere il problema, usare il codice seguente:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 Il codice seguente usa la funzione Safe String Manipulation, `sprintf_s`, per correggere il problema:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
