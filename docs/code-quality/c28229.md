# <a name="c28229"></a>C28229
avviso C28229: L'annotazione non supporta parametri

 Questo avviso indica che un'annotazione è stata usata con un parametro quando l'annotazione viene dichiarata senza parametri. Ciò non dovrebbe essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto affinché sia disponibile in uso tipico.
