# <a name="ca0055"></a>CA0055
Non è stato possibile caricare il file:\<path>

Lo strumento di analisi del codice non è riuscito a caricare il file di destinazione specificato per l'analisi. Questo errore si verifica se il file non viene trovato o se l'accesso al file è stato negato. In genere, questo errore si verifica se l'utente ha scelto di eseguire l'override della proprietà dell'assembly di input in Visual Studio o richiamato fxcopcmd.exe manualmente.
