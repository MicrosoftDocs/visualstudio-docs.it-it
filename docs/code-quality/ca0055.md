# <a name="ca0055"></a>CA0055
Impossibile caricare il file:\<percorso >

 Lo strumento di analisi codice non è riuscito a caricare il file di destinazione che è stato specificato per l'analisi. Ciò si verifica se il file non viene trovato o è stato negato l'accesso al file. In genere, ciò si verifica se l'utente ha scelto di eseguire l'override della proprietà dell'assembly di input in Visual Studio o richiamato fxcopcmd.exe manualmente.
