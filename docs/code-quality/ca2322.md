# <a name="ca2322-ensure-javascriptserializer-is-not-initialized-with-simpletyperesolver-before-deserializing"></a>CA2322: assicurarsi che JavaScriptSerializer non sia inizializzato con SimpleTypeResolver prima della deserializzazione

|||
|-|-|
|TypeName|EnsureJavaScriptSerializerIsNotInitializedWithSimpleTypeResolverBeforeDeserializing|
|CheckId|CA2322|
|Categoria|Microsoft.Security|
|Modifica|Senza interruzioni|

## <a name="cause"></a>Causa

È stato chiamato o fatto riferimento a un metodo di deserializzazione <xref:System.Web.Script.Serialization.JavaScriptSerializer?displayProperty=nameWithType> e il <xref:System.Web.Script.Serialization.JavaScriptSerializer> potrebbe essere stato inizializzato con un <xref:System.Web.Script.Serialization.SimpleTypeResolver?displayProperty=nameWithType>.

## <a name="rule-description"></a>Descrizione della regola

[!INCLUDE[insecure-deserializers-description](includes/insecure-deserializers-description-md.md)]

Questa regola trova <xref:System.Web.Script.Serialization.JavaScriptSerializer?displayProperty=nameWithType> le chiamate al metodo di deserializzazione o i riferimenti, quando è possibile che la <xref:System.Web.Script.Serialization.JavaScriptSerializer> sia stata inizializzata con un <xref:System.Web.Script.Serialization.SimpleTypeResolver?displayProperty=nameWithType>.

## <a name="how-to-fix-violations"></a>Come correggere le violazioni

- Verificare che gli oggetti <xref:System.Web.Script.Serialization.JavaScriptTypeResolver?displayPropertyName=nameWithType> non vengano inizializzati con una <xref:System.Web.Script.Serialization.SimpleTypeResolver?displayProperty=nameWithType>.
- Se il codice deve leggere i dati serializzati utilizzando una <xref:System.Web.Script.Serialization.SimpleTypeResolver>, limitare i tipi deserializzati a un elenco previsto implementando una <xref:System.Web.Script.Serialization.JavaScriptTypeResolver>personalizzata.
- Eseguire la prova di manomissione dei dati serializzati. Dopo la serializzazione, la firma crittografica dei dati serializzati. Prima della deserializzazione, convalidare la firma crittografica. Proteggere la chiave crittografica da divulgare e progettare per le rotazioni delle chiavi.

## <a name="pseudo-code-examples"></a>Esempi di pseudo-codice

### <a name="violation"></a>Violazione

```csharp
using System.Web.Script.Serialization;

public class ExampleClass
{
    public JavaScriptSerializer Serializer { get; set; }

    public T Deserialize<T>(string str)
    {
        return this.Serializer.Deserialize<T>(str);
    }
}
```

```vb
Imports System.Web.Script.Serialization

Public Class ExampleClass
    Public Property Serializer As JavaScriptSerializer

    Public Function Deserialize(Of T)(str As String) As T
        Return Me.Serializer.Deserialize(Of T)(str)
    End Function
End Class
```

### <a name="solution"></a>Soluzione

```csharp
using System.Web.Script.Serialization;

public class ExampleClass
{
    public T Deserialize<T>(string str)
    {
        JavaScriptSerializer s = new JavaScriptSerializer();
        return s.Deserialize<T>(str);
    }
}
```

```vb

Public Class ExampleClass
    Public Function Deserialize(Of T)(str As String) As T
        Dim s As JavaScriptSerializer = New JavaScriptSerializer()
        Return s.Deserialize(Of T)(str)
    End Function
End Class
```

### <a name="solution"></a>Soluzione

```csharp
using System;
using System.Web.Script.Serialization;

public class BookRecordTypeResolver : JavaScriptTypeResolver
{
    // For compatibility with data serialized with a JavaScriptSerializer initialized with SimpleTypeResolver.
    private static readonly SimpleTypeResolver Simple = new SimpleTypeResolver();

    public override Type ResolveType(string id)
    {
        // One way to discover expected types is through testing deserialization
        // of **valid** data and logging the types used.

        ////Console.WriteLine($"ResolveType('{id}')");

        if (id == typeof(BookRecord).AssemblyQualifiedName || id == typeof(AisleLocation).AssemblyQualifiedName)
        {
            return Simple.ResolveType(id);
        }
        else
        {
            throw new ArgumentException("Unexpected type ID", nameof(id));
        }
    }

    public override string ResolveTypeId(Type type)
    {
        return Simple.ResolveTypeId(type);
    }
}

public class BookRecord
{
    public string Title { get; set; }
    public string Author { get; set; }
    public int PageCount { get; set; }
    public AisleLocation Location { get; set; }
}

public class AisleLocation
{
    public char Aisle { get; set; }
    public byte Shelf { get; set; }
}

public class ExampleClass
{
    public BookRecord DeserializeBookRecord(string s)
    {
        JavaScriptSerializer serializer = new JavaScriptSerializer(new BookRecordTypeResolver());
        return serializer.Deserialize<BookRecord>(s);
    }
}
```

```vb
Imports System

    Private Dim Simple As SimpleTypeResolver = New SimpleTypeResolver()

        ' of **valid** data and logging the types used.

        End If
    End Function

    Public Overrides Function ResolveTypeId(type As Type) As String

    End Function
End Class
```
