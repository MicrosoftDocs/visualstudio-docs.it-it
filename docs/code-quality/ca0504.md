# <a name="ca0504"></a>CA0504
La directory di regole è stata ignorata perché non esiste

 CA504 avviso viene generato quando una directory specificata nella **CodeAnalysisRuleDirectories** proprietà in un file di configurazione di progetto o una compilazione non è stato trovato. Per risolvere questo problema, immettere un percorso di directory valido.
