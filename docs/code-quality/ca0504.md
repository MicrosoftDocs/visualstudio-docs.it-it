# <a name="ca0504"></a>CA0504
La directory della regola è stata ignorata perché non esiste

L'avviso CA504 viene generato quando non è possibile trovare una directory specificata nella proprietà **CodeAnalysisRuleDirectories** in un file di configurazione di progetto o di compilazione. Per risolvere questo problema, immettere un percorso di directory valido.
