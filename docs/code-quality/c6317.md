# <a name="c6317"></a>C6317
avviso C6317: operatore errato: Logical-Not (!) non è interscambiabile con i complemento a uno (~)

 Questo avviso indica che un NOT logico (`!`) viene applicato a una costante che probabilmente sarà un flag di bit. Il risultato di NOT logico è booleano; non è corretto applicare l'operatore and bit per bit (`&`) a un valore booleano. Usare l'operatore di complemento a uno (`~`) per capovolgere i flag.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & !FLAGS) // warning
  {
    // code
  }
}
```

 Per correggere il problema, usare il codice seguente:

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & ~FLAGS)
  {
    // code
  }
}
```
