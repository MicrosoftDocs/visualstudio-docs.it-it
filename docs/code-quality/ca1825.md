# <a name="ca1825-avoid-zero-length-array-allocations"></a>CA1825: Evitare allocazioni di matrici di lunghezza zero

|||
|-|-|
|CheckId|Ca1825|
|Category|Microsoft. performance|
|Modifica|Senza interruzioni|

## <a name="cause"></a>Causa

Viene allocato un oggetto vuoto senza <xref:System.Array> elementi.

## <a name="rule-description"></a>Descrizione della regola

L'inizializzazione di una matrice di lunghezza zero comporta un'allocazione di memoria non necessaria. Usare invece l'istanza di matrice vuota allocata in modo statico chiamando il <xref:System.Array.Empty%2A?displayProperty=nameWithType> metodo. L'allocazione di memoria è condivisa tra tutte le chiamate di questo metodo.

> [!NOTE]
> La regola ca1825 non è disponibile nell'analisi legacy. È stata introdotta per la prima volta negli [analizzatori FxCop](https://www.nuget.org/packages/Microsoft.CodeAnalysis.FxCopAnalyzers) versione 2.6.0.

## <a name="how-to-fix-violations"></a>Come correggere le violazioni

Per correggere una violazione, sostituire l'allocazione di matrici di lunghezza zero con una chiamata a <xref:System.Array.Empty%2A?displayProperty=nameWithType> . Ad esempio, i due frammenti di codice seguenti mostrano una violazione della regola e come risolverli:

```csharp
class C
{
    public void M1()
    {
        // Violates rule CA1825.
        var a = new int[0];
    }
}
```

```csharp
class C
{
    public void M1()
    {
        // Resolves rule CA1825 violation.
        var a = System.Array.Empty<int>();
    }
}
```

> [!TIP]
> Una correzione del codice è disponibile per questa regola in Visual Studio. Per usarlo, posizionare il cursore sull'allocazione di matrici e premere **CTRL** + **.** (punto). Scegliere **Usa Array. Empty** dall'elenco di opzioni presentate.
>
> ![Correzione del codice per ca1825: usare una matrice vuota](media/ca1825-codefix.png)

## <a name="when-to-suppress-warnings"></a>Quando escludere gli avvisi

È possibile eliminare una violazione di questa regola se non si è preoccupati dell'allocazione di memoria aggiuntiva.
