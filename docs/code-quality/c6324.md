# <a name="c6324"></a>C6324
avviso C6324: possibile utilizzo non corretto di \<funzione1 >: si intendeva utilizzare \<> funzione2?

 Questo avviso indica che è stata utilizzata una funzione di copia di stringa in cui deve essere utilizzata una funzione di confronto di stringhe. Un uso errato della funzione può causare un errore di logica imprevisto.

## <a name="example"></a>Esempio
 Il codice seguente genera questo avviso:

```cpp
#include <string.h>

void f(char *title )
{
  if (strcpy (title, "Manager") == 0) // warning 6324
  {
    // code
  }
}
```

 Per correggere il problema, usare `strcmp` come illustrato nel codice seguente:

```cpp
#include <string.h>

void f(char *title )
{
  if (strcmp (title, "Manager") == 0)
  {
    // code
  }
}
```
