# <a name="c28213"></a>C28213
avviso C28213: Il `_Use_decl_annotations_` annotazione deve essere utilizzata per fare riferimento, senza alcuna modifica, una dichiarazione precedente. \<il motivo per cui >

 Questo avviso viene segnalato quando viene fatto riferimento a una dichiarazione precedente senza i necessari `_Use_decl_annotations_` annotazione.
