# <a name="c28236"></a>C28236
avviso C28236: l'annotazione non può essere utilizzata in un'espressione  
  
 Questo avviso indica che è stato effettuato un tentativo di utilizzare una funzione dichiarata come un'annotazione in un contesto di espressione. Non deve essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto per la visualizzazione utilizzo tipico.
