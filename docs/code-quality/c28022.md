# <a name="c28022"></a>C28022
avviso C28022: Le classi di funzioni \<classlist1 > in questa funzione non corrispondono alla classe di funzione (es) \<classlist2 > sul typedef usato per definirlo.

 Questo avviso viene segnalato quando si verifica un errore nelle annotazioni. Sia il typedef che la funzione stessa hanno annotazioni `_Function_class_`, ma non corrispondono. Se vengono usati entrambi, devono corrispondere.
