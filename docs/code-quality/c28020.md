# <a name="c28020"></a>C28020
avviso C28020: L'espressione \<expr > non è true per questa chiamata

 Questo avviso viene segnalato quando l'espressione \_Satisfies @ no__t-1 elencata non è true. Spesso indica un parametro errato.

 Se questa situazione si verifica in una dichiarazione di funzione, le annotazioni indicano una condizione non possibile.
