# <a name="c28024"></a>C28024
avviso C28024: Il puntatore a funzione assegnato a viene annotato con la classe di funzione \<class >, che non Ã¨ contenuta nelle classi di funzioni \<classlist >.

 Questo avviso viene segnalato quando entrambe le funzioni sono state annotate con una classe di funzione, ma le classi non corrispondono.

 Questo avviso viene generato quando un puntatore a funzione ha un'annotazione `_Function_class_` che specifica che devono essere assegnate solo le funzioni di una determinata classe funzionale. In un'assegnazione o in un'assegnazione implicita in una chiamata di funzione, l'origine e la destinazione devono essere della stessa classe della funzione, ma le classi di funzioni non corrispondono.
