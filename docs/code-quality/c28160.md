# <a name="c28160"></a>C28160
avviso C28160: annotazione errore

 Questo avviso viene segnalato quando è stata rilevata un'annotazione `__drv_error`. Questa annotazione viene utilizzata per contrassegnare le procedure di codifica da correggere e può essere utilizzata con un'annotazione `__drv_when` per indicare combinazioni specifiche di parametri.
