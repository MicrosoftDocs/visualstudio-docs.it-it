# <a name="c26464-use_const_pointer_for_elements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  I valori a cui puntano gli elementi dell'array '% array%' sono assegnati una sola volta, contrassegnare gli elementi come puntatore a `const`. Vedere [ C++ linee guida di base con. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
