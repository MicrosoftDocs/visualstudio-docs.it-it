# <a name="c28208"></a>C28208
avviso C28208: la funzione \<function > stata precedentemente definita con un elenco di parametri diverso in \<File > (\<line >). Alcuni strumenti di analisi produrranno risultati non corretti

 Questo avviso viene segnalato quando la definizione nota di una funzione non corrisponde a un'altra occorrenza.
