# <a name="c26466-nostaticdowncastpolymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  Non usare `static_cast` downcast. Un cast da un tipo polimorfico deve usare dynamic_cast. Visualizzare [Type.2 linee guida di base di C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast).
