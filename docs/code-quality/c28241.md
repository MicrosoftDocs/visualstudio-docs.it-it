# <a name="c28241"></a>C28241
avviso C28241: l'annotazione per la funzione nel parametro non è riconosciuto.  
  
 È stato utilizzato un nome di annotazione non riconosciuto. Non deve essere possibile se vengono usate le macro standard per le annotazioni; Questo avviso non è previsto per la visualizzazione utilizzo tipico.
