# <a name="warning-c26814-use-constexpr-for-constants-whose-value-is-known-at-compile-time-con5"></a>Avviso C26814 utilizzare constExpr per le costanti il cui valore è noto in fase di compilazione. (Con. 5)
La variabile const '% variable%' può essere calcolata in fase di compilazione. Prendere in considerazione l'uso di constExpr (con. 5)

## <a name="example"></a>Esempio
```cpp
const int foo = 1234;  // C26814 reported here.
constexpr int getMagicNumber()
{
    return 42;
}

void bar()
{
    const int myval = 3; // C26814 reported here
    const int magicNumber = getMagicNumber(); // C26814 reported here.
}
```
