# <a name="c28285"></a>C28285
avviso C28285: Per la funzione, errore di sintassi nel parametro

 Lo strumento di analisi del codice indica che questo avviso quando viene rilevato un errore di probabili nel file del modello. Alcuni errori di file di origine possono causare anche tali errori.
