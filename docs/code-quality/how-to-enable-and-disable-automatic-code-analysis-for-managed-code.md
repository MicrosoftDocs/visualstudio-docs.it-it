# <a name="how-to-enable-and-disable-automatic-code-analysis-for-managed-code"></a>Procedura: abilitare e disabilitare l'analisi automatica del codice per il codice gestito

È possibile configurare l'analisi del codice da eseguire dopo ogni compilazione di un progetto di codice gestito. È possibile impostare un codice diverso le proprietà di analisi per ogni configurazione di compilazione.

## <a name="to-enable-or-disable-automatic-code-analysis"></a>Per abilitare o disabilitare l'analisi automatica del codice

1. In **Esplora**, fare clic sul progetto e quindi scegliere **proprietà**.

1. Nella finestra di dialogo proprietà per il progetto, scegliere **analisi del codice**.

1. Specificare il tipo di compilazione in **configurazione** e la piattaforma di destinazione in **piattaforma**.

1. Per abilitare o disabilitare l'analisi automatica del codice, selezionare o deselezionare il **Attiva analisi codice in fase di compilazione** casella di controllo.
