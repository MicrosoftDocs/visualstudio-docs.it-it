# <a name="c28300"></a>C28300
avviso C28300: < parameter_name >: operandi dell'espressione di tipi incompatibili per operator < operator_name >

 Questo avviso viene segnalato quando i tipi di operando in un parametro non sono compatibili con l'operatore.
