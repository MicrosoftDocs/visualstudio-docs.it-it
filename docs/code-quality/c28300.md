# <a name="c28300"></a>C28300
avviso C28300: < nome_parametro >: operandi dell'espressione di tipi incompatibili per l'operatore < operator_name >

 Questo avviso viene segnalato quando i tipi degli operandi in un parametro non sono compatibili con l'operatore.
