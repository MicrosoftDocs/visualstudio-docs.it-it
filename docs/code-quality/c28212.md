# <a name="c28212"></a>C28212
avviso C28212: Prevista espressione del puntatore per l'annotazione

 Questo avviso indica che il parametro numerato in un'annotazione (non la funzione annotata) è previsto un puntatore o tipo di matrice, ma si è verificato un altro tipo. L'annotazione deve essere corretto.
