# <a name="c28221"></a>C28221
avviso C28221: stringa di espressione previsto per il parametro nell'annotazione  
  
 Questo avviso indica che è previsto un parametro in un'annotazione deve essere una stringa, ed è stato rilevato un altro tipo. Questo in genere indica una macro di annotazione codice errato e non deve essere visualizzato durante il normale uso.
