# <a name="c28210"></a>C28210
avviso 28210: le annotazioni per il contesto `_On_failure_` non devono essere nel contesto preliminare esplicito

 Le annotazioni `_On_failure_` devono essere indicate in modo esplicito o implicito nel contesto `__post`, ovvero da applicare dopo la restituzione della funzione.  Usare `_drv_out` per assicurarsi che.
