# <a name="c26138"></a>C26138
avviso C26138: La sospensione di una coroutine durante il blocco viene mantenuto attivo \<blocco >.

Avviso C26138 genera un avviso quando una coroutine viene sospesa quando un blocco. In generale, è possibile conoscere quanto tempo sarà una coroutine rimangono nello stato sospeso in modo che questo modello può comportare più sezioni critiche del previsto.

## <a name="example"></a>Esempio
 Il codice seguente genererà C26138.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}

generator<int> mutex_acquiring_generator_report_once() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}
```

## <a name="example"></a>Esempio
 Il codice seguente consente di correggere questi avvisi.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator2() {
  {
    global_m.lock();
    ++var;
    global_m.unlock();
  }
  co_yield 1;                           // no 26138, global_m is already released above.
}
```
