# <a name="walkthroughs-using-excel"></a>Procedure dettagliate con Excel
  Le procedure dettagliate seguenti descrivono come usare personalizzazioni a livello di documento e componenti aggiuntivi VSTO per automatizzare Microsoft Office Excel e personalizzare l'interfaccia utente.

## <a name="document-level-walkthroughs"></a>Procedure dettagliate a livello di documento
- [Procedura dettagliata: Creazione della prima personalizzazione a livello di documento per](../vsto/walkthrough-creating-your-first-document-level-customization-for-excel.md) Excel viene illustrato come creare una personalizzazione di base a livello di documento per Excel.

- [Procedura dettagliata: Creare una scheda personalizzata usando la finestra di progettazione](../vsto/walkthrough-creating-a-custom-tab-by-using-the-ribbon-designer.md) della barra multifunzione illustra come aggiungere una scheda personalizzata alla barra multifunzione di una cartella di lavoro di Excel.

- [Procedura dettagliata: La raccolta di dati tramite Windows](../vsto/walkthrough-collecting-data-using-a-windows-form.md) form illustra l'uso di un Windows Form per raccogliere input utente e quindi inviare l'input a una cella in un foglio di lavoro di Excel.

- [Procedura dettagliata: Modificare la formattazione del foglio di](../vsto/walkthrough-changing-worksheet-formatting-using-checkbox-controls.md) lavoro utilizzando i controlli CheckBox illustra le nozioni di base sull'utilizzo di caselle di controllo in un foglio di lavoro di Excel per

- [Procedura dettagliata: Il testo visualizzato in una casella di testo di un foglio di](../vsto/walkthrough-displaying-text-in-a-text-box-in-a-worksheet-using-a-button.md) lavoro usando un pulsante illustra le nozioni di base sull'uso di pulsanti e caselle di testo nei fogli di lavoro di Excel.

- [Procedura dettagliata: Programma per gli eventi di un controllo](../vsto/walkthrough-programming-against-events-of-a-namedrange-control.md) NamedRange fornisce istruzioni dettagliate su come aggiungere un <xref:Microsoft.Office.Tools.Excel.NamedRange> controllo a un foglio di esecuzione e programmarlo in base agli eventi.

- [Procedura dettagliata: Data Binding semplice in un progetto](../vsto/walkthrough-simple-data-binding-in-a-document-level-project.md) a livello di documento illustra le nozioni di base sull'associazione di un singolo campo dati in un database SQL Server a un intervallo denominato in Excel.

- [Procedura dettagliata: Data Binding complesse in un progetto a livello di](../vsto/walkthrough-complex-data-binding-in-a-document-level-project.md) documento illustra le nozioni di base sull'associazione di più celle in un foglio di lavoro di Excel ai campi nel database di SQL Server Northwind.

- [Procedura dettagliata: Creare una relazione di dettaglio master usando un set](../vsto/walkthrough-creating-a-master-detail-relation-using-a-cached-dataset.md) di dati memorizzato nella cache per illustrare la creazione di una relazione master/dettaglio in un foglio di lavoro e la memorizzazione nella cache dei dati in modo che la soluzione possa essere usata offline.

- [Procedura dettagliata: L'aggiornamento di un grafico in un foglio di](../vsto/walkthrough-updating-a-chart-in-a-worksheet-using-radio-buttons.md) lavoro con pulsanti di opzione Mostra le nozioni di base sulla modifica degli stili dei grafici utilizzando pulsanti di opzione in un foglio di lavoro

- [Procedura dettagliata: Associare dati a controlli in un riquadro](../vsto/walkthrough-binding-data-to-controls-on-an-excel-actions-pane.md) azioni di Excel viene descritto come aggiungere controlli associati a un'origine dati a un riquadro azioni in Excel.
