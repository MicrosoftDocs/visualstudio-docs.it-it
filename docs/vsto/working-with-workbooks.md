# <a name="work-with-workbooks"></a>Lavorare con le cartelle di lavoro
  La classe <xref:Microsoft.Office.Tools.Excel.Workbook> e <xref:Microsoft.Office.Interop.Excel.Workbooks> contiene i metodi e le proprietà usati per eseguire attività con le cartelle di lavoro.  
  
|Attività|Routine|  
|----------|---------------|  
|Creare una nuova cartella di lavoro.|[Procedura: A livello di codice crea nuove cartelle di lavoro](../vsto/how-to-programmatically-create-new-workbooks.md)|  
|Aprire una cartella di lavoro specificata.|[Procedura: A livello di codice aprire cartelle di lavoro](../vsto/how-to-programmatically-open-workbooks.md)|  
|Chiudere una cartella di lavoro e salvare le modifiche.|[Procedura: A livello di programmazione chiudere cartelle di lavoro](../vsto/how-to-programmatically-close-workbooks.md)|  
|Salvare le cartelle di lavoro in formati Excel o formato XML.|[Procedura: A livello di programmazione salvare cartelle di lavoro](../vsto/how-to-programmatically-save-workbooks.md)|  
|Portare lo stato attivo sulla cartella di lavoro corrente o su una cartella di lavoro diversa.|  
|[Procedura: A livello di programmazione proteggere cartelle di lavoro](../vsto/how-to-programmatically-protect-workbooks.md)|  
|Ottenere una raccolta di file di una cartella di lavoro di usati di recente.|[Procedura: A livello di codice vengono usati i file della cartella di lavoro di elenco](../vsto/how-to-programmatically-list-recently-used-workbook-files.md)|  
|Aprire file di testo come cartelle di lavoro.|[Procedura: A livello di codice aprire i file di testo come cartelle di lavoro](../vsto/how-to-programmatically-open-text-files-as-workbooks.md)|  
  
 Per altre informazioni sulle attività e sul modello a oggetti, vedere [Cenni preliminari sul modello a oggetti di Excel](../vsto/excel-object-model-overview.md).  
  
 In alcuni casi, le modalità con le quali si eseguono queste attività nei componenti aggiuntivi VSTO sono diverse da quelle usate nelle personalizzazioni a livello di documento.
