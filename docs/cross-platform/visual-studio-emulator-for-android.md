# <a name="visual-studio-emulator-for-android"></a>Emulatore di Visual Studio per Android

L'emulatore di Visual Studio per Android è un'applicazione desktop per emulare un dispositivo Android.The Visual Studio Emulator for Android is a desktop application to emulate an Android device. È deprecato in Visual Studio 2015 e non è supportato in Visual Studio 2017 o Visual Studio 2019. Ti consigliamo di utilizzare invece gli emulatori Google Android. Per visualizzare le istruzioni dell'emulatore di Visual Studio per Android per Visual Studio 2015, scegliere Visual Studio 2015 nel selettore della versione della documentazione.

> [!IMPORTANT]
> Nella maggior parte degli scenari è consigliabile usare l'emulatore Android di Google anziché Visual Studio Emulator for Android:
> - L'emulatore di Visual Studio per Android non è supportato dopo Visual Studio 2015.
> - Le immagini dell'emulatore successive a Android versione 6.0 non sono disponibili per l'emulatore di Visual Studio per Android.
> - L'emulatore Android di Google ora supporta [Hyper-V](/xamarin/android/get-started/installation/android-emulator/hardware-acceleration#accelerating-with-hyper-v).
> - Sia il linguaggio c'è per lo sviluppo di dispositivi mobili che Visual Studio Tools per Apache Cordova funzionano con l'emulatore Android di Google. Per altre informazioni, vedere [Eseguire l'app Apache Cordova in Android](/visualstudio/cross-platform/tools-for-cordova/run-your-app/run-app-android#google-android-emulator). (Non è più necessario disabilitare Hyper-V come spiegato in questo articolo).
>
> Per ulteriori informazioni sulla configurazione e l'utilizzo dell'emulatore Android di Google, consultate [Configurazione dell'emulatore Android](/xamarin/android/get-started/installation/android-emulator/).
