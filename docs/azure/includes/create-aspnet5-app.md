1. Nel menu di Visual Studio selezionare **file**  >  **nuovo**  >  **progetto**.
2. Nella sezione **modelli** della finestra di dialogo **nuovo progetto** Selezionare **Visual C#**  >  **Web**.
3. Selezionare **Applicazione Web ASP.NET Core**.
4. Immettere un nome per la nuova applicazione (o usare il nome predefinito) e quindi fare clic su **OK**.
5. Selezionare **applicazione Web**.
6. Selezionare la casella di controllo **Abilita supporto Docker** .
7. Selezionare il tipo di contenitore desiderato (Windows o Linux) e quindi fare clic su **OK**.
