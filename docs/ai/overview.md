# <a name="overview-of-visual-studio-tools-for-ai"></a>Panoramica di Visual Studio Tools for AI

Visual Studio Tools for AI è un Integrated Development Environment (IDE) che è possibile usare per compilare, testare e distribuire soluzioni di apprendimento avanzato. Per altre informazioni su come usare gli strumenti di intelligenza artificiale, vedere le esercitazioni e gli esempi seguenti.

## <a name="install-the-extension"></a>Installare l'estensione

Vedere la pagina [Install Visual Studio Tools for ai](installation.md) per informazioni su come scaricare e installare l'estensione.

## <a name="5-minute-quickstart"></a>Guida introduttiva di 5 minuti 

Informazioni su come eseguire una soluzione di apprendimento avanzato con [TensorFlow e Python](tensorflow-local.md)

## <a name="step-by-step-tutorial"></a>Esercitazione dettagliata

Per informazioni su come creare una soluzione di apprendimento avanzato, vedere l'esercitazione su come eseguire il [training di un modello TensorFlow nel cloud](tensorflow-vm.md) .

## <a name="samples"></a>Esempi

Esaminare gli esempi nel [repository degli esempi](https://github.com/Microsoft/samples-for-ai) e nella [raccolta di Machine Learning](https://gallery.cortanaintelligence.com/projects).

## <a name="next-steps"></a>Passaggi successivi
