# <a name="overview-of-visual-studio-tools-for-ai"></a>Panoramica di Visual Studio Tools per l'iA

Visual Studio Tools per l'iaaaè è un ambiente di sviluppo integrato (IDE) che è possibile usare per compilare, testare e distribuire soluzioni di deep learning. Ulteriori informazioni su come usare gli strumenti AI nelle esercitazioni e negli esempi seguenti.

## <a name="install-the-extension"></a>Installare l'estensione

Vedere la pagina [Installare Visual Studio Tools per](installation.md) l'applicazione delle informazioni per informazioni su come scaricare e installare l'estensione.
