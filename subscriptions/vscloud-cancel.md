# <a name="buying-visual-studio-cloud-subscriptions---cancelling-subscriptions"></a>Acquisto di sottoscrizioni cloud di Visual Studio - Annullamento delle sottoscrizioni

Per annullare una sottoscrizione cloud di Visual Studio, accedere al [portale di amministrazione delle sottoscrizioni](https://manage.visualstudio.com/) di Visual Studio con un account autorizzato ad accedere alla sottoscrizione di Azure usato per effettuare l'acquisto originale.
1.  Nell'angolo superiore sinistro verificare di aver selezionato la sottoscrizione di Azure appropriata.
2.  Nel riquadro di spostamento a sinistra selezionare **Riduci** per la sottoscrizione che si vuole annullare.
3.  Verrà visualizzata una finestra per modificare la quantità di sottoscrizioni associate all'account.
4.  Ridurre il numero di sottoscrizioni alla quantità desiderata o a zero (0) per annullare e fare clic su **Continua**.
5.  Controllare l'ordine e fare clic su **Conferma**.
## <a name="faq"></a>Domande frequenti:
### <a name="q--can-i-still-use-a-subscription-after-i-cancel"></a>D: è ancora possibile usare una sottoscrizione dopo l'annullamento?
R: sì, i relativi vantaggi rimangono comunque accessibili per la durata rimanente della sottoscrizione.
