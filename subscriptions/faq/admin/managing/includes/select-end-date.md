## <a name="how-can-i-select-an-end-date-for-a-subscription"></a>Come è possibile selezionare una data di fine per una sottoscrizione?

Le sottoscrizioni vengono sempre assegnate usando la data di fine migliore disponibile. Ad esempio, se il contratto include ordini con la data di fine 20/6/2020 e un altro con data di fine 20/6/2021, la prima sottoscrizione da assegnare terminerà il 20/6/2021. Nel caso di un contratto attendibile con stato sovrassegnato, la data di fine sarà 3 anni dopo la data di assegnazione della sottoscrizione.
