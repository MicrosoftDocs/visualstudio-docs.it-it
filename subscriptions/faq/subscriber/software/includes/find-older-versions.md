## <a name="where-can-i-find-older-versions-of-visual-studio"></a>Dove è possibile trovare le versioni precedenti di Visual Studio? 

Se si stanno cercando versioni precedenti di Visual Studio 2019 in modo specifico, è possibile trovare i collegamenti di download per le versioni e le istruzioni riportate nell'articolo relativo alle [versioni di Visual studio 2019](https://docs.microsoft.com/visualstudio/releases/2019/history?branch=updates%2Fhistory) .

Se sono necessarie versioni precedenti di Visual Studio, ad esempio Visual Studio 2017, vedere [download precedenti di Visual Studio](https://visualstudio.microsoft.com/vs/older-downloads/).
