## <a name="where-can-i-find-older-versions-of-visual-studio"></a>Dove è possibile trovare le versioni precedenti di Visual Studio? 

Un elenco dei codici Product Key disponibili con la sottoscrizione di Visual Studio è accessibile nel portale delle sottoscrizioni nella scheda codici Product Key all'indirizzo <https://my.visualstudio.com/productkeys> .
