## <a name="my-organization-has-multiple-visual-studio-subscribers-can-their-monthly-azure-credits-be-pooled"></a>L'organizzazione ha più sottoscrittori di Visual Studio. I crediti di Azure mensili possono essere aggregati?

Non è possibile aggregare i crediti. Questo vantaggio è progettato per uso personale da parte del sottoscrittore di Visual Studio.
