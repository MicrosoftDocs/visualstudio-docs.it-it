## <a name="are-there-any-other-limitations-i-should-be-aware-of"></a>Esistono altre limitazioni di cui tenere conto?

Il credito mensile di Azure per i sottoscrittori di Visual Studio è destinato solo ad attività di sviluppo e il test e non prevede un contratto di servizio con copertura finanziaria. Microsoft si riserva il diritto di sospendere qualsiasi istanza (macchina virtuale o servizio cloud) eseguita in modo continuo per più di 120 ore o nel caso venga determinato che l'istanza è usata in produzione. Questa capacità viene resa disponibile ai sottoscrittori di Visual Studio senza alcuna garanzia di disponibilità della capacità.

Il credito mensile non è applicabile per l'acquisto dei servizi seguenti:

- Piani di supporto
- Application Insights
- Sottoscrizioni di Visual Studio
- Azure DevOps
- Visual Studio App Center
- Prodotti con marchio di terze parti
- Prodotti venduti tramite [Visual Studio Marketplace](https://marketplace.visualstudio.com/)
- Prodotti venduti in altro modo separatamente da Azure (ad esempio, Azure Active Directory Premium)

Per acquistare questi servizi, sarà necessario rimuovere il [limite di spesa](/azure/billing/billing-spending-limit) e fornire una carta di credito per la fatturazione.
