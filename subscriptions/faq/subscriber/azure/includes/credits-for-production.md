## <a name="can-i-use-my-monthly-azure-credits-to-run-production-applications"></a>È possibile usare i crediti mensili di Azure per eseguire applicazioni di produzione?

I crediti mensili di Azure per i sottoscrittori di Visual Studio sono destinati in modo specifico all'utilizzo per singole attività di sviluppo/test. Per eseguire carichi di lavoro di produzione, è necessario eseguire l'aggiornamento a una sottoscrizione di Azure a pagamento, ad esempio quella [con pagamento in base al consumo](https://azure.microsoft.com/offers/ms-azr-0003p/).
