## <a name="how-do-i-renew-visual-studio-subscriptions"></a>Ricerca per categorie rinnovare le sottoscrizioni di Visual Studio? 

È possibile acquisire le sottoscrizioni di Visual Studio in diversi modi e il processo di rinnovo delle sottoscrizioni esistenti dipende dal modo in cui sono state acquisite.

### <a name="how-to-renew-retail-subscriptions"></a>Come rinnovare sottoscrizioni al dettaglio 

Se si dispone di una sottoscrizione al dettaglio è possibile eseguire il rinnovo in due modi: 

- Connettersi alla [pagina Sottoscrizioni](https://my.visualstudio.com/subscriptions) nel [portale delle sottoscrizioni](https://my.visualstudio.com/benefits)   e fare clic sul collegamento "rinnovo". 
- Visitare il [Microsoft Store](https://www.microsoft.com/store)   e cercare "sottoscrizione a Visual Studio" per visualizzare le opportunità di rinnovo disponibili. Ecco alcuni collegamenti rapidi: 


    - [Visual Studio Enterprise](https://www.microsoft.com/p/visual-studio-enterprise-subscription/dg7gmgf0dst4?activetab=pivot%3aoverviewtab) 

    - [Visual Studio Professional](https://www.microsoft.com/p/visual-studio-professional-subscription/dg7gmgf0dst3?activetab=pivot%3aoverviewtab)

    - [Visual Studio Test Professional](https://www.microsoft.com/p/visual-studio-test-professional-subscription/dg7gmgf0dst6?activetab=pivot%3aoverviewtab) 

> [!Note]
> È necessario rinnovare l'abbonamento entro 30 giorni dalla scadenza per qualificarsi per il rinnovo dei prezzi.

### <a name="how-to-renew-cloud-subscriptions"></a>Come rinnovare le sottoscrizioni cloud
Le sottoscrizioni cloud sono quelle acquistate tramite il [Visual Studio Marketplace](https://marketplace.visualstudio.com/).  Le sottoscrizioni cloud mensili e annuali vengono rinnovate fino a quando non vengono annullate, quindi non è necessaria alcuna azione di rinnovo.

### <a name="how-to-renew-subscriptions-purchased-through-volume-licensing"></a>Come rinnovare le sottoscrizioni acquistate tramite contratti multilicenza
Se la sottoscrizione viene fornita dal proprio lavoro, è probabile che sia stata acquistata e rinnovata tramite il contratto multilicenza della società.  È possibile contattare l'amministratore usando il pulsante **Contatta l'amministratore** nella parte superiore destra del portale delle [sottoscrizioni](https://my.visualstudio.com/benefits).
