# <a name="visual-studio-subscriptions"></a>Sottoscrizioni di Visual Studio
##  <a name="developer-tools-cloud-services-software-support-and-training"></a>Strumenti per sviluppatori, servizi cloud, software, supporto e formazione

Sottoscrizioni di Visual Studio è il set di risorse più completo per creare, distribuire e gestire applicazioni nelle piattaforme e nei dispositivi preferiti, inclusi Windows, iOS, Android e Linux.
