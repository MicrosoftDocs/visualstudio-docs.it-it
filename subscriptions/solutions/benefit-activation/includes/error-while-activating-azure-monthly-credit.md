## <a name="im-getting-an-error-while-trying-to-activate-my-azure-monthly-credit"></a>Viene visualizzato un errore durante il tentativo di attivare il credito mensile di Azure.

È possibile avere un solo vantaggio mensile di sviluppo/test del credito singolo di Azure per ogni sottoscrizione di Azure associata alla sottoscrizione Visual Studio sottoscrizione. Se in precedenza è stato attivato il credito Azure mensile, potrebbe essere visualizzato un errore. Se non si è certi di aver attivato il credito, accedere al [portale di Azure](https://portal.azure.com/) per verificare. 

Se si ha una nuova sottoscrizione di Visual Studio e viene visualizzato l'errore "Non è possibile attivare questo vantaggio Visual Studio più di una volta" quando si tenta di attivare il vantaggio del credito Azure mensile, contattare il supporto tecnico di Sottoscrizioni di Visual Studio facendo clic sul pulsante "Richiesta online" nella parte inferiore di questa soluzione. 

## <a name="verify-your-email"></a>Verificare il messaggio di posta elettronica 

Quando si attiva il credito mensile, il messaggio di posta elettronica usato durante l'attivazione viene Visual Studio sottoscrizione.  

È possibile che sia stato associato [un account alternativo](https://docs.microsoft.com/visualstudio/subscriptions/vs-alternate-identity) alla sottoscrizione Visual Studio sottoscrizione. L'aggiunta di un account alternativo alla sottoscrizione di Visual Studio consente di accedere ai vantaggi della sottoscrizione, tra cui Azure DevOps e Azure, con un'identità diversa rispetto a quella a cui è assegnata la sottoscrizione.  

Sarà necessario accedere al portale di Azure con [il](https://portal.azure.com/) messaggio di posta elettronica con cui si è configurato l'account. Ad esempio, se è stato attivato il credito mensile con l'account alternativo, è sempre necessario accedere alla sottoscrizione di Azure con tale indirizzo di posta elettronica. È possibile controllare se è stato aggiunto un account alternativo nella parte inferiore della pagina [delle sottoscrizioni](https://my.visualstudio.com/subscriptions) nel portale.
