## <a name="how-to-activate-the-monthly-azure-devtest-individual-credit-benefit"></a>Come attivare il vantaggio mensile di sviluppo/test del credito singolo di Azure

1. Accedere al portale per [i sottoscrittori](https://my.visualstudio.com/benefits) 
1. Individuare il riquadro di Azure per il credito mensile nella pagina Vantaggi e fare clic sul collegamento Attiva nella parte inferiore del riquadro del vantaggio. Se non è possibile individuare il riquadro [](https://visualstudio.microsoft.com/vs/benefits/#azure?cat=visual-studio-enterprise-subscription) Credito mensile di Azure, usare la pagina di confronto dei vantaggi per verificare che la sottoscrizione includa un credito mensile. 
1. Se non si ha una sottoscrizione di Azure esistente, verrà chiesto di immettere le informazioni personali necessarie per crearne una.  
1. Accettare il contratto per le sottoscrizioni e si è pronti per iniziare. In futuro sarà possibile accedere alla sottoscrizione di Azure [all'indirizzo portal.azure.com](https://portal.azure.com/). 

## <a name="geographic-restrictions-may-apply"></a>Possono essere applicate restrizioni geografiche 

Le sottoscrizioni di Azure create Sottoscrizioni di Visual Studio vantaggio possono essere influenzate da restrizioni che non vengono imposte alle sottoscrizioni a pagamento. Alcune aree geografiche potrebbero non essere disponibili per la creazione di macchine virtuali e servizi di Azure e alcuni tipi di servizi potrebbero non essere disponibili. Potrebbe essere presente una quota predefinita inferiore rispetto alle sottoscrizioni a pagamento.  

## <a name="azure-credit-benefit-activation-errors"></a>Errori di attivazione del vantaggio del credito Azure

Se si ha una nuova sottoscrizione di Visual Studio e viene visualizzato l'errore "Non è possibile attivare più volte questo vantaggio Visual Studio" quando si tenta di attivare il vantaggio credito Azure mensile, contattare il supporto tecnico di Sottoscrizioni di Visual Studio facendo clic sul pulsante "Crea un ticket di supporto" nella parte inferiore di questa soluzione. 

Se la sottoscrizione è stata assegnata per un po' di tempo, ecco alcuni aspetti da controllare per primi:
- Verificare che il vantaggio non sia già stato attivato controllando il riquadro Credito mensile di Azure nella [pagina Vantaggi](https://my.visualstudio.com/benefits) del portale delle sottoscrizioni. 
- È possibile avere più directory come parte della sottoscrizione di Azure. Assicurarsi di usare la directory in cui è stata impostata la sottoscrizione di Azure. È possibile visualizzare e cambiare directory facendo clic sull'icona dell'account nell'angolo superiore destro del [portale di Azure](https://portal.azure.com/).
- È anche possibile associare [un account alternativo](https://docs.microsoft.com/visualstudio/subscriptions/vs-alternate-identity) alla sottoscrizione Visual Studio sottoscrizione. Se il credito mensile è stato attivato con l'account alternativo, è necessario accedere sempre alla sottoscrizione di Azure con tale messaggio di posta elettronica. 

## <a name="more-information-about-azure-credits"></a>Altre informazioni sui crediti di Azure

Non è previsto alcun costo per l'uso del credito mensile di Azure e non è necessaria alcuna carta di credito. Quando si esaurirà il credito assegnato per il mese, le risorse di Azure verranno arrestate fino a quando il credito non viene reimpostato il mese successivo oppure non si rimuove il limite di spesa e si aggiunge un metodo di pagamento.
