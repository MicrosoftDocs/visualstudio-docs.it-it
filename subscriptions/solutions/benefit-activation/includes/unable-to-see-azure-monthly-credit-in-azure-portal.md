## <a name="im-unable-to-see-my-azure-monthly-credit-in-the-azure-portal"></a>Non è possibile visualizzare il credito mensile di Azure nel portale di Azure

Esistono diversi motivi per cui potrebbe non essere possibile visualizzare il credito mensile di Azure. È possibile che si sta usando il messaggio di posta elettronica errato per accedere, che sia stato eseguito l'accesso con il messaggio di posta elettronica corretto ma nella directory errata oppure che non sia ancora stato attivato il vantaggio di credito mensile. 

## <a name="azure-subscription-account"></a>Account di sottoscrizione di Azure 

Quando si attiva il credito mensile, il messaggio di posta elettronica usato durante l'attivazione viene Visual Studio sottoscrizione.  

È anche possibile associare un [account alternativo](https://docs.microsoft.com/visualstudio/subscriptions/vs-alternate-identity) alla sottoscrizione Visual Studio sottoscrizione. L'aggiunta di un account alternativo alla sottoscrizione di Visual Studio consente di accedere ai vantaggi della sottoscrizione, tra cui Azure DevOps e Azure, con un'identità diversa rispetto a quella a cui è assegnata la sottoscrizione.  

È necessario accedere al portale di Azure con [il](https://portal.azure.com/) messaggio di posta elettronica con cui si è configurato l'account. Ad esempio, se il credito mensile è stato attivato con l'account alternativo, è necessario accedere sempre alla sottoscrizione di Azure con tale messaggio di posta elettronica. 

## <a name="azure-directories"></a>Directory di Azure
 
È possibile avere più directory come parte della sottoscrizione di Azure. Assicurarsi di usare la directory in cui è stata impostata la sottoscrizione di Azure. È possibile visualizzare e cambiare directory facendo clic sull'icona dell'account nell'angolo superiore destro del portale di Azure.
