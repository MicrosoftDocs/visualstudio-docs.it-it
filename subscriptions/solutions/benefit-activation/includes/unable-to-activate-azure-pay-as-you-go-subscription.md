## <a name="im-unable-to-activate-my-azure-dev--test-pay-as-you-go-subscription"></a>Non è possibile attivare la sottoscrizione di Azure Dev/Test con pagamento in base al carico.

Come sottoscrittore Visual Studio sottoscrizione, si ricevono tariffe orarie scontate esclusive per i diritti di sviluppo/test e uso cloud per il software di sottoscrizione in Azure. 

## <a name="set-up-a-subscription"></a>Configurare una sottoscrizione

1. Accedere al portale [delle sottoscrizioni e](https://my.visualstudio.com/benefits) fare clic sul collegamento Attiva nella parte inferiore del riquadro del vantaggio Sviluppo/test con pagamento in base al carico di Azure.
1. Nella pagina Microsoft Azure di iscrizione è possibile configurare l'offerta Sviluppo/test con pagamento in base al consumo (payg) o eseguire l'aggiornamento all'offerta Visual Studio Enterprise. Fare clic su Continua con l'offerta corrente (Sviluppo/test con pagamento in base al consumo) se non si vuole eseguire l'aggiornamento. 
1. Aggiungere un metodo di pagamento, accettare il contratto di sottoscrizione e si è pronti per iniziare. In futuro sarà possibile accedere alla sottoscrizione di Azure [all'indirizzo portal.azure.com](https://portal.azure.com/).
