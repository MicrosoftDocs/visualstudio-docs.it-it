## <a name="azure-devops-technical-support"></a>Supporto tecnico di Azure DevOps  

Il supporto delle sottoscrizioni di Visual Studio offre assistenza per le sottoscrizioni di Visual Studio, vendite, sottoscrizioni, account e fatturazione. Se si ha una domanda o un problema tecnico di Azure DevOps, la domanda deve essere indirizzata al [supporto di Azure DevOps](https://azure.microsoft.com/support/devops/). Di seguito sono riportate alcune risorse che consentono di trovare le risposte.

## <a name="documentation--azure-devops"></a>Documentazione | Azure DevOps 

Esaminare le esercitazioni, le guide per la risoluzione dei problemi, le risorse utili e una serie di altri argomenti nella [documentazione di Azure DevOps](https://docs.microsoft.com/azure/devops/?view=azure-devops&preserve-view=true).

## <a name="developer-community--azure-devops"></a>Community degli sviluppatori | Azure DevOps

Cerca la [community degli sviluppatori](https://developercommunity.visualstudio.com/spaces/21/index.html) online e Ottieni risposte dagli esperti da un forum supportato dalla community. È possibile cercare parole chiave e leggere, votare, commentare tutti senza eseguire l'accesso. Se non si trova ciò che serve con una ricerca, è possibile suggerire una funzionalità o segnalare un problema nel menu di navigazione. 

## <a name="stack-overflow--azure-devops"></a>Overflow dello stack | Azure DevOps

Una community aperta per tutti i codici. Per ottenere le risposte alle domande già richieste oppure iscriversi a un account e inviare una nuova domanda, cercare il [Tag Public Q&a con Azure-DevOps](https://stackoverflow.com/questions/tagged/azure-devops?tab=Newest) .
