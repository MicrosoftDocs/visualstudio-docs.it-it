## <a name="im-unable-to-download-or-activate-visual-studio-community"></a>Non è possibile scaricare o attivare la community di Visual Studio
Visual Studio community è la versione gratuita e leggera dell'IDE e non è necessaria una sottoscrizione a pagamento di Visual Studio per usarla. È possibile [scaricarlo direttamente](https://visualstudio.microsoft.com/vs/community/) dal sito Web di Visual Studio.
