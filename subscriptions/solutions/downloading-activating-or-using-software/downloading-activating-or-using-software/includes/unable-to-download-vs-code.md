## <a name="im-unable-to-download-or-activate-visual-studio-code"></a>Non è possibile scaricare o attivare Visual Studio Code

Visual Studio Code è un editor di codice gratuito e non è necessaria una sottoscrizione a pagamento di Visual Studio per usarlo. È possibile [scaricarlo direttamente](https://code.visualstudio.com/download) dal sito Web di Visual Studio.
