## <a name="im-unable-to-download-or-activate-visual-studio-for-mac"></a>Non è possibile scaricare o attivare Visual Studio per Mac

I sottoscrittori di Visual Studio hanno accesso all'ambiente IDE di Visual Studio. Visual Studio per Mac è un download separato che è possibile trovare nella [pagina Download](https://my.visualstudio.com/Downloads) del portale sottoscrizioni.  

## <a name="find-and-download-visual-studio-for-mac"></a>Trovare e scaricare Visual Studio per Mac 
1. Accedere al portale delle [sottoscrizioni](https://my.visualstudio.com/benefits)e vedere una [visualizzazione filtrata](https://my.visualstudio.com/Downloads?q=Visual%20Studio%20for%20mac&pgroup=) dei download disponibili per Visual Studio per Mac. Sono disponibili solo le versioni più recenti del prodotto. 
2. Seguire le istruzioni per il download e la configurazione. 
1. Usando il messaggio di posta elettronica a cui è assegnata la sottoscrizione, sarà possibile accedere immediatamente all'IDE dopo l'installazione. non è necessario alcun codice Product Key.

## <a name="no-download-available"></a>Nessun download disponibile 
I download del software disponibili dipendono dalla sottoscrizione in uso. Se non viene visualizzato il download che si sta cercando, è possibile che non sia disponibile. Per verificare quali sottoscrizioni sono disponibili, accedere alla [pagina Sottoscrizioni](https://my.visualstudio.com/subscriptions) del portale. È possibile [scaricare un elenco completo](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx)dei   titoli software disponibili per tutti i tipi di sottoscrizione.
