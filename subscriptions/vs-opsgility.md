# <a name="the-opsgility-training-benefit-in-visual-studio-subscriptions"></a>Vantaggio per il training Opsgility nelle sottoscrizioni di Visual Studio

Alcune sottoscrizioni di Visual Studio includono fino a sei mesi di accesso a training esperto su richiesta con possibilità di accedere ad ambienti lab live e percorsi di apprendimento con certificazione.

## <a name="activation-steps"></a>Procedura di attivazione
1.  Accedere a [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).

2.  Per usufruire del vantaggio Skill Me Up, fare clic sul collegamento **Ottieni il codice** nella parte inferiore del riquadro del vantaggio.   Si riceverà la notifica del recupero del codice del coupon.  Fare clic su **Attiva** per continuare.
    > [!div class="mx-imgBorder"]
    > ![Riquadro del vantaggio Opsgility](_img\vs-opsgility\vs-opsgility-tile.png)


3.  Verrà effettuato il reindirizzamento alla pagina di destinazione [Opsgility Online Training for Visual Studio Enterprise](https://www.opsgility.com/vse) (Training online Opsgility per Visual Studio Enterprise).  Può presentarsi l'opportunità di avviare una sessione di chat con uno specialista del supporto tecnico.  Se non si vuole avviare una chat, è possibile ridurre a icona tale finestra.  Compilare le informazioni di contatto richieste e fare clic su **Register** (Iscriviti).
    > [!div class="mx-imgBorder"]
    > ![Pagina di destinazione del vantaggio Opsgility](_img\vs-opsgility\vs-opsgility-registration.png)

4.  Questo è tutto.  È tutto pronto per iniziare l'apprendimento.  Per rendere più facile iniziare, ecco l'elenco di percorsi di apprendimento suggeriti, disponibile sul lato sinistro della pagina.  Buona lettura.
    > [!div class="mx-imgBorder"]
    > ![Home page del vantaggio Opsgility](_img\vs-opsgility\vs-opsgility-home-cropped.png)

## <a name="eligibility"></a>Idoneità
| Livello di sottoscrizione                                                 |     Canali                                            | Vantaggio                                                          | Rinnovabile?    |
|--------------------------------------------------------------------|---------------------------------------------------------|------------------------------------------------------------------|---------------|
| Visual Studio Enterprise (Standard, cloud annuale)   | VL, Azure, Retail | 12 mesi       |  No.  Disponibile solo per i nuovi sottoscrittori          |
| Visual Studio Professional (Standard, cloud annuale) | VL, Azure, Retail                                       | 6 mesi                                                            |No.  Disponibile solo per i nuovi sottoscrittori         |
| Visual Studio Test Professional (Standard)                         | VL, Retail                                              | 6 mesi                                             |  No.  Disponibile solo per i nuovi sottoscrittori         |
| MSDN Platforms (Standard)                                          | VL, Retail                                              | 6 mesi                                              | No.  Disponibile solo per i nuovi sottoscrittori         |
| Visual Studio Dev Essentials | N/D  | 3 mesi | No.  Disponibile solo per i nuovi sottoscrittori  |
| Visual Studio Enterprise, Visual Studio Professional (cloud mensile) | Azure                                       | Non disponibile                                                           |N/D|

*Esclude:  Microsoft Partner Network (MPN), Microsoft Valued Partner (MVP), Microsoft Certified Trainer, BizSpark, Imagine, NFR, FTE*

Non si è certi della sottoscrizione in uso?  Connettersi a [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) per visualizzare tutte le sottoscrizioni assegnate al proprio indirizzo di posta elettronica. Se non sono visualizzate tutte le sottoscrizioni, è possibile che una o più sottoscrizioni siano assegnate a un indirizzo di posta elettronica diverso.  È necessario accedere con tale indirizzo di posta elettronica per visualizzare le sottoscrizioni.
