# <a name="locating-your-subscription-id-for-visual-studio-subscriptions"></a>Individuare l'ID sottoscrizione per le sottoscrizioni di Visual Studio

## <a name="what-is-a-subscription-id"></a>Che cos'è un ID sottoscrizione?
L'ID sottoscrizione è un GUID di 32 cifre associato alla sottoscrizione o al programma di un sottoscrittore.  In alcuni casi, il team di supporto può chiedere a un sottoscrittore di fornire l'ID sottoscrizione per limitare i potenziali problemi di una sottoscrizione specifica. 

## <a name="how-to-find-your-subscription-id"></a>Come trovare l'ID sottoscrizione
1. Passare a [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs).
2. Accedere alla sottoscrizione.  (Informazioni per l'[accesso](/visualstudio/subscriptions/signing-in).)
3. Dopo aver effettuato l'accesso, verrà visualizzato un elenco delle sottoscrizioni associate all'indirizzo di posta elettronica usato per l'accesso.  Sono indicate le informazioni seguenti:
    - Sottoscrizione/Programma
    - ID sottoscrizione 
    - Tipo (Versione finale, NFR e così via)
    - Data di validità  (scadenza)
    - Collegamenti per il rinnovo, per ottenere informazioni di supporto e, nel caso di Visual Studio Dev Essentials, per uscire dal programma.
