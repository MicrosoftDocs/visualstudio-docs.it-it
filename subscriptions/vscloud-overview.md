# <a name="buying-visual-studio-cloud-subscriptions---overview"></a>Acquisto di sottoscrizioni cloud di Visual Studio - Panoramica

Con le sottoscrizioni cloud è possibile noleggiare Visual Studio, VSTS e i vantaggi per i sottoscrittori senza un contratto a lungo termine. La fatturazione è gestita tramite la sottoscrizione di Microsoft Azure, che supporta il pagamento con carta di credito, gli acquisti tramite il contratto Enterprise Agreement* e i partner [Cloud Solution Provider](https://www.microsoft.com/solution-providers/search). Se non è ancora disponibile una sottoscrizione di Azure, non c'è da preoccuparsi. La configurazione è gratuita ed è possibile eseguirla al momento del primo acquisto.

Confrontare [vantaggi e prezzi delle sottoscrizioni](https://www.visualstudio.com/vs/pricing/) cloud.

## <a name="before-you-start"></a>Prima di iniziare
Prima di acquistare, seguire questa procedura per prepararsi per una transazione corretta • Per fatturare gli acquisti, è necessaria una [sottoscrizione di Azure](https://azure.microsoft.com/pricing/purchase-options/).  È possibile [iscriversi](https://portal.azure.com/) prima o durante il primo acquisto in Visual Studio Marketplace.  

• Per iscriversi per la sottoscrizione di Azure ed effettuare acquisti da Visual Studio Marketplace, è necessario un [account Microsoft personale](https://www.microsoft.com/account) o un [account aziendale o dell'istituto di istruzione](https://azure.microsoft.com/en-us/documentation/articles/sign-up-organization/), nel caso di acquisti per un'organizzazione che gestisce le identità con Azure Active Directory (Azure AD).
