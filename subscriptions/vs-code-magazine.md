# <a name="code-magazine-included-in-visual-studio-subscriptions"></a>CODE Magazine incluso nelle sottoscrizioni di Visual Studio

CODE Magazine è una rivista di sviluppatori software indipendente leader che copre le tecnologie di sviluppo in tutte le piattaforme.  Le sottoscrizioni di Visual Studio selezionate possono ricevere una sottoscrizione di un anno per le edizioni stampate o digitali della rivista.

## <a name="activation-steps"></a>Procedura di attivazione
Il processo di configurazione dell'abbonamento a CODE Magazine è semplice.  È sufficiente seguire questa procedura:

1. visita[https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
2. Individuare il riquadro CODE Magazine nella sezione Professional Development or Education del portale, quindi fare clic su **Get Code**.
   > [!div class="mx-imgBorder"]
   > ![Piastrella della rivista CODE](_img/vs-code-magazine/vs-code-magazine-tile.png "Piastrella CODE Magazine")
3. Verrà visualizzato un messaggio che indica che il codice è stato recuperato.  Fare clic su **Attiva** per continuare.  (Se non sei pronto per l'attivazione, va bene.  Puoi sempre tornare al riquadro per attivarlo.  Il link "Ottieni codice" leggerà "Attiva" dopo aver recuperato il codice.)
   > [!div class="mx-imgBorder"]
   > ![Codice CODE Magazine recuperato](_img/vs-code-magazine/vs-code-magazine-success.png "Codice recuperato correttamente")
4. Verrai reindirizzato alla pagina di destinazione di CODE Magazine. Seleziona il tuo paese e scegli come desideri ricevere la rivista. 
   > [!NOTE]
   > I membri di Visual Studio Dev Essentials hanno accesso solo alla sottoscrizione digitale. 
5. In base alle selezioni effettuate, verrà visualizzata una nota che mostra il prezzo dell'abbonamento insieme all'importo del coupon applicato.
   > [!NOTE]
   > Per gli abbonamenti stampati:
   > - Gli abbonati al di fuori degli Stati Uniti possono incorrere in spese di spedizione. 
   > - Si prega di attendere diverse settimane per la consegna della prima edizione stampata.
      > [!div class="mx-imgBorder"]
      > ![Dettagli dell'abbonamento a CODE Magazine](_img/vs-code-magazine/vs-code-magazine-details.png "Dettagli e prezzo dell'abbonamento")
6. Successivamente, dovrai fornire le informazioni dell'abbonato e un indirizzo di consegna.  L'indirizzo di consegna è necessario anche per gli abbonamenti digitali, in quanto supporta i processi di evasione, selezione dell'offerta e accesso digitale.
7. Se vivi al di fuori degli Stati Uniti e hai scelto di ricevere un abbonamento a una rivista stampata, ti potrebbe essere richiesto di fornire un indirizzo di fatturazione e informazioni di pagamento per coprire le spese di spedizione. 
8. È possibile acconsentire esplicitamente alla newsletter HTML CODE facendo clic sulla casella di controllo.
   > [!div class="mx-imgBorder"]
   > ![Informazioni per gli abbonati a CODE Magazine](_img/vs-code-magazine/vs-code-magazine-subscriber-info.png "Informazioni sull'abbonato e indirizzo di consegna")
9. Fare clic su **Ordina** per completare l'abbonamento.  
Dopo aver completato l'ordine di abbonamento, verrà visualizzata una pagina di conferma, incluso un collegamento che è possibile utilizzare per vedere i problemi ricevuti: https://codemag.com/my/fulfillment.

## <a name="frequently-asked-questions"></a>Domande frequenti
### <a name="q-if-the-subscription-is-free-why-am-i-being-asked-for-a-credit-card"></a>D: Se l'abbonamento è gratuito, perché mi viene chiesta una carta di credito?  
R: È proibitivo inviare copie stampate gratuite a livello internazionale.  Agli abbonati che scelgono l'edizione cartacea della rivista CODE e risiedono al di fuori degli Stati Uniti verrà addebitato un costo per la spedizione. Per evitare le spese di spedizione, scegli la versione digitale della rivista. 

### <a name="q-why-do-i-need-to-provide-a-delivery-address-for-a-digital-subscription"></a>D: Perché è necessario fornire un indirizzo di consegna per un abbonamento digitale?
R: CODE Magazine raccoglie il tuo indirizzo, indipendentemente dal fatto che l'abbonamento sia per un abbonamento cartaceo o digitale per supportare i processi di evasione, selezione dell'offerta e accesso digitale.  Per ulteriori informazioni su ciò che CODE Magazine fa con i tuoi dati, consulta la loro [Informativa sulla privacy](https://www.codemag.com/Home/Privacy).
