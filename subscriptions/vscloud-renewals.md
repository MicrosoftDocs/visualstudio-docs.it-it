# <a name="buying-visual-studio-cloud-subscriptions---renewals"></a>Acquisto di sottoscrizioni cloud di Visual Studio - Rinnovi

Il rinnovo delle sottoscrizioni cloud avviene automaticamente e gli addebiti per il rinnovo compariranno nella fattura di Azure.  Di seguito sono riportati alcuni esempi di cosa aspettarsi.
## <a name="monthly-subscriptions"></a>Sottoscrizioni mensili
Microsoft ripartirà proporzionalmente l'addebito per il primo mese.  

**Esempio:** se l'acquisto venisse effettuato il giorno 15, nella prima fattura si riceverebbe un addebito pari a circa il 50%.  Ogni mese successivo verrà poi addebitato sempre a prezzo intero il primo di ogni mese.

## <a name="annual-subscriptions"></a>Sottoscrizioni annuali
Le sottoscrizioni annuali vengono rinnovate il primo del mese successivo all'anniversario della data di acquisto.  Dopo il primo rinnovo, la data del rinnovo rimarrà coerente.

**Esempio:** la data di rinnovo per una sottoscrizione acquistata il 15 gennaio 2018 sarà 1 febbraio 2019.

## <a name="faq"></a>Domande frequenti:
### <a name="q--what-if-i-dont-want-to-auto-renew"></a>D: come è possibile evitare il rinnovo automatico?
R: per evitare i costi del rinnovo, è necessario annullare la sottoscrizione cloud di Visual Studio prima della data di rinnovo automatico.  Se si decide di annullare la sottoscrizione, i relativi vantaggi rimangono comunque accessibili per la durata rimanente della sottoscrizione.
