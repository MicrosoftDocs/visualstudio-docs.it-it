# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Enterprise Mobility + Security (EMS) nelle sottoscrizioni di Visual Studio

Enterprise Mobility + Security, che in precedenza era disponibile in modo indipendente nelle sottoscrizioni di Visual Studio selezionate, è ora inclusa nella sottoscrizione di Microsoft 365 Developer inclusa nelle stesse sottoscrizioni di Visual Studio selezionate.  

Di conseguenza, Enterprise Mobility + Security non verranno più offerti separatamente.  

Per attivare la sottoscrizione di Microsoft 365 Developer:
1. Accedere al portale delle [sottoscrizioni di Visual Studio](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
1. Individuare il riquadro sottoscrizione per sviluppatori Microsoft 365 e fare clic su **attiva**.  
1. Seguire le istruzioni per completare la creazione dell'account.  

Altre informazioni sull'attivazione di [Microsoft 365 sottoscrizione per sviluppatori](vs-m365.md). 

## <a name="support-resources"></a>Risorse di supporto
- Serve aiuto con EMS?  Consultare la documentazione di [Enterprise Mobility + Security](/enterprise-mobility-security/)online.
- Per il supporto tecnico con qualsiasi componente di EMS, visitare le [domande frequenti sul supporto Enterprise Mobility + Security](https://docs.microsoft.com/enterprise-mobility-security/solutions/ems-support-faq) per trovare il supporto appropriato per la domanda.
- Per assistenza in merito a vendite, sottoscrizioni, account e fatturazione per le sottoscrizioni di Visual Studio, contattare il [supporto delle sottoscrizioni](https://visualstudio.microsoft.com/subscriptions/support/)di Visual Studio.
- Per domande sull'IDE di Visual Studio, Azure DevOps Services o altri prodotti e servizi Visual Studio,  Visita il [supporto tecnico di Visual Studio](https://visualstudio.microsoft.com/support/).
