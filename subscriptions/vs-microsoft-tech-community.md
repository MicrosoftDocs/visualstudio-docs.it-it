# <a name="microsoft-tech-community-forum-support-in-visual-studio-subscriptions"></a>Supporto del forum Microsoft Tech community nelle sottoscrizioni di Visual Studio
È possibile connettersi e interagire con gli altri utenti e con esperti tecnici Microsoft sulle procedure consigliate, sull'architettura di progettazione e sugli aggiornamenti più recenti.


## <a name="access-the-microsoft-tech-community"></a>Accedi a Microsoft Tech community 
L'accesso al forum della community è semplice.  

1. Accedi a <https://my.visualstudio.com/benefits>
0. Individuare il riquadro **Microsoft Tech Community** e fare clic su **Ottieni assistenza**

    > [!div class="mx-imgBorder"]
    > ![Riquadro Microsoft Tech community](_img/vs-tech-community/vs-tech-community-tile.png "Fare clic su "Ottieni assistenza" nel riquadro Microsoft Tech community")

0. Se la community non è stata registrata in precedenza, **fare clic su inizia**
0. Nella sezione **passaggio 1: configurare il profilo** :
   - Selezione della lingua preferita
   - Immettere l'indirizzo di posta elettronica aziendale
   - Creare il nome del profilo 
0. Nella sezione relativa al **passaggio 2: selezionare le community a cui partecipare**, è possibile selezionare diverse community facendo clic sul pulsante di **join** elencato nella scheda informazioni per ciascuna di esse.  È anche possibile usare la casella di ricerca all'estrema destra per filtrare l'elenco delle community disponibili.  Per visualizzare l'elenco completo delle community di 90, fare clic sul collegamento **Visualizza tutto (90)** sotto le schede informazioni. 
0. Per il **passaggio 3: codice di comportamento**, leggere il codice di comportamento, quindi fare clic sulla casella di controllo per indicare l'accettazione del codice di comportamento e le condizioni del servizio.
0. Fare clic su **registra** per completare la configurazione del profilo.

Tutto qui.  A questo punto si è pronti per usare il portale della community.  Il home page presenta notizie sulla community.  Per visualizzare il contenuto specifico per le community di cui è stato aggiunto un join, fare clic sull'elenco a discesa **Hub Community** nella parte superiore della pagina e scegliere una community dall'elenco.  È sempre possibile aggiungere community all'elenco facendo clic su **Sfoglia tutti gli hub della community** facendo clic sull'hub desiderato, quindi facendo clic sul pulsante **Unisci** . 

## <a name="eligibility"></a>Idoneità
La disponibilità del vantaggio Community di Azure dipende dalla sottoscrizione come mostrato di seguito:

|                                          Livello di sottoscrizione                                           |     Canali      |    Vantaggi    | Rinnovabile? |
|-------------------------------------------------------------------------------------------------------|-------------------|---------------|------------|
|                           Visual Studio Enterprise (standard), NFR<sup>1</sup>                            | VL, Azure, Retail |   Disponibile    |    Sì     |
|                           Sottoscrizione di Visual Studio Enterprise con GitHub Enterprise                           | VL, Azure, Retail |   Disponibile    |    Sì     |
|                          Visual Studio Professional (Standard)                          | VL, Azure, Retail |   Disponibile    |    Sì     |
|                          Sottoscrizione di Visual Studio Professional con GitHub Enterprise                          | VL, Azure, Retail |   Disponibile    |    Sì     |
|                              Visual Studio Test Professional (Standard)                               |    VL, Retail     |   Disponibile    |    Sì     |
|                                       MSDN Platforms (Standard)                                       |    VL, Retail     |   Disponibile    |    Sì     |
| Visual Studio Enterprise (cloud mensile), Visual Studio Professional (cloud mensile)|        Tutti        | Non disponibile |     No     |

<sup>1</sup>  *include: not for Resale (NFR), Microsoft ETP, most VALUABLE Professional (MVP), Regional Director (Rd), Visual Studio Industry Partner (VSIP), BizSpark, Imagine, Microsoft Certified Trainer (MCT). Esclude NFR Basic.*

> [!NOTE]
> Microsoft non offre più sottoscrizioni annuali di Visual Studio Professional e Visual Studio Enterprise nelle sottoscrizioni cloud. Non verrà apportata alcuna modifica all'esperienza dei clienti corrente, né alle possibilità di rinnovo, espansione, riduzione o annullamento delle sottoscrizioni esistenti. I nuovi clienti sono invitati a passare a [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) per esplorare diverse opzioni per l'acquisto di Visual Studio.

Non si è certi della sottoscrizione in uso?  Connettersi a per [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) visualizzare tutte le sottoscrizioni assegnate all'indirizzo di posta elettronica. Se non sono visualizzate tutte le sottoscrizioni, è possibile che una o più sottoscrizioni siano assegnate a un indirizzo di posta elettronica diverso.  È necessario accedere con tale indirizzo di posta elettronica per visualizzare le sottoscrizioni.
