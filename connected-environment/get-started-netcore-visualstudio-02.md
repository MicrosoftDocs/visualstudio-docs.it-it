# <a name="get-started-on-connected-environment-with-net-core-and-visual-studio"></a>Introduzione a Connected Environment con .NET Core e Visual Studio

Passaggio precedente: [Installare gli strumenti](get-started-netcore-visualstudio-01.md)

## <a name="create-an-aspnet-web-app"></a>Creare un'app Web ASP.NET
Da Visual Studio 2017 creare un nuovo progetto, attualmente di tipo **Applicazione Web di ASP.NET Core**. Denominare il progetto '**webfrontend**'.

![](images/NewProjectDialog1.png)

Selezionare il modello **Applicazione Web (Model-View-Controller)** e assicurarsi di selezionare **.NET Core** e **ASP.NET Core 2.0** come destinazione nei due elenchi a discesa nella parte superiore della finestra di dialogo. Fare clic su **OK** per creare il progetto.

![](images/NewProjectDialog2.png)
